　　不说卡皮尔带着一群勤劳的懒汉如何种地，却说这大唐长安城近来出了一件新鲜事。
<br/>　　一个名叫杨常林的方士在西市那里开了一家名叫百草行的药店，号称包治百病，很多长安城的百姓对这家店那时趋之若鹜，俨然把这里当成了比大唐医学院还要高一级的圣地。
<br/>　　李承乾原本并不知道这件事情，可架不住身边有人撺掇，于是便换了普通的衣服，邀请了孙思邈一同过去看看。
<br/>　　……
<br/>　　一路无话到了西市，远远便听到了阵阵喧哗，又行了一阵，赶车的苏猛将车停了下来：“殿下，前面过不去了！”
<br/>　　过不去了？李承乾愣了一下，西市虽然热闹，但还没有热闹到把路都堵死的程度吧？
<br/>　　可当他掀开车帘之后，却发现，似乎苏猛并没说错。
<br/>　　只见前面的主路已经完全被人堵死，无数长安百姓正围在一起，翘首向一个方向看着。
<br/>　　“殿下，不如下去看看？”孙思邈老道士同样也对前面的一切感到有些好奇，看清了前面的一切之后，向李承乾发出邀请。
<br/>　　李承乾对此自然是无可无不可，点了点头便下了马车与老孙结伴挤入了人群之中。
<br/>　　……
<br/>　　“杨神医，杨神医，你还记得我不？我从幽州来的，我家老父亲瘫痪在床上十来年了，前些日子吃了你的药没用三天竟然就能动了，所以我特地不远千里赶过来，就是想再买几副。”
<br/>　　刚刚挤进人群，李承乾和孙思邈便听到了这样的一声喊，使得两人都是一愣。
<br/>　　瘫痪十来年，吃了点药就能动了？这不是扯蛋么？就算是仙丹也不能这么灵吧？
<br/>　　不过，还不等两人反应过来，又一个声音响了起来：“杨神医，我也是来买药的，我母亲咳嗽十多年了，这十多年一直在吃你的药，果然轻了不少，这次我要再买一百副回家。”
<br/>　　卧槽？真的假的？这人有病吧？咳嗽治了十多年还没好，然后还执着的买这家伙的药？这是夸人还是损人呢？
<br/>　　……
<br/>　　而就在此时，众人中间一个年龄大概在五十岁左右的老者抬起双臂向下按了按：“安静，大家都安静一下！老夫有话要讲！”
<br/>　　“唰”的一下，就好像那老者有什么魔力一般，话一出口，场面瞬间就平静了下来，刚刚还在喊个不停的众人全都不说话了。
<br/>　　“诸位，俗话说的好，药医不死病，佛渡有缘人！老夫的药也是如此，不可能保治百病！但是，但是有一点老夫可以向你们保证，只要你们吃了我的药，不管你们原来的病有多重，一定可以让你们恢复如初。”
<br/>　　“好！”一片欢呼之声。
<br/>　　李承乾和孙思邈都已经听傻了，尴尬的对望着，心说这老货怎么说的都是废话呢，前面还说不能保治百病，但是后一句就成了只要有病吃了药就能好，这和包治百病有又有什么区别？
<br/>　　“而且……，这一次老夫特地去了一次蛮荒，哦，就是林邑，在那里收集了一批血参。”
<br/>　　“林邑大家都知道吧？那里可是地地道道的蛮荒之地，瘴气遍地，野兽横行，还有大量吃人的野人，也只有这样的地方才会有血参这种可以活死人，肉白骨的天材地宝。”
<br/>　　“其实说来老夫本来可以不去那里的，可是，可是老夫想到我大唐还有无数人病体缠身，还有无数人因为无药可医而绝望，老夫的良心就一直在受到谴责。”
<br/>　　“所以老夫去了，不光是去了，而且还带回了大量的血参，并以血参为主配制了一种名为百消丹的良药，只要是对症的病，只要连服五颗就能见效，连服十五颗就能让大家明显感觉到身体恢复，而如果你信得着老夫，连服三十颗，保证可以让你恢复如初，和正常人没有任何区别……。”
<br/>　　听到这里，李承乾已经全都明白了，敢情这货就特么是个骗子。
<br/>　　还特么活死人肉白骨，扯蛋还有这么扯的？三七，哦，也就是所谓的血参最大的功效就是止血镇痛好不好！
<br/>　　看着场中夸夸其谈的那老者，李承乾一时间脑中闪过一个念头，对着他喊道：“老神医，我这段时间后背总是痒痒，你那药治么？”
<br/>　　“治，这有什么不能治的，既然老夫的药叫做百消丹，止痒自然也是其功效的一种。”
<br/>　　“这都能治？不是挠挠就好了么？”
<br/>　　“挠挠只是治标，而不是治本，正所谓……”。
<br/>　　不等那杨常林把话说完，李承乾再次问道“不是，那你这个多少钱一颗啊？”
<br/>　　左右都是个骗子，李承乾当然不会听他的歪理斜说，之所以问来问去，不过就是想要逗逗他。
<br/>　　不过那杨常林哪里知道这些，在他看来这李承乾就是最好的免费托儿，于是想也不想就说道：“老夫这药虽然来的不容易，但大家生活的也都不容易，所以这药原本老夫是打算一贯钱一颗的，但现在百钱一颗，卖光为止。”
<br/>　　“好人啊，大好人啊！”还没等李承乾再说什么，下面一群知明就里的百姓就开始感概。
<br/>　　李承乾与孙思邈对视一眼，尴尬的笑了笑，双双又挤出了人群回到了外面的马车上。
<br/>　　没啥可看的了，一个骗子完全不值得他们去看。
<br/>　　不过话说回来，这事儿也不能就这么算了，以李承乾一肚子坏水的德性，怎么可能放过一个近在眼前的骗子？
<br/>　　于是乎在把老孙送回到医学院之后，李承乾又再次回到了西市。
<br/>　　……
<br/>　　还是原来的位置，还是原来的人，不过那些百姓已经散去了，百草行已经恢复了安静。
<br/>　　李承乾在小白和夜魅的陪同下拾阶而上，走进了店里。
<br/>　　“几位客人，请问看病么？”一个二十多岁的女子迎了上来，笑着问道。
<br/>　　“杨常林老师在么？在下有些顽疾，不知道是不是治的好，已经看了无数的医生，都没有什么好办法。”李承乾叹了口气说道。
<br/>　　“没问题的，杨常林老师正在后面休息，客人您可以稍等片刻。”那女子先是安慰了李承乾一下，然后便去里面端出热茶，放到他的身旁。
