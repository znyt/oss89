　　数十年前，帝释天就调查过海族的信息，在他的心目中，都是些不见棺材不落泪的野蛮人，现在居然跑来打搅自己的计划，简直该死。
<br/>　　“主人英明，我们现在是继续打听呢，还是向神龙岛内部前进呢？”
<br/>　　“既来之，则安之，我们继续前进，找寻神龙所在，至于海族中人，不出现也罢，若是敢来打搅本座大事，定让他们有来无回。”
<br/>　　话语刚落，慕清霄一行人就朝着神龙岛内部前进，当走到一片茂密的丛林中时，狂风四起，异变突生。
<br/>　　帝释天突然抬起头来，望着瞬间昏沉下来的天空，以及缺失一角的太阳，疯狂的大笑起来：“天狗食日，真是惊瑞之象啊......”
<br/>　　伴随着帝释天的狂笑声，四周狂风四起，山体不断发出剧烈的轰鸣声，
<br/>　　旋即，整座岛屿开始地动山摇起来，地面出现巨大的裂缝，建筑和树木开始倒塌，神龙岛深处传来一阵惊天动地的龙鸣声，震的人耳膜生疼。
<br/>　　天狗食日转眼即逝，待龙鸣声消散，岛屿的地面就窜出一条长达数百米的苍龙，翱翔于天际，带着龙鸣，以极快的速度穿梭于整个神龙岛。
<br/>　　“惊瑞之期，苍龙现世。”
<br/>　　抬头望着盘旋于高空的苍龙，慕清霄神色震撼，真正的神龙，与华夏传说中的神龙形象完全一致，比起凤凰带给他的震撼，有过之而无不及。
<br/>　　等待半年的时间，终于来临，慕清霄虽表面平静，但内心却无比的激动，龙元就在眼前。
<br/>　　天上的苍龙，头似驼，角似鹿，眼似兔，耳似牛，项似蛇，腹似蜃，鳞似鲤，爪似鹰，掌似虎，其背有八十一鳞，具九九阳数，其声如戛铜盘。
<br/>　　口旁有须髯，颔下有明珠，喉下有逆鳞，头上有博山，又名尺木，龙无尺木不能升天。
<br/>　　“快点走啊，还愣着干什么！”
<br/>　　终于见到神龙了，自己长生不老的愿望，终于能实现了，帝释天狂笑声停止，激动的大喝起来。
<br/>　　众人不再犹豫，身影闪烁，连忙朝岛屿东部的石谷冲去。
<br/>　　神龙岛一行，其中皆是高手中的高手，速度自然是快若闪电，仅仅两分钟的时间，就抵达神龙岛东边的石谷。
<br/>　　当帝释天等人来到石谷，苍龙却早已失去了踪影，只剩下面前一道深不见底的深渊，苍龙就是在这里消失，想必就在这深渊之下。
<br/>　　这正是所谓的飞龙离天，随云入渊，潜龙在渊，随云上天。
<br/>　　步惊云等人手握神兵，警惕着四周，丝毫不敢大意，毕竟苍龙带给他们的震撼，实在是太大。
<br/>　　还有刚刚的那股气息，简直恐怖。
<br/>　　慕清霄并未出过剑，而步惊云手握赤霄剑，怀空的天罪，破军的白鹤剑与九黎刀，剑晨的英雄剑，皇影的惊寂刀，断浪手握的则是一柄无名之剑。
<br/>　　这时，步惊云等人已经来到万丈深渊旁，神色凝重的望着看不见底的深渊。
<br/>　　断浪开口说道：“苍龙肯定在深渊下，我们在这里空等也不是办法，我们用剑气，将它逼出来。”
<br/>　　闻言，步惊云等人相视一眼，这也是最好的办法，毕竟深渊万丈，谁都不敢轻举妄动。
<br/>　　当剑气消散，深渊下根本没有传来丝毫的动静，可见下面的深度根本不是剑气能够抵达的。
<br/>　　“可能这招也无法奏效，你们在这里等我，我当诱饵将它给引出来。”
<br/>　　话语刚落，步惊云毫不犹豫，纵身跃下深渊，瞬间就不见了踪影。
<br/>　　慕清霄面无表情的站在一旁，熟知剧情的他，清楚神龙即将现身，已经开始调动灵气，石谷山巅，帝释天饶有兴致的望着这一幕。
<br/>　　“昂昂昂......！”
<br/>　　霎时间，深渊底部发出一阵咆哮声，苍龙带着数百米的身躯，冲上云霄，慕清霄自然不会放过这样的机会，阴阳灵气聚于掌心，一掌拍出。
<br/>　　天山阴阳掌带着巨大的力量，直接轰在苍龙身上，破军等人也不甘落后。
<br/>　　但是，连续几招下来，苍龙身上的鳞甲依旧散发着刺眼的光亮，根本没有一丝受损的痕迹，百米身躯盘旋于天际，俯视着地面这群攻击自己的蝼蚁。
<br/>　　苍龙赤红色的龙眸盯着慕清霄等人，其中嗜血的光芒，毫不掩饰。
<br/>　　感受到上空传来的压迫感，慕清霄瞳孔一缩，心脏狂跳，有种骂娘的冲动，这股气息，根本就不是金丹九层巅峰，而是半步元婴！
<br/>　　心弦紧绷的同时，他也不忘观察苍龙赤红色的龙眸。
<br/>　　顿时发现，它好像并没有开启灵智，否则，以它半步元婴的修为，一尾巴都能将在场所有人砸成肉泥！
<br/>　　苍龙目光死死盯着下方，慕清霄并未暴露气息，而是调动九牛一毛的阴阳灵气，汇聚出一柄虚幻而真实的长剑，随时准备攻击。
<br/>　　山巅上，帝释天望着苍龙，寒冰面具下的眼神中，满是激动之色：“畜生，你终于现身了。”
<br/>　　“昂昂昂......！”
<br/>　　忽然，上空的苍龙发出一连串的龙吟，直接低头俯冲而下，灵活无比，速度奇快，直奔慕清霄等人。
<br/>　　帝释天依旧站在山巅上观望，而怀空等人已经手握神兵，纷纷合力抗击苍龙。
<br/>　　“轰隆隆......！”
<br/>　　苍龙带着百米身躯，以及锋利的龙爪直接碾压而来，所过之处巨石炸裂，地面都被压出一条深深的沟渠，四只龙爪带着无比强劲的力量，直接横扫而过。
<br/>　　慕清霄身形一闪，巧妙的躲过苍龙的利爪，由阴阳灵气所汇聚而成的长剑，直接劈在苍龙的身上，最终只留下一条浅白色的伤痕。
<br/>　　慕清霄修炼凝气化剑，已经有很长一段时间，几乎是信手沾来，虽然攻击力不足魂剑本体，可威力也不弱，最起码不弱于断浪手中的无名长剑。
<br/>　　然而，如此锋利的剑都无法伤它分毫，苍龙不愧是苍龙，虽然没有开启灵智，但肉身却不是一般的强大，若是开启灵智，这里的人根本不够它杀。
<br/>　　要知道，半步元婴已经迈入更高的领域，根本不是金丹能够相提并论的。
