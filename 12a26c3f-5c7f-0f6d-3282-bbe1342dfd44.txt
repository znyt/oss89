　　苍穹神宫，一座阁楼之中。
<br/>　　苏莫端坐在主座上，他的下首两侧，则端坐着苏洪，以及帝释、海明江、东海龙王、钱副阁主等几名苍穹神宫的高层。
<br/>　　“莫儿，你可是把我们吓死了，现在外界到处都在传言，说是你已经死了！”苏洪满脸后怕的说道。
<br/>　　现在别说是在外面的城池，就算是在途山镇，关于苏莫的传言，都是经久不绝。
<br/>　　什么真圣榜第一，妖孽天才苏莫，在真圣界大放异彩，强势打败巫族妖孽天才巫殇、巫人王和巨灵族巨摩，已经在武帝境之中无敌。
<br/>　　但是，苏莫自寻死路，不仅将巫殇和巫人王击杀，还大肆辱骂天冥星巫族一脉的大祭司，最终被大祭司灭杀，一代妖孽，黯然陨落云云。
<br/>　　总之，现在外界关于苏莫的消息，简直是铺天盖地，充斥在任何一个角落。
<br/>　　目前，整个苍穹神宫的人都慌了神，百万弟子都以为苏莫死了，一下子失去了主心骨。
<br/>　　还有大量的弟子扬言，要杀去巫族，前去为苏莫报仇。
<br/>　　若不是宫中下了严令，严禁任何人离开，怕是很多人都要去找巫族拼命了。
<br/>　　苏莫在一众苍穹神宫弟子的心中，地位无可取代，苏莫死了，这是所有人都无法接受的事情。
<br/>　　“是啊！苏莫，这到底是怎么回事？”帝释疑惑的问道。
<br/>　　“宫主，巫族势大，你以后还要当心啊！”海明江满脸的凝重。
<br/>　　苏莫闻言一笑，摆手道：“大家放心，是我瞒过了巫族，巫族之人已经认为我死了，以后不会有什么危险了！”
<br/>　　“那就好！”众人点了点头，若是如此的话，的确是安全了很多。
<br/>　　随即，苏莫略一沉吟，沉声道：“关于我是苍穹神宫宫主的消息，一定要严密的封锁，不能有一丝一毫的泄露！”
<br/>　　“宫主放心吧，我等早已交代了下去，门下所有弟子都会守口如瓶！”海明江道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即看向帝释，道：“不过为了安全起见，免得哪个弟子不小心泄露了消息，从今以后，我就暂离宫主之位，由岳父你担任宫主吧！”
<br/>　　“这……不太好吧？”
<br/>　　帝释闻言一怔，面色略显迟疑，担任宫主他没有意见，但是他的实力在苍穹神宫也不是多强，海明江、东海龙王、钱副阁主等人，实力都不比他弱，怕是别人会有意见。
<br/>　　“你们都同意吗？”苏莫看向其他几人。
<br/>　　“我们没有意见！”众人齐齐开口，既然苏莫说了，他们自然没有意见。
<br/>　　再说了，苏莫就算不做宫主，但在全宫所有人的心中，他依旧是真正的宫主，永远不会改变。
<br/>　　“好，既然都没有意见，从今以后，岳父担任苍穹神宫宫主！”苏莫郑重的宣布道。
<br/>　　“是！”众人齐齐应声。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即目光环视几人，沉声说道：“各位，现在有一个任务需要你们安排人手去办！”
<br/>　　“什么事？宫主你尽管吩咐！”海明江说道。
<br/>　　“是这样的，我在神武学府有两位朋友，叫聂乾和韩盈，他们被学府执法堂关押，需要你们派一些人，前去打探情况！”苏莫道。
<br/>　　“苏莫，那神武学府，不是其内弟子的话，也不好进去吧？”帝释皱眉说道。
<br/>　　“无需进去，只要派些人手到神武学府附近的城池，发现有神武学府的弟子，可以通过那些人了解消息！”苏莫告诫道，他现在不想太接近神武学府，免得被人认出来。
<br/>　　他虽然改变了面貌，但见过他的武尊境以上武者，还是很容易识破他的身份。
<br/>　　“好，我这就去安排！”海明江立刻起身，二话没说就欲前去安排。
<br/>　　“海老且慢！”
<br/>　　苏莫急忙喊住了对方，告诫道：“记住，告诉所有人，一定要谨慎，不要泄露丝毫和我有关的事情！”
<br/>　　“明白！”海明江点了点头，随即便告辞离去。
<br/>　　随后，苏莫又和其他几人聊了片刻，详细的了解了一番苍穹神宫的发展。
<br/>　　苍穹神宫的发展，可谓是一日千里，不仅一众高层修为不断飙升，门下弟子修为更是一日千里。
<br/>　　像帝释，钱副阁主等人，修为都达到了武帝境三重境界，就连苏洪都达到了真玄境三重境界。
<br/>　　要知道，在一年之前，苏洪还只是真罡境八重的境界，短短一年的时间，连续飙升四重境界，其中还要跨越一个大境界。
<br/>　　这是因为现在的苏洪，已经是双生武魂的武者，而且其中一种还是战魂。
<br/>　　而在门下的弟子之中，修为达到武王乃至武皇的武者，多不胜数，武帝又增添了三十多位。
<br/>　　可以说，现在的苍穹神宫，比刚离开苍穹世界之时，强大了不知道多少倍。
<br/>　　只不过，让苏莫忧心的是，苍穹神宫的资源严重不足。
<br/>　　他上次留下的灵石矿脉，已经用完了。
<br/>　　没有办法，苍穹神宫可是有百万之众，区区一座灵石矿脉，根本支撑不了多久。
<br/>　　而苍穹神宫的人，平常多是努力的修炼，能赚取的灵石或者灵晶，实在是有限。
<br/>　　好在梁通的炼器堂，发展的不错，有炼器师八百多人，每日都能炼制出大量的神兵利器。
<br/>　　苍穹神宫为此，还在附近的几个城池开设了店铺，专门售卖兵器，为苍穹神宫赚取灵石和灵晶。
<br/>　　只不过，炼器堂炼制不出高等级的兵器，所能带来的收益，非常的有限，根本不足以支撑苍穹神宫巨大的消耗。
<br/>　　待苏洪、帝释等人全部离开之后，苏莫独自一个人坐在阁楼中。
<br/>　　“哎！”
<br/>　　一声长叹声从苏莫的口中响起，声音中充满了无奈与疲累，一切还是要靠他。
<br/>　　想要让苍穹神宫发展壮大，仅让苍穹神宫自行发展，实在是太慢了。
<br/>　　苏莫感觉他肩膀上的担子很重，苍穹神宫要发展、宏青璇要去寻找，还要找轮回之门救治芊寻月，他自己也要快速强大起来。
<br/>　　每一件事都不能落下，一切的一切，都让他放松不下来，感觉心力都有些交瘁了。
<br/>　　但是，他却不想舍下任何一样，这或许就是性格吧！
<br/>　　性格决定了他的人生！
<br/>　　注定了他这一生都要被牵绊！
<br/>　　被身边的人牵绊，被身边的事牵绊，永远都无法摆脱，除非——他能走到巅峰！
<br/>　　他若是能抛弃一切，就能潇洒的畅游星空，去追寻自己人生，但是，他却无法做到。
<br/>　　“分身！”良久之后，苏莫的眼眸中精光一闪，想到了比较好的办法。
