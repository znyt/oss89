　　之前的所有经历都只是十阶考核，不是楚涵的真正目的，哪怕连他自己都分不清，这次考核中的场景，到底是不是现实世界。
<br/>　　而且还是个限时考核！
<br/>　　也就是说远在阴阳谷冰湖的那处石壁，光线已经在渐渐变暗。
<br/>　　异度空间点楚涵又有些不确定，但他在最后一刻选择进入，也没有了退路可言！
<br/>　　深吸一口气让自己平静下来，忘却过去八个小时的经历，楚涵定定的看着自己四周。
<br/>　　他已经身处异度空间点！
<br/>　　从一开始的黑暗无声，渐渐的开始有声音传来，似乎还很多，就在楚涵的身边。
<br/>　　紧接着，眼前的场景开始由模糊到清晰，逐渐出现了一条大道，雪白透亮，楚涵自己就站在道路的中间，前后左右人来人往络绎不绝。
<br/>　　再次深吸一口气，楚涵强忍着怒骂考核石壁的冲动，开始用他平生最强的观察力洞悉四周。
<br/>　　恩，是个异世界。
<br/>　　楚涵立马下了结论，这里的人衣着风格与地球上下五千年的都不同，也跟珈蓝国的不同，这里的人似乎很喜欢白色，很多人衣服松松垮垮的就用一条白布搭着，随意的用一个别针扣住。
<br/>　　女人穿着很好看，甚至还带着妩媚感，但男人穿这种衣服就很古怪了。
<br/>　　紧接着，楚涵发现自己也穿着这种衣着……
<br/>　　难怪站在这里半天也没人注意他，早在进入最终异度空间点考核的时候，考核石壁就被他的衣着换了！
<br/>　　真是……丑的掉渣！
<br/>　　楚涵看向建筑，又看向河堤和船只，这里的文化似乎很复杂，什么都有。
<br/>　　但最吸引楚涵的，还是远处天空上的一艘飞船！
<br/>　　“还真是什么都有……”楚涵有些被吓到的喃喃自语。
<br/>　　这艘飞船飞船巨大，也幸亏距离地面很远，否则都会将这里的光线完全挡住。
<br/>　　楚涵是十阶新人类，还是很强的那种十阶，所以倒是可以看得见飞船的大致情况。飞船上连着一根极长的天梯，与地面下方的一处高台相接，有人在其上行走。
<br/>　　这应该就是飞船的停驻点，类似于码头那种。
<br/>　　收敛了情绪，楚涵大致逛了一圈，没有发现任何可进行的事件。
<br/>　　异度空间点在九阶之前都是打架，打的你死我活，杀的天荒地老，楚涵都已经习惯这种走在刀尖上的模式。
<br/>　　眼下他变强了，却忽然变成了需要自行探索的最终任务，让楚涵实在有力无处使，又二丈摸不着头脑。
<br/>　　他是谁？他在哪？他要干啥？
<br/>　　总该有个指示吧！
<br/>　　无奈的楚涵看了眼回炉系统，这一看就让他震惊不已。
<br/>　　珈蓝国他去过，而且还得到了天运国这一地点的信息，只是一直没有解锁。
<br/>　　但是现在，回炉系统上竟然显示着自己就在天运国！
<br/>　　这里是天运国？
<br/>　　楚涵惊讶的瞪大眼睛，而后继续迷茫。
<br/>　　若他记得没错，与天运国有关的唯一线索，就是一个叫做天女的人物，难道他的最终考核是找天女？
<br/>　　楚涵抬起脚，开始漫无目的的朝前行走，也不自觉的靠着飞船的方向前行。
<br/>　　“喂！还不赶快过来搬东西！”这时候一个呵斥声响起，直冲着楚涵：“耽误了飞船起飞时间有你好看！”
<br/>　　楚涵看着旁边，正有大批人将一个个箱子搬上高台，也就是楚涵称之为的飞船码头。同时他还发现一个细节，那就是似乎别针代表了身份。
<br/>　　自己身上的别针是白色，与来往正在搬东西的人群一样。
<br/>　　而眼前这个冲自己呵斥的人，别针是蓝色。
<br/>　　楚涵只是略微一顿，便抬脚搬起一个大箱子跟着人群走上高台，正好他想近距离看看那艘飞船。
<br/>　　时间好似很紧急，搬运东西的人一个个都赶集似得，但楚涵没有在他们的面上看到疲惫和排斥，每一个人都兴奋无比的样子。
<br/>　　干苦力还这么高兴？
<br/>　　不解间，楚涵也已经将一个大箱子一步步搬上了高台，随着大部队延着旋转阶梯爬了很久才搬上来。这些东西被放置在铁索上，一一运向飞船内部。
<br/>　　这处高台从远处看并不大，但一走进才发现广阔的很，也不知道造这样高又大的地方，要消耗多少人力财力。
<br/>　　前方的人在排着队不知道走向那里，楚涵也跟着一路观光。
<br/>　　这时候后方的一处升降梯亮起，透明色的玻璃罩中，几个明显身份不同的人走了出来，直径走向天梯。
<br/>　　他们要上飞船！
<br/>　　“是天女！”
<br/>　　“天女！”
<br/>　　人群欢呼起来，楚涵也朝那里望去，但却根本看不到天女的脸。
<br/>　　因为她被前后左右的人层层包围在中间，保护很好的样子，外圈的人又人高马大，完全将之挡住。
<br/>　　这一行人延着天梯直接走上飞船才停下，这时候也不知道谁在操作，天梯缓缓回缩至高台，最终完全与飞船脱轨。
<br/>　　“要起飞？”楚涵满是好奇。
<br/>　　当天梯收回完毕，飞船上的那行人也一字排开，天女站在了最中间，左右两边的人开始轮流讲话，每说完一句都引发了下方全体人的欢呼和鼓掌。
<br/>　　楚涵却在这一瞬间，望着天女的容貌震惊到呆住！
<br/>　　“我们天运国能有今天，都是天降神女，助我天运国能有朝一日探索更多的未知星域。天女是我天运国的最大依仗，也是我们天运国繁荣的象征！”
<br/>　　“现在我们天运国的飞船就要起航！去往刚出现在信号上的全新未知星球上！天女给这个星球命名为地球！”
<br/>　　“祝我们的旅途一切顺利！”
<br/>　　话落，震耳欲聋的鼓掌声响起在耳边。
<br/>　　紧接着，飞船上的人就一一走进舱室，飞船舱门开始闭合，一阵轰鸣声响起。
<br/>　　升空！
<br/>　　楚涵还在发愣，甚至还揉了好几次眼睛，直到飞船远去看不见，他都没能从震惊中回神。
<br/>　　天运国的天女，是白允儿？！
<br/>　　是白允儿没错，长相气质一毛一样，他不可能认错！
<br/>　　最关键的是刚刚那人说什么来着？
<br/>　　天女给星球命名为地球，他们这群要去地球？
<br/>　　白允儿在时空夹缝中，落在了天运国！
