        顾千凝顿时一脸懵：“星儿什么时候去勾搭明司寒呢？”
<br/>
<br/>        “勾搭司寒也就算了，还放司寒鸽子。我还真没想到，她连星儿居然这么渣！”赵子晋还滔滔不绝起来。
<br/>
<br/>        顾千凝皱眉道：“你到底在胡说八道什么啊？”“连星儿不是给司寒递了一份情书吗？自己有男朋友了，还向司寒表白，真渣！”赵子晋唾弃道。
<br/>
<br/>        顾千凝这才恍然大悟：“那封表白信，不是星儿写的。”
<br/>
<br/>        那天，她陪星儿去给明司寒递信，那信是星儿的高中同学朱小溪写的。
<br/>
<br/>        赵子晋却不以为然：“你是她好朋友，肯定帮她说话。信的署名，就是她连星儿的名字，还说不是她写的！”
<br/>
<br/>        “据我所知，星儿确实有喜欢的人，但她喜欢的人确实不是明司寒。但是，那封信，如果署名是星儿的话，我想，星儿一定是被她那个高中同学给坑害了！”顾千凝振振有词地说道。
<br/>
<br/>        赵子晋仍旧不信。
<br/>
<br/>        顾千凝忽然想到了什么，掏出手机，将拍下来的一篇写着纸上的小短文，拿给赵子晋看：“这是我为了抄星儿的作业，特意拍下来的，我把它转发给你，你拿回去要明司寒比对一下字迹，不就知道那封表白信到底是不是星儿写的了。”
<br/>
<br/>        赵子晋见顾千凝如此较真，一定是为了不想自己的朋友被人冤枉。
<br/>
<br/>        想到这里，赵子晋信了顾千凝的话，但是他也没拿顾千凝手机的这张照片。
<br/>
<br/>        “既然是个误会，那就让它继续误会着好了。”赵子晋耸了耸肩。
<br/>
<br/>        顾千凝不解道：“为什么啊？”
<br/>
<br/>        “连星儿有男朋友是这是事实吧？”赵子晋反问。
<br/>
<br/>        顾千凝点了点头，她只知道，星儿称呼她的男朋友叫“小纽扣”。
<br/>
<br/>        赵子晋接着说道：“她既然有男朋友了，那么总不可能让司寒去挖墙脚吧？所以，干脆让司寒误会着算了。这样一来，司寒过不了多久，就会不喜欢连星儿了。这叫，长痛不如短痛。”
<br/>
<br/>        “可是，这样对星儿好不公平哦！你们这样误会她是渣女。”顾千凝唉声叹气地说。
<br/>
<br/>        赵子晋摊了摊手：“这也没办法！不过，我知道就行了。”
<br/>
<br/>        “对了，明司寒他人呢？这段时间都没看到他了。”顾千凝又好奇地问。
<br/>
<br/>        赵子晋瘪了瘪嘴，回答道：“人家可是自己开公司的，哪像我们只能天天待学校里上课。”
<br/>
<br/>        “他这么厉害？！”顾千凝惊赞道。
<br/>
<br/>        赵子晋随之附在顾千凝的耳边，悄悄地说道：“你们女生应该知道明希哲吧？”
<br/>
<br/>        “当然知道！当红大明星，我偶像洛羽的死对头！”顾千凝义愤填膺道。
<br/>
<br/>        赵子晋嘴角微扬：“他是司寒的亲哥哥！”
<br/>
<br/>        “！！！”顾千凝顿时瞪大了眼睛。
<br/>
<br/>        “牛吧？”赵子晋笑道。
<br/>
<br/>        顾千凝猛点头，顺便调侃了一句：“怪不得星儿不喜欢明司寒。”
<br/>
<br/>        “你这话怎么说？”赵子晋一脸懵，
<br/>
<br/>        顾千凝会心一笑：“因为星儿喜欢的明星是洛羽啊！而明希哲是洛羽的死对头。所以，星儿对明司寒才会没兴趣。”
<br/>
<br/>        “爱屋及乌？”赵子晋若有所思起来。
<br/>
<br/>        顾千凝微笑着点了点头。
<br/>
<br/>        而这件事情，很快就经赵子晋之口传到了明司寒的耳中。
<br/>
<br/>        此刻，明司寒带着自己的助理，正在滨江城的傅氏集团总部，和傅夜沉谈电竞投资的项目。
<br/>
<br/>        但是他们做的企划案并未得到傅氏集团这边的认可，投资评估未通过后，傅夜沉也就拒绝了明司寒的来意。
<br/>
<br/>        刚刚工作上的事情，明司寒的心情上受到了影响，结果赵子晋发过来的简讯，让他的心情更加的糟糕。
<br/>
<br/>        连星儿真的是洛羽的粉丝这么简单吗？
<br/>
<br/>        不经意间，明司寒猛然想起了之前的那些画面。
<br/>
<br/>        由此看来，连星儿的男朋友不是别人，正是洛羽。
<br/>
<br/>        二哥明希哲拜托他查的事情也不需要再查了，洛羽是傅夜沉的准女婿，傅夜沉帮洛羽也是理所当然。
<br/>
<br/>        离开傅氏集团后，明司寒让助理带着团队的人先回去，他想一个人去散散心。
<br/>
<br/>        其实，他不是那种轻易就动情的男生。
<br/>
<br/>        只是连星儿于他而言，有太多的独一无二。
<br/>
<br/>        他去了自己的网咖，登录游戏后，发现排行榜上那个“Star大小姐”是在线的。

<br/>        明司寒虽然删除了连星儿的好友还将她拉入的黑名单，但是，排行榜上的名次，他还是能看到。

<br/>        他“余寒犹厉”，永远都是排行榜的第一位。

<br/>        另一边，连星儿坐在电脑桌前，看着电脑屏幕，同样在关注排行榜。
