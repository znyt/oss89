　　　　“何止是蕴含一定的药性”苏羽眼睛明亮着：“我甚至觉得这是什么神丹妙药了，嚼一口浑身充满力量，精神百倍”
<br/>　　苏羽好不吝啬自己的夸奖：“没想到涂兄弟一表人才，还精通医理，将一块小小的蛇皮，制作成神药”
<br/>　　都说蛇浑身是宝，蛇皮更是一味很重要的中药，可以治疗好多的病症，也是人们常用的一味中药，但这蛇皮的药性未免强的有点匪夷所思。
<br/>　　苏羽可不是一般人，能将他都惊到，赞不绝口，可不寻常，可见他对涂小安的蛇皮是何等的推崇备至。
<br/>　　“言过了，苏老板想要，我送你几片便是”涂小安现在压根不太在意第一次蜕下来的蛇皮。
<br/>　　他手中第二次的蛇皮效果更是惊人。
<br/>　　而且将来他随着实力的提升，还会不断的脱皮，蛇皮用都用不完。
<br/>　　“送？那么贵重的东西，苏某可受之不起”苏羽顿时对涂小安的好感度蹭蹭的往上涨。
<br/>　　那么好的神药，说送就送，这是何等的豪气跟魄力。
<br/>　　“苏老板你有点吹的太过了吧”李清照见苏羽将区区的蛇皮说的此物只有天上有似的，忍不住好奇心的从他手中夺来。
<br/>　　然后放到嘴巴中咬了一口，这一口咬下去，却见苏羽的脸颊微微抽搐起来，他既然在心疼。
<br/>　　“咦...”
<br/>　　李清照一口咬下去，顿时有明显的效果，衬得皮肤白如雪，微微散发光泽。
<br/>　　“这蛇皮我也要”
<br/>　　苏羽诚不欺她。
<br/>　　倏然，李清照一双美眸烙印在涂小安的身上，一副你不给我我就跟你誓不罢休的样子。
<br/>　　涂小安一个白眼丢过去，这丫头什么都要凑热闹。
<br/>　　从裤兜中装模作样的拿出了二十片蛇皮，苏羽十片，李清照同样十片。
<br/>　　李清照甜美的嫣然一笑，清雅的不可方物，摸了摸自己的小脸蛋，小嘴还呢喃着：“这蛇皮好像还可以美容养颜，嘻嘻”
<br/>　　她的不客气，让苏羽有点小尴尬，他可不能也同样不客气，他潜意识的将李清照当成了涂小安的女朋友。
<br/>　　女朋友朝自己的男友要东西，是理所应当，他不过是一个外人。
<br/>　　有心想推脱，可这十片晶莹的蛇皮在前，苏羽实在从心底不想拒绝。
<br/>　　他刚才才嚼了一小口，就能有明显的效果，要是将这片蛇皮吃进去，他都感觉自己以后能百病不侵了。
<br/>　　犹豫了下，苏羽从西装内侧，拿出一张支票，干脆利落的从上面填上了五百万，然后将这张五百万的支配划到涂小安的面前：“之前涂兄弟救在下的龙蛇，我还没感谢，现在岂敢无功又受禄”
<br/>　　涂小安看了眼支票，上面七位数，微微有点诧异，这一出手就五百万，如果是大手笔呀。
<br/>　　不过，这五百万涂小安受的起，他的蛇皮有病治病，无病可强身，除非是一些非常罕见的绝症，否则都没什么大问题。
<br/>　　这十片蛇皮，真正的价值绝对不会逊色五百万。
<br/>　　只是他自己现在没将这第一次脱下来的蛇皮当一回事而已。
<br/>　　“涂兄弟，莫非是苏羽小气了，让你嫌少？”
<br/>　　苏羽看到涂小安只是看了一眼支配，便无动于衷，心中有点忐忑，想着要不要将支票在拿回来，重新改一下上面的数字。
<br/>　　涂小安闻言，看了苏羽一眼，伸出手将支票收了起来，笑着说：“那我就不跟苏老板客气了”
<br/>　　有钱不拿是傻子，况且涂小安不是白拿，先是救了苏羽的龙蛇，现在送上十片蛇皮，这五百万，他心安理得。
<br/>　　况且从白镇离开，他身上到也没带什么钱，不好意思从老姐那里拿钱。
<br/>　　“是苏某占便宜了，你这个朋友，我交定了”
<br/>　　李清照在旁边看的目瞪口呆，这区区十张蛇皮对方直接给了五百万，那么她这十张蛇皮莫非也要出五百万不成？
<br/>　　一时之间，李清照都想将蛇皮还给涂小安了，太贵了，贵的吓人。
<br/>　　正当李清照左右为难的时候，手机一条微信显现出现，她打开看了一眼，顿惊，连忙看向涂小安：“不好了，苏菲老师给我发信息过来，说校园又出现了群蛇，而且数量不计其数，整个学校都乱套了”
<br/>　　嗯？
<br/>　　涂小安闻言便蹙眉，看来事情果然没有那么简单。
<br/>　　他本来就没想明白之前的群蛇是为何出现，现在晚上又出现了，那么一定有什么东西将这些蛇给吸引过来。
<br/>　　这就好比当初蛇谷中，蛇莓散发的异香，将几十公里外的毒蛇全部给吸引过去。
<br/>　　难道这温城大学，也有类似的宝物吸引蛇类疯狂吗。
<br/>　　“走，我们回去看看”
<br/>　　既然是蛇事件，涂小安自然不会无动于衷，他到真想知道，到底是因为什么。
<br/>　　“你们是温城大学的学生？”苏羽忽然看着涂小安跟李清照。
<br/>　　“我不是，她是”
<br/>　　苏羽意外道：“李姑娘说的苏菲老师，正是在下的大姐，在温城学校任教生物学”
<br/>　　李清照惊讶：“不会吧，苏菲老师是苏老板的姐姐呀，这还真巧了”
<br/>　　有时候这个世界就是那么的小。
<br/>　　“苏老板，这顿饭，我们改天在吃，先走了”
<br/>　　涂小安惦记着校园群蛇，跟苏羽打了声招呼，就拉着李清照离开味鲜阁。
<br/>　　不到二十分钟，涂小安来到了温城大学的校门，此时里面的人络绎不绝的慌乱跑了出来，一个个脸上心惊胆战，仿佛逃过一劫。
<br/>　　校园里的学生都跑出来了？
<br/>　　真的到了这种地步了吗。
<br/>　　要知道下午校园出现上千多条蛇，也仅仅引发了恐慌，还没到让人落荒而逃的地步。
<br/>　　此时，十几辆警车停在校门口，绝大部分的警员都进去将一些被蛇困住的学生给安全的报送出来。
<br/>　　“小安哥，苏菲老师说里面现在遍地都是蛇，简直成了一个蛇窟”
<br/>　　李清照罕见的没有露出兴奋的表情，因为整个校园被群蛇笼罩，所有人都心惊胆战，这是非常严重的大事件了。
<br/>　　绝对不是闹着玩。
<br/>　　如果说下午是上千条毒蛇，那么此时校园内，起码是上万，十几万，乃至于更多。
<br/>　　乱了，彻底的乱了，人人自危，你站在校门口，都能远远看到里面狰狞可怖的蛇影子。
