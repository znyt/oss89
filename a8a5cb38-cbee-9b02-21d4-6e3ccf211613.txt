　　随着和修政的命令，所有的搜查官都忙碌了起来，不过十几分钟的时间，月山家族的所有相关资料都发送到了指挥舱的电脑上。
<br/>　　“和修局长，问出来了，月山习藏匿的地方，是月山集团的附属公司美林传媒的总部大楼，今晚会有直升机，将月山习送往国外。”
<br/>　　看着面前已经被摧残的不成样子的喰种，有马贵将掏出对讲机，将所获得的情报如实禀报上去。
<br/>　　“美林传媒么，我知道了，我会下发一个坐标，所有人立即行动，前往坐标指示的地方。”坐在直升机的指挥舱里，和修政查看着电脑上的资料，吩咐了下去。
<br/>　　在任何国家，直升机出境都是需要很复杂的手续，短时间内，根本不可能处理好。
<br/>　　如果没有相应手续，任何出境的直升机都会被逼落下来，如果反抗，更是会被国家的防空武器击落，不可能有机会逃离的。
<br/>　　但月山家依旧做出了这样的撤离选择，也就是说，月山家的喰种，很早就做好了撤离的准备，可是，他们CCG明明在今天才开始准备行动的。
<br/>　　这么短时间，都能让月山家族反应过来，唯一的解释，是CCG内部，出了内鬼。
<br/>　　脸上带着一丝晦暗，和修政心中决定，等这次行动结束，是要彻查一下这件事情了，他很想知道，是什么人会和喰种勾结到一起。
<br/>　　……
<br/>　　13区，美林传媒大楼，周遭所有的建筑都是一片黑暗，只有他们大楼内灯火通明。
<br/>　　此时一楼的大厅中，一群带着红色面具的喰种集结在这里，看着刚刚走进来的月山习等人。
<br/>　　“我们都是月山集团的喰种，听闻少爷有难，就迅速赶往了这里。”一个身穿高级西服的矮胖中年人，站在队伍前方，一脸和气的说道。
<br/>　　剩余几人也是满脸微笑，看不见有任何紧张，他们每一个人，在人类社会都是各种行业内的精英，也是月山集团之所以会发展到如今地步的根本所在。
<br/>　　“直升机还要有一段时间才能到达，我们会负责保护少爷的安全。”矮胖中年人的身旁，一个带着眼睛的瘦弱男子面带微笑，向月山习微微躬了躬身。
<br/>　　似乎，他们即将面对的，并不是一场你死我活的战斗，所有人的脸上，都带着极为绅士的笑容。
<br/>　　看着面前的一众长辈，月山习嘴唇微微动了几下，神色有些感动，同样也对这些长辈感到惋惜。
<br/>　　他知道自己应该去做什么，他的任务，就是将家族的未来给传承下去。
<br/>　　他必须要逃走，就算不是为了自己，也要想尽办法的拼命生存下去，为了让他活着，牺牲的已经足够多了。
<br/>　　“叔叔们，谢谢，我一定会逃出去，并且，会招呼好你们的家人。”深深鞠了一躬，月山习按下了身边电梯的按钮。
<br/>　　能留给他的时间，已经不多了，CCG的那些家伙，估计很快就会追来。
<br/>　　跟在矮胖中年人的身后，刘哔脸上带着和周围人群一模一样的红色面具，按照分配守护在自己负责的区域里。
<br/>　　因为无法预料这次计划的危险程度，什造被他安置在了附近的一家住户中，那里，相对来说很安全。
<br/>　　凭借他自身SSS级别的力量，再加上恶魔之翼和赫者化的加成，解决那些青铜树和月山家的喰种并不困难。
<br/>　　为了避免引起可能存在的监视，他特地花费积分，从商城内兑换了一张结界卷轴，将整个别墅都给封锁了起来。
<br/>　　这种结界卷轴并没有太大用处，唯一的功能，就是隔离声音以及能量波动，用在这种时候，倒也颇为合适。
<br/>　　战斗结束之后，他就带着什造悄无声息的逃离了出来，安置好什造，刘哔就跟随自己提前留下的线索，找到了月山习藏匿的地方。
<br/>　　本来对这个世界并不熟悉的刘哔，想找到刻意隐匿的月山习并不容易，这次之所以能追踪到，也是运气使然。
<br/>　　先前月山习前往库克利亚接他的时候，他从轿车内熄灭了一个烟头，那个烟头上面，留下了他身上的剩余的一缕鬼气。
<br/>　　这个世界中是没有鬼气存在的，至少这个国家没有，按照留下的这一缕鬼气，他很快就追踪到了月山习的位置。
<br/>　　但想到月山家现在已经对自己起了戒心，刘哔也没有贸然露面，而是找机会击杀了一名上厕所的喰种，混入了月山集团的队伍中。
<br/>　　接下来，月山习上楼，领头的那几名喰种安排好人员的分配之后，也结伴而去。
<br/>　　看了一眼身边和自己临时搭档的喰种，刘哔找了个借口离开，暗中查看了一下几名领头喰种的去向。
<br/>　　……
<br/>　　“叮~”
<br/>　　电梯缓缓停留在了15楼，三名看起来和常人无异的喰种从电梯内走出，交谈着往前方走去。
<br/>　　几乎是紧随其后，步行楼道的出口处，显出了刘哔的身影，以他的速度，是不可能跟丢这几个人的。
<br/>　　目送着前方三人进了一间屋子，刘哔查看了一下四周，迅速跟了上去。
<br/>　　这似乎是一间监控室，四周摆满了屏幕，上面显示着一个个的画面，都是大厦内的场景。
<br/>　　“哎呀，我只是在习少爷年幼的时候见过他一次，没想到已经长成了这样的猛男。”之前说话的那个矮胖中年人，随手关上了身后的屋门，摊开双手说道。
<br/>　　大难临头，他似乎一点都没有察觉到，脸上的笑容很是灿烂。
<br/>　　随手在面前屏幕上操作了几下，带着眼睛的瘦弱男子转过身来，语气轻松的回答道：“是啊，真是感慨万分，不知不觉，我们都已经老了呢。”
<br/>　　远处身穿紫色西装的男子，刚刚沏好了一杯咖啡，嘟囔着走上了前来：“真想让他做我们那里的模特啊，他做模特的话，肯定会大获成功的。”
<br/>　　“喰种的广告牌吗，不错的注意。”漫不经心的附和了一句，中年男子转过身来，面对着监控室的屏幕：“这些都是为了观母大人，我们将在这里报答月山家一族的恩情。”
<br/>　　注视着屏幕上的画面，中年男子收起轻松的笑容，屏幕上，大厦已经被密密麻麻的车辆包围。
