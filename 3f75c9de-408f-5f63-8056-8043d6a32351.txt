问号的。<br/>　　<br/>　　第一个病人，是陈默以前住在这里的客厅位置。现在隔离出来一个房间，里面的家具都搬走了，现在就是一张床和一个床头柜，还有一个衣柜和一个书柜，其他就没有什么家具了。<br/>　　<br/>　　房间按照标准的医院格局装修，以蓝色和白色为主。陈默进来后，就发现病人是其中四个男士的其中一个，三十来岁的他，此刻躺在病床，打着点滴和吸着氧气，另外还带着监护。这些医疗设备，都是随着病人一起过来的，陈默这里是没有的。<br/>　　<br/>　　病人看着陈默一行进来后，却只是扫了一眼，然后转头并没有说什么，神情有些冷漠。<br/>　　<br/>　　“这位叫世兴文，这是他的病历！”常文斌见到世兴文的表情，顿时有些尴尬，只能出面自己来解释一下。<br/>　　<br/>　　其实在早上的时候，这些病人对于陈默也都不是十分热情，主要是看在袁若珊的面子上，才和他打了个招呼。当然还不包括昏迷的，所以此刻世兴文的表情，陈默并没有介意。<br/>　　<br/>　　“陈医生，你也知道，因为世兴文的伤一直拖了一年多，都没有好。所以他的心情有些不好，你多担待些！”<br/>　　<br/>　　李普河害怕陈默生气，所以上前赶紧给陈默耳边小声说道，也是避免医生有怨气，那么治疗病人就不会怎么用心。<br/>　　<br/>　　“别再一旁唧唧歪歪！我受伤不假，但是没有伤到耳朵！”<br/>　　<br/>　　还没有等陈默说什么呢，躺在病床~上的世兴文转过头来，对着李普河就怼了一嘴。<br/>　　<br/>　　“文哥，你……？”李普河看见世兴文转过头了，赶紧上前安慰道，但是却一时不知道说什么。<br/>　　<br/>　　“你就是这里的医生？”<br/>　　<br/>　　“不错！”陈默回答。<br/>　　<br/>　　“还有没有其他医生了？”<br/>　　<br/>　　“没有了，这里就我一个医生！”陈默再次回答道。<br/>　　<br/>　　“局里是什么意思？是不是看我躺了一年多，就放弃了？来到这个地方，既然找这么年轻的人给我们治病？给我那电话来，我倒是要问问领导的意思，真的要是放弃我们，那我也不埋怨什么，送我回家好了！”世兴文听完陈默的回答后，心里顿时是失望透顶！<br/>　　<br/>　　“说话声音这么大，看来你还挺有精神的么？”陈默的声音有些平淡的问道。<br/>　　<br/>　　“我有没有精神与你何干？就你这么年轻的医生，还是不要BB的好！”<br/>　　<br/>　　这话说的，有些冲！世兴文其实并不是这样的人，但是他没有想到局里就像是丢垃圾一样，将自己几个人转到了这里，看着这个年轻的不靠谱的医生，心中要是没有怨气才怪！<br/>　　<br/>　　他又不是那几个昏迷不醒的人，虽然受伤不能下床，但是脑袋还是好好的，所以一想就认为，这是看自己不能站起来了，因此就想着法子的将自己这些人丢弃。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
