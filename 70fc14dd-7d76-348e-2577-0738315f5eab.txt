　　“随老夫来吧！”
<br/>　　火元圣师招呼了苏莫一声，随即脚步抬起，虚空踏步，大步向神武峰而去。
<br/>　　苏莫深深的吸了口气，兵来将挡水来土掩，既然火元圣师说没事，应该不会有太大的问题。
<br/>　　火元圣师应该不会害他，不然，以对方的修为和身份，何需这么麻烦！
<br/>　　随即，他身形一动，立刻跟着火元圣师，向神武峰飞去。
<br/>　　神武峰，是内府的主峰，也是府主居住的山峰，位于内府的中央，是整个内府这片虚幻的星空中最为高大雄伟的山峰。
<br/>　　此峰高达万丈不止，整个山峰上郁郁葱葱，长满了各种植被。
<br/>　　神武峰距离火元峰不远，两个呼吸之间，火元圣师和苏莫便来到了神武峰上。
<br/>　　神武峰上建筑不多，只有寥寥几座宫殿，其中一座非常的宏伟，殿门之上金色的牌匾上‘武殿’两个大字龙飞凤舞，气势摄人。
<br/>　　苏莫跟着火元圣师，身形直接落在了武殿的大门前。
<br/>　　就在这一瞬间，他感觉到凌厉的目光落在了自己的身上，这目光中蕴含的杀机，让他身躯瞬间冰冷，全身血液都瞬间凝固了。
<br/>　　而在同一时刻，苏莫也看到发出目光之人，正是天冥星巫族一脉的大祭司。
<br/>　　此刻，大祭司正端坐在大殿中，主座的右下首，一双眼眸如锋利的尖刀，眸光狠狠的刺在苏莫的身上。
<br/>　　不过，大祭司并没有出手，一是因为苏莫的身前有火元圣师，二是大殿之中还有一人。
<br/>　　此人是一位中年人，穿着一身明黄色的长衫，面容儒雅，气质出众，仿若是一名潇洒文士。
<br/>　　但是，此人端坐在主座上，便表明了其身份，正是神武学府的府主。
<br/>　　“咳！”
<br/>　　火元圣师轻轻咳嗽了一声，随即大步向大殿内走去。
<br/>　　而苏莫，在火元圣师一声轻咳之下，身上一松，压力全消，他立刻紧跟在火元圣师的身后，走进了大殿之中。
<br/>　　“见过府主！”火元圣师走进大殿，向儒雅中年人微微抱拳。
<br/>　　“见过府主！”苏莫不敢怠慢，亦是立刻抱拳行礼。
<br/>　　“火元导师，坐吧！”儒雅中年人瞥了一眼苏莫，随即面带微笑的示意火元圣师就坐。
<br/>　　苏莫闻言心中惊疑，府主居然称呼火元圣师为导师，这是什么情况？
<br/>　　看来火元圣师比他想的还要不简单啊，对方不仅是武圣境的炼器师，可能还是府主的前辈，指点过府主。
<br/>　　“多谢府主！”火元圣师没有失了礼数，抱拳回礼，然后走到了左边的坐席上端坐了下来。
<br/>　　苏莫不敢独自站在殿中面对大祭司，紧跟着火元圣师，站到了对方的身后。
<br/>　　大殿中寂静无声，气氛略显压抑，苏莫的目光紧紧的盯在大祭司的身上。
<br/>　　而大祭司同样望着苏莫，微眯的眼眸中，冰寒刺骨，让人不敢直视。
<br/>　　“你就是苏莫，传闻中的真圣榜第一？”
<br/>　　少倾之后，府主目光凝视着苏莫，淡淡的开口问道，其声音充满磁性，不喜不怒。
<br/>　　“正是弟子！”苏莫不卑不亢的点了点头。
<br/>　　“不错！”
<br/>　　府主微微颌首，略一停顿，凝声道：“听巫仑大祭司所言，你杀了他百余万族人，此事是否属实？”
<br/>　　“属实！”苏莫深深的吸了口气，便点了点头，他无法否认这个事实，也不能否认。
<br/>　　连他在战界的战斗影像，都能被人括印下来，他灭杀巫族小镇的情况，肯定也不可能瞒得住。
<br/>　　此事，估计大祭司早就调查清楚了。
<br/>　　所以，他无法否认！
<br/>　　“你犯下了弥天大罪，现在巫仑大祭司要将你带走，你可有话说？”府主再次开口道。
<br/>　　苏莫面色凝重，他从府主的话中，听不出对方维护他的意思，也听不出对方不维护他的意思。
<br/>　　总之，对方的言语不偏不正，完全是秉公办事的姿态。
<br/>　　“禀府主，我之所以会杀巫族百万人，是因为巫族杀我千亿族人！”苏莫沉声说道。
<br/>　　“什么？”府主闻言微微愕然，巫族杀苏莫千亿族人？这可能吗？
<br/>　　连火元圣师，都是神情一怔，被苏莫的话惊讶到了。
<br/>　　“一派胡言！”
<br/>　　大祭司闻听此言，顿时一声叱喝，冷声道：“苏莫，你灭绝人性，杀我族人百余万，任何人都救不了你！”
<br/>　　府主和火元圣师两人，闻听大祭司此言，均是眉头微挑，此言看似是在斥责苏莫，实则是隐晦的告诉他们，任何人都不能阻止他带走苏莫。
<br/>　　“我灭绝人性？”
<br/>　　苏莫闻言冷笑一声，目光直视对方，厉声道：“你们巫族灭了苍穹世界，杀死苍穹世界千亿生灵，难道就不是灭绝人性？这比灭绝人性还有狠毒一万倍！”
<br/>　　事到如今，苏莫也不怕大祭司了，反正惧怕对方也无用。
<br/>　　有火元圣师和府主在此，对方也不能强行将他怎样，现在他倒是要好好和对方理论一番。
<br/>　　“苍穹世界灭了也就灭了，一个小世界而已！”大祭司淡漠的说道，完全不在乎。
<br/>　　“一个小世界而已？”
<br/>　　苏莫闻言眸中一片冰冷，怒声道：“你知道苍穹世界有多少人吗？何止千亿，说有万亿都不过分，万亿苍穹世界之人，全部因你而死，你说我们谁更灭绝人性？”
<br/>　　“一群土著的贱命，怎能与我百万巫族之人相提并论？”大祭司冷漠的说道。
<br/>　　“哈哈哈哈！！！”
<br/>　　苏莫闻听此言，顿时疯狂大笑了起来，怒急而笑，声震八方。
<br/>　　随即，他的脸色一片铁青，怒声咆哮了起来：“贱命？我们苍穹世界的人都是贱命，你们巫族之人高贵？敢问大祭司，你们巫族之人哪里高贵？”
<br/>　　苏莫是真的怒了，他最恨的就是巫族自喻高贵，贬低苍穹世界的人都是土著，都是贱命。
<br/>　　整个宇宙，无尽星空，无数种族，从来没有什么种族天生高贵，更没有什么种族天生低贱。
<br/>　　在他看来，每个种族，每个生灵，都只是命运不同而已！
<br/>　　“伶牙俐齿！”
<br/>　　大祭司冷笑一声，缓缓的道：“老夫告诉你，什么是高贵，什么是低贱；我族掌控数百颗星辰，纵横古灵星河，名震天荒星域，这就是高贵；而你们苍穹世界的人，一群土著，活在弹丸之地，这就是低贱，天生贱命！”
