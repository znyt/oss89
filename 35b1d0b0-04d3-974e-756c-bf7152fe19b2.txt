　　枯枯戮山山顶，奇犽站在黑洞洞火山口前。
<br/>　　探头向坑内看了一眼，漆黑不见底的火山，好似无底深渊，让人不寒而栗。
<br/>　　“糜稽就在这里面修炼？”感受着从地底冲上来的滚滚热浪，奇犽一脸的怪异：“这是人能呆的地方？”
<br/>　　“糜稽！出来！快出来啊糜稽！”奇犽双手拢在嘴边，想着坑洞底部大声呼喊。
<br/>　　喊了半天，仍然不减人回应：“不会掉到岩浆湖里面了吧？还是睡着了？看来需要更大的刺激！”奇犽一脸的认真。
<br/>　　当苏重从火山内冲出来时，就看到奇犽站在火山口旁边。身前摆着一个五六米高的巨大石块，奇犽这会儿正急赤白咧的推石头。
<br/>　　脸涨的通红，好似玩儿的正起劲儿！
<br/>　　苏重无语：“你就不怕砸死我？”
<br/>　　“你怎么这就出来了？”
<br/>　　苏重犯了个白眼。你那一副失望至极的表情是闹哪样，还真要砸死我才开心？！不愧是揍敌客家的熊孩子。
<br/>　　“什么事？”苏重没好气道。
<br/>　　奇犽竟然犹豫起来。
<br/>　　这到让苏重有些好奇。可能由于前身经常找茬的原因，奇犽面对前身时，从来都是有一说一从不含糊。因为含糊也没用，前身总会找他麻烦。
<br/>　　“那种能力到底是什么？”好半晌，奇犽才下定决心，低着头小声问道。
<br/>　　如果不是苏重听力过人，甚至都听不清奇犽说了什么。
<br/>　　哪种能力？念？苏重旋即了然。虽然不知道是通过什么途径，但奇犽显然比原本的世界，更早的接触到了念。
<br/>　　“那种能力叫做念，是操纵身体气场的独特能力。能够让人的实力快速提高。”苏重没打算隐瞒。
<br/>　　相对于这个世界的绝大多数普通人，念是神秘的，是不可触摸。甚至有人一辈子都不知道什么是念。
<br/>　　但显然，揍敌客家不在此列。
<br/>　　“能交给我吗？”奇犽满脸希冀的看着苏重。
<br/>　　“不行，还不到时候。”苏重毫不犹豫的拒绝道。
<br/>　　奇犽还没完成家族安排的训练，身体素质并未达到最佳状态。这个时候开启念力，虽然能快速提高实力，却不能让实力最大化提高。
<br/>　　原著中，奇犽一觉醒念，实力立刻突飞猛进。念的数量，很快就超过同期的其他人。
<br/>　　天赋固然是一种因素，但远超他人的身体素质，同样起到决定作用。
<br/>　　“不是时候！又是不是时候！所有人都知道，只有我不知道！”奇犽突然抬头，狠狠瞪着苏重：“你说不说！”
<br/>　　凶狠的气势猛然冲向苏重。
<br/>　　奇犽虽然年纪不大，却已经是一名合格杀手。一旦放开杀气，甚至单凭气势就能震慑他人！
<br/>　　可在苏重看来，这种气势甚至连挠痒痒都不够。饶有兴趣的看着满脸凶狠，好似发怒的幼兽一般的奇犽，苏重呵呵一笑。
<br/>　　奇犽怒火腾的一下冒了出来。这一刻，苏重那张带着玩味的笑脸，分外可恶。
<br/>　　啊！
<br/>　　一声怒喝。奇犽猛然冲到苏重身前，右手并指，尖锐的指甲猛然突出。好似猛兽伸出利爪，狠狠刺向苏重。
<br/>　　苏重不紧不慢，脸上似笑非笑表情不变。从容退后一步，正好躲过奇犽发力顶端。右手伸出，好似鹤嘴轻巧叼住奇犽手腕。
<br/>　　身体猛然一转，奇犽身体骤然离地而起。呼的一下，就被扔出去十多米远。
<br/>　　身在半空，奇犽猛然翻滚身体。落地后，右手成爪子紧紧扣紧泥土。低伏着身体，满脸狰狞的盯着苏重。
<br/>　　可恶！自己全力突袭，竟然被扔了出来。那种轻松，就好似大人和小孩子在玩耍！
<br/>　　两年前，他还能够完胜糜稽。正因为如此，奇犽才会不在乎糜稽的找茬。因为糜稽打不过他，所以也就不敢得罪他。
<br/>　　可现在，糜稽不再找他的麻烦。奇犽却发现，他已经打不过糜稽了。甚至根本就不再一个层级上！
<br/>　　肯定是因为那种能力！即便学不会，也一定要亲身感受那种力量！
<br/>　　看过苏重战斗影响，奇犽知道自己和苏重之间的差距，那是天与地之间的差距。
<br/>　　砰！
<br/>　　奇犽身影瞬间消失在原地。周围陡然出现十多个一模一样的奇犽，从各个角度一起攻击苏重。
<br/>　　肢曲？真是让人怀念的技艺。他已经很久没有使用这类迷惑技能了。苏重发现，随着魔龙七段的修炼，他越来越喜欢硬刚了。
<br/>　　左手背负身后，苏重空着右手随意的摆动，却恰到好处的挡住攻击。轻松写意，毫无烟火之气。
<br/>　　砰！
<br/>　　奇犽再次被苏重扔飞出去。
<br/>　　苏重在理解了揍敌客暗杀术后，甚至独创了自己的龙牙暗杀术。对付只能使用基础暗杀术的奇犽，意外的轻松。
<br/>　　甚至在苏重看来，这根本就不是战斗，而是在放松。放松数月修炼以来的紧绷精神。
<br/>　　“可恶！”奇犽气的脸通红。他怎么可能看不出苏重的轻松随意。自己拼尽全力，那家伙却还那么的轻松随意？太可气了！
<br/>　　他刚想再度冲上去。
<br/>　　一道黑影突兀出现在苏重头顶。
<br/>　　砰砰砰……
<br/>　　一连串攻击如倾盆大雨，毫不留情的轰击在了苏重头顶。
<br/>　　黑影身形一阵模糊消失在，在远处显出身形。
<br/>　　“大哥？你在干什么！”奇犽满脸震惊。
<br/>　　苏重头顶密密麻麻的插满了圆头钉子，浑身僵硬的站在原地一动不动。
<br/>　　“大哥，你把糜稽杀啦？”奇犽的声音里带着丝丝绝望。
<br/>　　“杀？”伊尔迷时钟呆呆的表情，突然变得扭曲。圆圆眼睛中再也没有往日的深邃，反而全都是狂热。看向苏重的表情，就好似在看一个稀世珍宝！
<br/>　　“我可舍不得杀。”伊尔迷声音里满是兴奋，甚至甚至可以说是亢奋：“横推一切的力量，束缚所有的丝线。多么完美的能力，把这种能力操控在家族手中，必然能让揍敌客更加强大！”
<br/>　　奇犽睁大眼睛，不可思议的盯着伊尔迷：“可糜稽本来就是揍敌客家的人啊！为什么还要……”
<br/>　　“为什么？你难道没有发现，糜稽太过随性了吗？以前还没什么，这两年，他越发不把揍敌客放在心中。”伊尔迷冷然道：“上次惹了多大的乱子，你难道不知道？”
<br/>　　伊尔迷意味不明的看着苏重：“这一次他扛了过去，可如果惹了更大的乱子，扛不过去了呢？不仅他自己要死，揍敌客也要受牵连！”
<br/>　　伊尔迷脸色恢复肃然：“力量，要受到约束。不能再让糜稽继续放任下去了。”
<br/>　　“力量确实要受到约束，可我的力量，你约束的了吗？”一道淡淡的声音突然响起。
<br/>　　伊尔迷浑身一僵。不知何时，苏重竟站在了他身后不远处。
<br/>　　身前一个，身后一个，竟然出现了两个苏重！两人衣服一个干净整洁，一个破破烂烂。除此之外，两人面貌一样，眼神灵动蕴含神采，怎么看都是活生生的两个人！
<br/>　　如果不是知道糜稽底细，他都以为这是一对双胞胎！
<br/>　　分身！真假难辨的独特分身能力！
