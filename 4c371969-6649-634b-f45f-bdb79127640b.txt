　　在青色莲花的下面，极为细长的根茎，足足有十多米长，在根茎之上，遍布着数之不尽的细小触须。

<br/>　　这朵青色莲花，就这样悬浮在无尽的岩浆之中，犹如大海中的浮萍一般，四处飘泊。
<br/>　　若非他是凤凰，对热量敏感到了极点，能够感随着热量追寻到此处，恐怕找上几天几夜，也不可能在庞大的岩浆领域中，寻找出这么一朵相对极为渺小的青色莲花。
<br/>　　抱着期待的心情，慕清霄双翅一振，就来到青色莲花前。
<br/>　　近距离看青色莲花，简直美轮美奂，这种近乎完美的东西，恐怕也只有靠时间与大自然的磨练，才能创造出来吧。

<br/>　　嘴巴一张，在恐怖的吸力下，整个莲台迅速缩小，转眼就被慕清霄吞入腹中。
<br/>　　不仅是青莲地心火，这朵孕育了青莲地心火的青莲，以及其中的莲子，都是珍贵的天材地宝，世间就只此一份，珍贵程度可想而知。
<br/>　　正所谓千年成莲，吸收了数千年岩浆中的狂暴力量火属性能量，其中蕴含的庞大能量，对于旁人是毁灭性的东西，但对慕清霄而言，绝对是大补之物。
<br/>　　“呃！”
<br/>　　将青莲地心火连同莲花一起吞入腹中后，慕清霄打了个饱嗝，就是不知道能给自己提升多少实力，抱着期待的心情，以极快的速度向岩浆上面疾驰而出。
<br/>　　刚冲出洞穴，看到碧空如洗的蓝天，慕清霄舒适的长啸一声，嘹亮的凤鸣响彻整个洞穴。
<br/>　　“嗯？”
<br/>　　凤鸣消散，慕清霄却依旧保持着凤凰形态。
<br/>　　因为他察觉到，有几道颇为强悍的气息赶了过来，其中还有一道极为熟悉，若是现在变化成人形，自己的底牌估计会暴露出来。
<br/>　　果然，下一秒在慕清霄不远处，空间陡然扭曲起来。
<br/>　　天空上，炽热的阳光倾洒而下，照射在扭曲的空间处，下一秒，一道曼妙的曲线娇躯，缓缓出现在了他的视线中。
<br/>　　突兀出现的美丽女子，身着一件雍容的紫色锦袍，锦袍之下的娇躯，丰满玲珑，犹如那成熟的蜜桃一般，渗透出淡淡的妩媚。
<br/>　　一头三千青丝，随意的从香肩披散而下，垂直那纤细的柳腰之间，而在那锦袍之下，露出一截紫色的蛇尾，蛇尾微微摆动，一股野性的妖娆诱惑，使人莫名其妙的浑身有些燥热。

<br/>　　霸气冷艳，妖娆无双，倾国倾城，气质魅惑，身材曲线火爆，风华绝代，绝对是世间少有的尤物。
<br/>　　然而，在那抹冷艳的气质下，却偏偏噙着一抹女皇般的高贵与雍容，这股雍容的气质，与云韵几乎相当，不对，更甚云韵。
<br/>　　“美杜莎！”
<br/>　　看到女子的瞬间，慕清霄就确定了她的身份。
<br/>　　此处也只有美杜莎，才拥有如此妖艳的容颜，且能在斗皇境界初步掌握空间奥妙，实力比一般斗皇强悍的多。
<br/>　　否则，冰皇也不会被她设下封印，打回斗灵级别。
<br/>　　慕清霄看到了美杜莎，美杜莎自然老远的就看到了他，更何况翱翔在天空上的黑凤神骏无比，散发出一股令她余悸的气息，想不注意到都难。
<br/>　　看到黑凤的瞬间，美杜莎仿佛看到了天敌一样。
<br/>　　娇躯猛的一颤，好在她实力强大，能够压制来源于灵魂深处的恐惧，若是换做其他蛇族，现在恐怕已经匍匐在地上瑟瑟发抖。
<br/>　　“凤族！不可能！凤族怎么可能出现在此处，而且形态也与我所知道的凤族有些不同。”
<br/>　　望着天空上的慕清霄，美杜莎瞳孔中闪过一丝惊骇。
<br/>　　听到美杜莎的话语，慕清霄有些意外，没想到她还知道凤族，不过斗破这种位面中的凤族，他可不认为拥有完整的血脉，估计也就是一堆杂毛鸟。
<br/>　　毕竟，龙族也都是西方龙种类型的，估计血脉也是稀薄的很。
