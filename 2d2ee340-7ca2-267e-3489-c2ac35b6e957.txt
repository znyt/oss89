        &lt;h3&gt;第三卷第七十三章  开会&lt;/h3&gt;
<br/>
<br/>        休了几天假，陈子昂满血复活，带着郝景文发给她的文案上班了。
<br/>
<br/>        江俞轩还在西城，西城的房产事业将会步入一个新的台阶，他要接手家族事业了。
<br/>
<br/>        陈子昂既然知道江俞轩是江家大少，她就必须得做好准备，江俞轩随时都会回去，她也必须得培养一个得力的助手。
<br/>
<br/>        北京算是总部，他们在各个地区都会有代理商，下属也有很多的经理，从下面挑选一下人才回总公司不是不行，但是像江俞轩这样的怕是不多的。
<br/>
<br/>        她还是将眼光放在了西城，西城的呼延寰宇后期要和她去甸城，李凌菲应该是可以调到北京来的，后期让李凌菲管理商务部们，耗材也是需要一个有能力的人来推动。
<br/>
<br/>        赵绾绾主管信息化，没有赵绾绾信息部门的支撑，无论是前期还是后期都无法立足。
<br/>
<br/>        倪小美年龄比较大，很稳重，话少，但是倪小美会多国语言，曾经做过翻译，后期如果耗材出口的话让倪小美和李凌菲一起搭档。
<br/>
<br/>        还有青岛的田星也是一个不可多得的人选。
<br/>
<br/>        在人事上，陈子昂有些为难了，其实在人事上江俞轩比她知道的多得多，而且惟人才是用，每一个人都被他安排的好好的。
<br/>
<br/>        只好等江俞轩回来了再商量。
<br/>
<br/>        原本他们就是做信息化，他们自己公司内部有一套系统，分级管理，赵绾绾把它叫做领导管理系统，陈子昂可以分级实行管理，也能看到各部分的财务报表和花费情况，人员考勤状况。
<br/>
<br/>        这个管理系统还有一个好处就是，可以开视频会议，这些都是经过赵绾绾他们部门经过不停的试验摸索出来的。
<br/>
<br/>        陈子昂他们已经搬到了他们新的办公地点，他们自己的厂房。
<br/>
<br/>        公司的同事其实都已经把公司当成了自己的事情，陈子昂到达公司的时候，大家都已经整整齐齐的等着开早会了。
<br/>
<br/>        倪小美早已经将视频会议系统打开了，他们很多人都是出差在外，能聚到北京的除非是重大的会议或者节假日，但是节假日也就限于过年，每年过年公司会开年会，表彰一些先进的人。
<br/>
<br/>        公司的播放室里正在播放着“我爱你祖国。”和一些宣传片，陈子昂才惊觉快要到国庆节了。
<br/>
<br/>        郝景文的方案是绝密的，这个是目前市面上有没有的，按照他们的设想和陈子昂的估算，项目的成功率在百分之九十，但是耗费的资金和时间也是漫长的，陈子昂并不打算和大家共同讨论，她只和管信息化的高层讨论就好。
<br/>
<br/>        “大家早上好。”陈子昂面对着镜头，看着开会的一众人，心里也是感慨万千，她的公司现在可以说是处在黄金年龄，社会形势一片大好，公司也是乘势而上。
<br/>
<br/>        因为是视频会议，大家各自面对着视频，陈子昂看了一下，几个高层基本都在，就是少了江俞轩，陈子昂叹了口气，看来，江俞轩是真的快要离开了。
<br/>
<br/>        “今天，也没有太多的事情要说，目前的状态不知道大家都满不满意？对于公司大家还有没有更好的建议？”
<br/>
<br/>        陈子昂觉得开会都是废话，有事情要做的话大家一起协商，如果啰啰嗦嗦的就是浪费大家的时间，她抛出去一句话，引导大家讨论，其实是比吩咐干某件事的效果要好很多。
<br/>
<br/>        大家在陈子昂的领导下，也都是言简意赅，一句话能说清楚的绝对不会去累述，而且他们开会都是固定时间的，如果有问题了都提前准备好，在会上就解决了，正所谓众人捧柴火焰高。
<br/>
<br/>        大家的问题都在会议上解决了。
<br/>
<br/>        陈子昂对大家说：“资金的问题大家可以直接和财务总裁凤盛夏联系，信息项目上的事情是赵绾绾经理，另外，后期我们的耗材会要大批量的投入使用，你们手中的项目都是极好的资源，请大家转变一下思路，推动耗材的发展。”
<br/>
<br/>        “众所周知，打印机其实和汽车一样，只要开汽车你就得烧汽油，相同的，只要你用打印机就需要用到耗材，目前，国产耗材处于起步阶段，以后的市场份额不会少，只会更多。”
<br/>
<br/>        陈子昂看着大家都不语，然后说：“我还要众筹耗材品牌名称，大家有好的合适的名称可以建议。”
<br/>
<br/>        总经理办公室的胡蜜儿已经将公司里的放假计划安排好了，大家有七天假，陈子昂心里说：“怎么就忘记了还有十一七天假呢，要不然前几天就不用休假了。”
<br/>
<br/>        开完会，陈子昂留下赵绾绾，倪小美和皇甫双双。
<br/>
<br/>        她拿出来了郝景文的文案，在大屏幕上给大家一遍一遍的过着。
<br/>
<br/>        “我们需要设计软件，一套能够呼叫和行车记录的软件，能够适时的发现行车的位置。”
<br/>
<br/>        “我们需要深入了解一下。”赵绾绾对这个还是比较感兴趣的，对于他们来说，他们需要挑战，需要不停的去发掘新的领域。
<br/>
<br/>        “行车位置，目前市上已经有了成品的软件，只是该怎么和这个软件对接是个问题。”陈子昂说。
<br/>
<br/>        “而我们需要研发的这个软件的功能和概念目前来说还是模糊不清的。大家动动心思，每个人拿出一个方案来。”三个臭皮匠顶个诸葛亮，集思广益也许有更好的发现。
<br/>
<br/>        “陈疯子，你这总是给我们找事情干，耗材还没有拓展出去呢，新的玩意又来了？”苟江凌手中写写画画，头也不抬的说。
<br/>
<br/>        “要不如此，怎么能体现你们的价值呢，在座的各位大佬们一个个都是响当当的人物，我总不能让你们躺在功劳簿上等死吧？”
<br/>
<br/>        “要等死也是我们的事情与你何干？”
<br/>
<br/>        “我于心有愧啊。”苟江凌只要见到陈子昂总会挤兑两句。
<br/>
<br/>        “俞轩怎么没有开会？”倪小美发现开会的时候没有江俞轩。
<br/>
<br/>        “俞轩在西城还有项目要做，对了告诉大家一件事情，江俞轩是江家大少，后期可能要回江家。”
<br/>
<br/>        “你说什么？”几个人同时惊问。
<br/>
<br/>        “我也是不久前才知道，他要结婚了，也要回江家了。”
<br/>
<br/>        “啊，啊，江家大少啊，等他回来，我要他带我吃遍全北京的美食，哼，谁让他欺骗我们的感情。”皇甫双双咬牙切齿的样子真的好像是想将北京的美食全都吃下去。
<br/>
<br/>        “吃遍全北京有啥意思呢？要吃遍全国。”苟江凌
<br/>
<br/>        “我什么都不吃，等他结婚的那一天不让他进，洞，房。”赵绾绾幽幽的说。
<br/>
<br/>        “你们够狠。”陈子昂笑着说。
<br/>
<br/>        江俞轩在西城竞拍会上打了大大的一个喷嚏。
