　　第722章
<br/>　　冯姓剑客名唤冯四方，用一柄铁剑，所以人们也称他铁剑先生。
<br/>　　他是三个月前出现在陈家庄，也是落魄走投无路，暂且在陈家庄借宿几日。然后几日便十几日，十几日变成数十日。
<br/>　　他在陈家庄人缘不错，与壮丁喝酒吃肉，称兄道弟。纵然也有人在陈德山面前告他的黑状，说他白吃白喝，平时好吃懒做，明显是将陈德山当做二傻子。
<br/>　　陈德山都是一笑置之，说些江湖豪客自然有江湖豪客要做的事情，不必在意这些。
<br/>　　也就是这次荆棘岭的事情闹出来，陈德山才想起这位铁剑先生。敲定三千两白银已经令陈德山心里憋着一口气，谁料想其在关键关头，竟还要坐地起价。
<br/>　　不过，陈德山毕竟是见过世面的人，他压下心中的火气，依旧客气道：“请冯先生开口。”
<br/>　　冯四方抱剑而立：“听说陈庄主家中有一尊六手金佛，金银不要，我要这尊金佛。”
<br/>　　老仆怒不可遏，指着冯四方的鼻子道：“大胆，你在陈家白吃白喝多日，现在竟还要趁火打劫，俺真是瞎了眼，错认你这狼心狗肺的畜生。”
<br/>　　陈德山膝下只有一女，今年正当双九年纪，陈德山对其视若掌上明珠。只是这女孩从小体弱，怕不是长命之人。陈德山就花了大价钱铸了一尊金佛，请佛祖保佑。
<br/>　　冯四风冷冷道：“我有一剑在手，天下宝物尽可取自，让你自己拿出来，是给你面子。”
<br/>　　陈德山眼皮跳动两下，冲陈忠道：“退下，事成之后，双手奉上。”
<br/>　　冯四方这才转过身，面向荆棘岭十三名匪人。他双手一抱拳，道：“在下扬州冯四方，江湖人称铁剑先生，还请诸位兄弟买我个面子，揭过这件事。”
<br/>　　“雷动。”马上为首一人抱抱拳，道：“姓冯的，你算哪里来的鸟，也想我们兄弟看你面子。今天也不说我们赶尽杀绝，你能胜过我们兄弟，我就认你的名号。”
<br/>　　冯四方一摊手道：“请。”
<br/>　　雷动目光转了一圈，看向那扛大锤的汉子：“李胖子，你去会会他，摸摸他的底。”
<br/>　　“好嘞。”
<br/>　　那李胖子抹抹嘴，扛着大锤走到冯四方面前，嘿嘿笑了一声，一锤砸向冯四方脑袋。来得险又急，冯四方往后退了三步，锵地一声，铁剑出鞘。他在身前画了个圆，弧光一闪，将李胖子逼退。
<br/>　　陈德山紧紧盯着这一幕，若能躲过这场无妄之灾，便是损失一尊金佛也算不得什么。
<br/>　　委实小看了这粗犷的胖子，他虽然体态臃肿，但动作实在灵便。浑身肥肉不停抖动，竟一而再再二三的往前进攻。
<br/>　　冯四方守得方寸不乱，胖子的几次进攻都被他挡下。雷动看在眼里，悄悄皱起眉头。
<br/>　　胖子的力气可急不可长，只要顶住他开始几轮进攻，胖子非败不可。
<br/>　　“杨枭，赵寒，你们上。”
<br/>　　“好嘞。”
<br/>　　痛快两声答应，从队伍中又跃出两人，一人用弯刀，一人用长枪。三人合力围攻，冯铁剑顿时陷入狼狈不堪的境地，只能用铁剑护住身体。
<br/>　　“以多欺少，卑鄙。”陈忠骂道。
<br/>　　雷动根本没搭理他，什么卑鄙不卑鄙，老子是山贼，要不卑鄙，在就去读书考状元了。
<br/>　　赵寒一枪刺中冯四方小腿，他仰面跌倒在地，等在回过神来时，已经被三名凶神恶煞的壮汉围住。
<br/>　　李胖子哈哈大笑：“大哥，挑断他的手筋脚筋，回家做成肉馅包子。”
<br/>　　冯四方赶忙开口：“好汉饶命，好汉饶命，我早就仰慕荆棘岭好汉的大名，好汉饶我一命，我愿投靠荆棘岭。”
<br/>　　冯四方，号称铁剑先生。但很显然，他的铁剑挡不住四方，三面他都挡不住，但这见风使舵，八面玲珑的本事当真不弱。
<br/>　　雷动冷笑：“荆棘岭可以不要那贪生怕死，见利忘义之徒。”
<br/>　　“哈，还是让俺一锤将他脑袋砸成稀巴烂，免得这腌臜货恶心我。”李胖子举起大锤。
<br/>　　“慢，不要杀我。”冯四方大吼：“我知道陈家把女儿藏在何地，放我一条性命，我告诉诸位好汉。”
<br/>　　雷动摆了摆手，让李胖子先放过此人。这次闯院抢人，有个知晓陈家庄内情的人也好做事些。
<br/>　　陈德山牙齿咬碎，道：“老朽活了多半辈子，今天才知道是一双瞎眼，不认得人间畜生。”
<br/>　　屋顶上，刘发财也撇撇嘴：“这孙子做事很不地道呐。”
<br/>　　程大雷倒是不以为意，生逢乱世，行走江湖，说来说去，无非就是活着二字而已。因为此二字，杀人放火，打家劫舍并不算什么。毕竟大家也都是这么做的，至于所谓的行侠仗义，路见不平，程大雷也是听见得多，见过的少。
<br/>　　“姓刘的，你下去，平了这几个贼人。”
<br/>　　“这……”刘发财打了个哈哈：“点子扎手，我这两下子估计也够呛，还是要大当家亲自出马。”
<br/>　　程大雷白了他一眼，不过荆棘岭这几人本事却都不简单，刘发财的本事，怕是连冯铁剑都比不过。
<br/>　　但什么事都由自己出马，显得也不是很气派。
<br/>　　几人在房顶上费舌头，那雷动忽听见动静，仰头一瞥：“谁在说话？”
<br/>　　得，被人看破了，程大雷也没打算继续瞒着。纵身一跃，身体轻飘飘落地，手中折扇啪的一声打开。
<br/>　　“某家京州牛三斤，逢雨借宿此地。”
<br/>　　这一番动作漂亮干脆，如果忽略掉脸，当真有几分浊世佳公子的派头。
<br/>　　声音刚刚落地，就听得身后骨碌碌声响，刘发财和徐神机从屋顶上滚了下来。
<br/>　　刘发财武艺本就不如何，何况又带着徐神机，一个立足不稳就摔在了地上。好在屋顶不高，否则非得摔个腿断胳膊折不可。
<br/>　　程大雷深吸一口气，隐隐觉得有些头疼，摆摆手道：“你可要看清楚了，我和他们，不认识。”
<br/>　　雷动盯着程大雷，仔细端详片刻，随之哈哈大笑：“好好好，天堂有路你不走，地狱无门你闯进来，今天碰见个想要多管闲事的。”
<br/>　　程大雷耸耸肩：“抱歉，被你看出来了。”
<br/>(本章完)
