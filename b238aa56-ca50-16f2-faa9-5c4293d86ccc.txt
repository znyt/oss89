　　嗖…嗖…嗖…
<br/>　　一条条吞灵冰鱼在极光照耀下，一片流光溢彩中，刚出现于海面，便被吸摄而出。
<br/>　　一众参赛武者，纷纷大喜。
<br/>　　这些吞灵冰鱼，可是不可多得之物。
<br/>　　吸一条，抵得上数天苦修了。
<br/>　　而第一轮冰海之缘，只有七天时间。
<br/>　　高空中的冰宫执事，担任裁判，会详细记录各参赛武者的情况。
<br/>　　根据所获得的吞灵冰鱼数量、大小等等，给出相应的分数。
<br/>　　七天之后，根据排名，将给出相当丰厚的奖励。
<br/>　　无论是为了第一轮的奖励，还是现在马上就可以得到并感受到的修为增加，都让参赛武者们为之疯狂和兴奋。
<br/>　　这时，数十里之外，响起了一片激烈的战斗声响。
<br/>　　数十个武者的混战，让得那片区域灵气混乱，各种攻击，搅得冰面翻腾不已。
<br/>　　参与混战的，都是些独行武者。
<br/>　　萧逸，现在算是明白为何之前裂天剑姬前辈会说，如果没有名额，前来参加冰缘大会将相当麻烦。
<br/>　　冰缘大会，全炎龙大陆任何人都可以参加。
<br/>　　前提是在50岁以下。
<br/>　　拥有名额的武者，可直接前来参加。
<br/>　　而若没有名额，则需要完成许多考核和任务。
<br/>　　这些考核或者任务，五花八门。
<br/>　　有的是，要求斩杀十万头雪地冰狮；有的更过分，斩杀一万头暴风雪狮。
<br/>　　暴风雪狮，可是六级妖兽。
<br/>　　甚至有的，需要在这漫漫无边的极寒之地中，寻找某些特定而稀有的天材地宝。
<br/>　　总之，这些任务，五花八门，但无一例外，都相当繁琐，难度颇大。
<br/>　　先不说有没有那个实力完成这些任务。
<br/>　　就算有实力，也未必有足够的时间。
<br/>　　若有些妖兽太过特殊，有些天材地宝太过稀有，导致迟迟无法完成任务和考核。
<br/>　　在冰缘大会开始前尚未完成，便算失败。
<br/>　　无缘此次冰缘大会。
<br/>　　另外，就算侥幸通过了考核，能参加冰缘大会了。
<br/>　　参加时，也会有诸多麻烦。
<br/>　　比如这第一轮，捕获吞灵冰鱼。
<br/>　　有名额的武者这边，互相不可干扰，各捕各的。
<br/>　　而那些没有名额，通过完成考核而来的武者，则完全没有任何限制。
<br/>　　彼此间，可以通过抢夺的方式，抢夺他人刚刚捕获的吞灵冰鱼。
<br/>　　甚至大打出手，生死相拼，也没有任何问题。
<br/>　　这是一种相当恶性的竞争。
<br/>　　这次冰缘大会的参加者，有近一千五百人。
<br/>　　其中，9成以上都是有名额的武者。
<br/>　　只有不足1成，是通过考核而来的独行武者。
<br/>　　当然了，能通过考核而来的武者，个个都实力非凡。
<br/>　　只不过，他们可以互相打斗，却不能影响有名额的武者。
<br/>　　言归正传。
<br/>　　数十里之外的战斗，并未能影响萧逸等人这边。
<br/>　　这边有名额的武者，一个个都在快速捕获着吞灵冰鱼。
<br/>　　并将其吸收，化为自身修为。
<br/>　　不过，有名额的武者，也并非是一帆风顺的。
<br/>　　这不，萧逸脚下的冰块下方，此刻忽然出现了一团黑影。
<br/>　　黑影，倏忽而出，跃出海面。
<br/>　　那是一条噬人的冰海巨鱼，正张着血盆大口，朝萧逸咬去。
<br/>　　这条巨鱼的实力，绝对在地元境层次。
<br/>　　萧逸手指轻扬，一道剑气，瞬间将巨鱼分尸。
<br/>　　这里是极寒冰海，是一处险地。
<br/>　　海面之下，可是拥有着无数凶猛极寒海兽的。
<br/>　　海里的妖兽，可比陆地上的妖兽庞大得多，也凶猛得多了。
<br/>　　众人一边捕获着吞灵冰鱼，也要一边提防海兽的袭击。
<br/>　　这片冰海之下，可是有着不少连天元境武者都闻风丧胆地强大妖兽。
<br/>　　甚至于，冰海深处，或许还有着更强的存在。
<br/>　　这也是冰宫长老们也御空飞起，密切注视着下方参赛者情况的原因。
<br/>　　一旦有不妥，他们会马上出手。
<br/>　　当然了，能拥有名额来此参加冰缘大会的武者，背后的势力，都非泛泛之辈。
<br/>　　必然给予了足够的护身之物给后辈们。
<br/>　　所以，安危方面，倒无需太过担心。
<br/>　　言归正传。
<br/>　　一众武者‘欢快’地捕捉吞灵冰鱼。
<br/>　　虽不时会遭到海兽袭击的影响，而导致敏感的吞灵冰鱼受惊立马潜回冰海底下。
<br/>　　但，吞灵冰鱼毕竟全身都由灵气而成。
<br/>　　其灵智相当低。
<br/>　　敏感和逃跑，只是它们的本能。
<br/>　　逃跑没几秒，当感受到冰面上出现的大量真气或者元力时，它们又会马上游来。
<br/>　　人类武者在冰面上散发的力量，是它们的食物，对它们有着极大的诱惑。
<br/>　　时间，渐渐过去。
<br/>　　该打斗地，还在打斗。
<br/>　　该欢快捕捉地，还在捕捉着。
<br/>　　随着大量参赛武者自身元力或真气的不断放出。
<br/>　　这方圆百里内的灵气程度，激增到一个相当恐怖的情况。
<br/>　　一天后。
<br/>　　越来越多的吞灵冰鱼被吸引而来。
<br/>　　随之而来的，自然也是越来越多的海兽。
<br/>　　海兽暂时不必多管。
<br/>　　反倒是吞灵冰鱼这边，根据各范围灵气浓郁程度的不同，而导致密集度不同。
<br/>　　“嗯？”萧逸皱了皱眉头。
<br/>　　他明显发现，虽然吞灵冰鱼的数量增多了。
<br/>　　但出现在他、以及叶铭等人周遭的，却反而下降了。
<br/>　　几乎所有吞灵冰鱼，都朝着别的方向游去。
<br/>　　那个方向，正是极寒之地这边一些有名势力的参赛弟子的方向。
<br/>　　如雪刀门弟子、冰河教弟子、冰武教弟子，三家势力的参赛弟子周遭，聚集了密密麻麻的吞灵冰鱼。
<br/>　　不过短短时间，三家捕获的吞灵冰鱼，比别的势力参赛弟子数十家加起来还要多。
<br/>　　“该死。”在萧逸另一旁的数百米处的一块冰块上，钟无忧愤愤地说着。
<br/>　　“那些家伙，修的本就是寒冰一道。”
<br/>　　“体内释放的灵气，亦是寒冰属性，与极寒冰海这边的气息相当契合。”
<br/>　　“自然对吞灵冰鱼的诱惑力更大。”
<br/>　　“他们三家联手，捉一个时辰，比我们捉一天都要多。”
<br/>　　萧逸、白冰雪等五人，虽然彼此隔着数百米。
<br/>　　但都在一个范围上。
<br/>　　白冰雪亦是修习寒冰一道的武者，论寒冰气息，不比雪刀门三家弱。
<br/>　　只是，她现在也脸色难看到极点。
<br/>　　只因，雪刀门三家的参赛弟子，正不断往海面投放冰属性丹药。
<br/>　　观丹药的气息，绝对品阶不低，而且都是无杂质丹药。
<br/>　　另外，一件件冰属性灵器、元器，也被他们祭了出来。
<br/>　　虽说这些元器只是下品元器，但元器蕴含的冰属性力量，可比武者本身释放的，要浓郁得多。
<br/>　　难怪他们三家吸引吞灵冰鱼的速度和数量如此之多，如此之快。
<br/>　　这时，远处的风花月不屑地看了萧逸这边一眼。
<br/>　　“哼，裂天剑宗又如何。”
<br/>　　“在极寒之地，我们才是霸主。”
<br/>　　“想跟我们比？你们差远了。”
<br/>　　也不知是巧合还是别的原因。
<br/>　　风花月等人选的冰块位置，本就是最靠近萧逸等人的范围位置。
<br/>　　“裂天剑宗的杂碎，我要你们这一轮垫底。”风花月冷笑一声。
<br/>　　三家参赛子弟，加大了并属性气息和力量的释放。
<br/>　　顿时，周遭几乎所有吞灵冰鱼，都被吸引而去。
<br/>　　而萧逸等人这边范围的吞灵冰鱼，却寥寥无几。
<br/>　　若是按这般情况下去，恐怕七天时间，萧逸等人都捉不了几条吞灵冰鱼。
<br/>　　七天时间已过，必是成绩垫底。
<br/>　　“混账东西。”钟无忧冷喝一声。
<br/>　　“若非这第一轮不得互相干扰，老子真想砍人。”
<br/>　　叶铭高声道，“萧逸师弟，你向来手段过人，可有办法？”
<br/>　　白冰雪也将目光看向萧逸，道，“萧逸，别堕了我裂天剑宗的威风。”
<br/>　　凌羽也看向萧逸，满脸希冀目光，只是脸色有些复杂，不语。
<br/>　　萧逸点点头，淡笑道，“原来还能这样吸引吞灵冰鱼，也罢，他们过分，我们也不能客气。”
<br/>　　说着，清澈的目光，看了周遭几人一眼，道，“准备好，捉鱼。”
<br/>　　淡淡的话音落下。
<br/>　　萧逸体内冰山力量，瞬间爆发。
<br/>　　论寒冰气息和寒冰力量的精纯与澎湃，什么东西能比得过固态元力？
