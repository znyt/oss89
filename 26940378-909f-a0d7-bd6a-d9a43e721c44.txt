　　赵天宇跟司机说了地址，让司机把罗昊在小区门口放下。
<br/>　　罗昊跟赵天宇道了声别，赤手空拳什么也没拿，就这么下车朝着小区大门走去。
<br/>　　高档的小区售价之所以高，不仅是小区本身园林的设计、住宅风格、地理位置这些硬件，还有就是小区物业服务这样的软件。
<br/>　　一个住宅区的质量如何，往往通过小区物业服务就能了解一二，这里的保安二十四小时值守，而且其中一部分还是从武警部队退役的老兵，就是他们站岗时的姿势就不是普通小区那种制服扣子扣不齐，年纪普遍在五十岁以上，也就是混个退休年纪的中年保安能比的。
<br/>　　住在这个小区里的业主都是非富即贵的，说句难听的，进出这个小区开辆奔驰e系列都只能算“低档”。
<br/>　　小区中的保安不仅对小区业主很熟悉，而且对业主的交际也能记住那么一部分，高端的圈子其实并不会很大，来来去去就这么些人，不像混社会道上的，什么杂七杂八的人都能被称为“朋友”和“兄弟”。
<br/>　　站在门口的保安看到罗昊，立刻给他打开了电子门。这个保安叫孙平，是武警部队退役的士官，看到罗昊手臂上缠着绷带，脸上贴着创可贴立刻就明白了罗昊从哪里回来，也没有多问，只是对着罗昊敬了个礼。
<br/>　　罗昊在年纪上确实要比自己小上那么几岁，但是孙平认为对方当得起自己这个军礼。罗昊也是对着孙平回了一个礼，才迈步走进小区。
<br/>　　站在门口整了整自己仪容，罗昊才按了门铃，云熙穿着一身居家服打开了门，罗昊对着她伸出手，说道：“想我没？”
<br/>　　“想你个大头鬼！”
<br/>　　云熙瞥了眼罗昊，让他先让进屋，把门一关，小宇宙就爆发了，对着他一通数落，“别以为这样就能转移我的注意力，我告诉你，不可能！老实交代，这次出去伤了几处，严不严重？”
<br/>　　“两处！”罗昊张口回答道。
<br/>　　叩！
<br/>　　云熙圈起手指照着罗昊额头弹了一下，冷哼道：“你当我瞎啊，就我看到的已经不止两处了，还回答的理直气壮。给你次机会，重新回答。”
<br/>　　罗昊仔细想了想，看了看自己手臂上缠得绷带，认真地回答道：“五处。”
<br/>　　看到云熙眼中闪过紧张，罗昊立刻又补充了一句，“全都是皮外伤，就是擦破点皮，开了个小口子，绝对没有枪伤。”
<br/>　　“我不信！”云熙站起身，对罗昊命令道：“起身，脱衣服，趴好！”
<br/>　　罗昊双手护在胸前，“紧张”地问道：“你想要干嘛？我可不是随便的人。”
<br/>　　“少来这套。”云熙撇嘴，把放在客厅里的药箱拿了过来，说道：“你身上我有什么地方是没看过的？”
<br/>　　云熙把罗昊背后那些贴着纱布的地方拆掉，重新消毒后，给他换上新的透气纱布，小伤口如果不注意处理，也是会引起大问题的。处理完伤口，云熙正打算把药箱收拾起来，发现罗昊竟然趴在沙发上已经睡着了。
<br/>　　作为行动指挥官，他要承担比别人更多的压力，云熙也知道罗昊这段时间没怎么休息，就算是在飞机上睡了一觉，那也不会舒服。
<br/>　　云熙没有去打扰罗昊，轻手轻脚的收拾完药箱，上楼给罗昊拿了条毯子。
<br/>　　罗昊这一觉睡得很舒服，傍晚是被一阵饭菜香味给叫醒的。罗昊使劲嗅了嗅鼻子，一骨碌从床上爬了起来，连衣服都没穿，穿上拖鞋就往厨房跑，看到正在厨房里忙碌的云熙，从背后抱住她，问道：“晚上做了什么好吃的？”
<br/>　　“神出鬼没的，吓我一跳！”
<br/>　　云熙白了罗昊一眼，把锅子里的炒蛤蜊盛了出来，说道：“衣服都不穿就乱跑。你衣服被我洗了，自己去楼上找件干净的衣服，然后洗手准备吃饭。”
<br/>　　“明白！”
<br/>　　罗昊退后两步，立正转身朝着楼上跑去。
<br/>　　晚饭后，在小区中散步的时候，罗昊跟云熙说起了佟千雪的事情，云熙听完叹了口气，说道：“都是女人，我能想象到如果佟千雪真的无法怀孕，她会有什么反应。当年你失踪之后，如果我再失去悠儿和罗战，我估计我会疯掉的。不过，医生也说了，如果好好调养还是有可能恢复的，这个我觉得佟千雪可以去尝试中医调理和调养，毕竟在这一方面，中医比西医更有效果。”
<br/>　　罗昊闻言眼前顿时一亮，兴奋地说道：“我怎么就没有想到！”
<br/>　　“你想到什么了？”云熙好奇地看向罗昊，不明白是什么样的事情让他如此兴奋。
<br/>　　“找龙猫啊！”罗昊说道：“司徒家不仅是武术世家，家传武学了得，在中医药理上也是十分出众，你忘记在龙穴里那几个我们用来泡澡治疗暗伤的药浴桶了？”
<br/>　　“是啊，怎么把龙猫给忘了。”
<br/>　　云熙一听罗昊的话，也是跟着高兴起来。现在外面坑蒙拐变的假专家太多了，导致很多人对中医有了很深的误解，甚至认为中医远不及西医；可是要论历史，西医绕着赤道跑三圈都赶不上中医，中医老祖宗那手“望闻问切”绝技横行天下的时候，西方世界压根就没有医学这个概念。
<br/>　　与其在外找不靠谱的中医尝试，还不如直接去问司徒空。如果佟千雪的情况能改善，司徒空肯定不会藏着掖着。
<br/>　　事情有了希望，罗昊也是跟着轻松了一点。
<br/>　　如果佟千雪真的被确定为终身不孕，罗昊是非常担心慕容飞状况的。如果这家伙脑子一抽，因为这件事情而做出什么出格的举动来，刑警、特警、武警都拿他没任何办法，只能自己这边出面去解决，到那时候就真的是不死不休的局面了。
<br/>　　有朝一日把枪口对准昔日有着过命交情，在战场上以性命相托的兄弟，这是最难让罗昊接受的事情。不仅是罗昊，任何一支特种部队的指挥官最不愿意去做的事情，就是“清理门户”！
