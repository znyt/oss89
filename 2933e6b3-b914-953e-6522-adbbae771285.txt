　　叶少阳鼻子里哼了一声，“徐公，实话说，如果是昨天你找我，我可能还会答应，但是现在……哦对，你刚才也看到了，那个慕寒，你知道他是谁吧？”
<br/>　　徐文长眼神有些闪烁，点头道：“知道，他日前当上掌教时，给阴司上表了。”
<br/>　　“那就行了，”叶少阳摊了摊手，“人家是天选之子，现在又成了法术界的盟主，你们找他就好了，他肯定能圆满完成任务，我啊，你们用不着了。我就是来看看热闹，现在要回去了。”
<br/>　　说完冲他拱了拱手，站起来要走。
<br/>　　“砰！”
<br/>　　徐文长用力拍了桌子。
<br/>　　叶少阳转头看他时，他正怒气冲冲地站起来，一下子有点懵，怔怔地看着他。
<br/>　　“你你你，亏我们这么信任你，亏我还拼着犯错也要保护你！什么天选之子，四个人，谁又知道天选之子是谁，就算不是你，那又怎么样，你没了斗志吗，你很委屈吗？你从前也不知道自己是天选之子，不也一样以天下为己任，你怕什么，他是天选之子，难道就一定能压住你？
<br/>　　我如果是你，我一定要更加争口气，通过行动来告诉大帝，他错了，所有人通通都错了，你不是天选之子，但你比天选之子还要强，让他们知道，你才是人间的王！！”
<br/>　　说到最后，几乎是咆哮起来。
<br/>　　一行人怔怔地看着他，叶少阳喃喃道：“认识你这么久，还是第一次见你发这么大脾气……”
<br/>　　“我他妈是被你给气的！”徐文长一巴掌将桌上的茶壶茶杯全扫到地上去，指着叶少阳鼻子骂道：“大敌当前，你不出手，还在跟人斗气，而且还在跟一个不如你的人斗气，你——”
<br/>　　“我没斗气，说实话，我挺看不上他的，他的确是天才，但是，他连做我对手的资格都没有。”
<br/>　　“那你怕什么，就因为大帝得罪你了，大帝得罪你了，阴司又没得罪你，万千生灵又没得罪你，你如果这般小肚鸡肠，那你一辈子也就这样了，你爹在轮回司服役多年，就是想看到你有一天能力挽狂澜，成为这三界中流砥柱，你师父不去投胎、去劳什子修罗界，也是为了等你崛起，挑起大梁、剿灭太阴山！他们真是瞎了眼了！”
<br/>　　“靠，你骂我我听着，别骂我爸和我师父！”叶少阳也嚷起来。
<br/>　　四宝等人赶紧拦在中间劝解。
<br/>　　“老大，徐公说的没错，这事我们干了！”瓜瓜咬牙说道。
<br/>　　“干了，怕他怎的！”橙子和四宝也跟着附和。
<br/>　　叶少阳火气一下子就下去了，冲徐文长拱了拱手，“刚才失态了，虽然你用的是激将法，但也说的没错，我们干了。”
<br/>　　徐文长跟变脸似的，笑容立刻绽开，“好说好说，我就知道你会答应的，之前那些话，我收回。”
<br/>　　叶少阳叹道：“实话跟你说，就算你不说这些，我也会答应的，只是阴司之前那么对我，我有些不爽，而且话说清楚，我这次帮忙，不为阴司，只是为了那些需要投胎的生灵，我不需要阴司给我任何好处。”
<br/>　　徐文长赞许地点点头，“这才像你。”
<br/>　　叶少阳耸耸肩。
<br/>　　四宝举手说道：“我有问题啊，既然我们确定出手，还有岳家军帮忙，为什么不干脆清理掉堵在奈何桥上的那些鬼卒，重新打通进城的路？”
<br/>　　徐文长摇头：“没用，我们之前试过了，太阴山这次是奔着阴司来的，数量之多，你们难以想象，其中还潜伏着许多强者，杀灭一批还有一批，源源不断，但南海海眼不同，那边在酆都城与鬼域大漠的反方向，有酆都城阻隔，他们的兵力无法伸到那边，只能派一支部队过去，我们双管齐下，杀光他们的同时，切断退路，他们再无办法。”
<br/>　　叶少阳皱眉道：“我不明白，既然他们都围城了，绕过酆都城过去支援，不是很容易的事吗？再说，他们哪有源源不断的兵力？”
<br/>　　徐文长道：“你们以为，太阴山大军是怎么攻打过来的？”
<br/>　　“不是一路推过来的吗？”叶少阳问。
<br/>　　“放屁，谁跟你说的？”
<br/>　　“呃，大家都这么说啊，他也这么说。”叶少阳指了指萧逸云。
<br/>　　萧逸云不好意思地干咳了一声，“我也是听说的……”
<br/>　　徐文长道“这支部队是，但那不是主力，吸引了我们在鬼域前线布置的兵力，然后成功拖住，然后，他们真正的大军，是一路从荫水河中潜游过来的，酆都城这才被围，否则你们以为那些前线岗哨和大军都是假的？”
<br/>　　一干人目瞪口呆。
<br/>　　“我操，无极鬼王，够阴的啊。”叶少阳倒吸冷气。
<br/>　　“他们既然发动进攻，就一定是准备充分了，阴司对太阴山了解不多，天知道他们出动了多少兵力……”徐文长担忧地摇了摇头，继而说道：
<br/>　　“荫水河在下方有断流，有貔貅山镇压，他们没法通过河道一直游到海眼去，那兵力就跟不上，因此，海眼是我们唯一的机会。”
<br/>　　大伙这才彻底明白了。
<br/>　　“而今，岳家军已经甩掉追兵，回师来救，预计两天之后能赶到，到时候他们会直奔海眼，你要跟他们汇合，把鬼兵杀光，保护我布阵！”
<br/>　　“保护你？”叶少阳以为自己听错了。
<br/>　　“对，我跟你们一起去，我有大帝信物，一旦布阵，就算是来千军万马，也无法破坏了。”
<br/>　　叶少阳茫然地点了点头，他在考虑怎么打这一架，现在任务是清楚了，但具体怎么干，脑子里还是一团浆糊，朝门外走去。
<br/>　　徐文长追上来，说道：“此事若成，作为答谢，转轮王有厚礼相赠。”
<br/>　　“厚礼？多厚啊，两米厚的纸钱？”
<br/>　　“眼下不便明说，届时小天师就知道了。”
<br/>　　叶少阳也没追问，转头跟橙子和萧逸云说道：“你们回去吧。”
<br/>　　“这次行动，我们也参加！”橙子见叶少阳犹豫，上前抓着他胳膊摇晃起来，“你都好久没带我一起啦，以前就不说了，这次是阴司的行动啊，我们一定要参加，小狼是不是。”
<br/>(本章完)
