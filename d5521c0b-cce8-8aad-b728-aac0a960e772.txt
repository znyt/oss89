　　现场一片死寂，三分之一这个伤亡率对于狼牙战团来说，已经是个极大的数字，这一点鲁初雪和段氏基地的高层们都万分清楚，所以一时间都被吓的面色惨白。
<br/>　　空军死了三分之一，那是多惨烈的状况？！
<br/>　　只是这时候的高少辉又再次开口：“不过楚涵说空军战绩很好，以三十架直升机坠毁的代价赢得了空战的大胜，几万头飞行丧尸也被全部歼灭了。”
<br/>　　静——
<br/>　　众人惊呆了，而后彻底无语！
<br/>　　几名段氏高层刚惋惜出口的话就这么被骤然卡在了喉咙，不上不下最终只能强行咽了下去，还差点把自己噎死！
<br/>　　鲁初雪也在一呆之后表情带上了温怒，明明就是敌军惨烈的战况，眼前这人到底会不会简述战况？
<br/>　　“前半场战役貌似打的很顺利，后半场应该很难赢。”这会儿的高少辉摸了摸下巴，紧接着便冲着鲁初雪道：“不细说了，你赶紧准备物资，我得去找下和上。”
<br/>　　话落高少辉就直径离开，留下原地的鲁初雪等人有些凌·乱。
<br/>　　“这个人说的不清不楚，我们也猜不出南署那究竟如何。”鲁初雪想了想道：“先运一批枪支弹药过去，顺便查探下战况。”
<br/>　　“是。”一人立即点头去办。
<br/>　　此时一名少将开口问道：“南部联盟军那边，要通知消息了吧？”
<br/>　　鲁初雪这才猛然想起这一点，重重点头：“立即传消息过去！”
<br/>　　连距离南署基地最近的段氏基地众人都不知道尸潮已经抵达边境，甚至狼牙战团已经与这群数不清的丧尸战斗了两天，更何况更远方的南部联盟军众将领？
<br/>　　同时很多人也在此时明白为何面对尸潮那些将领会如此紧张，这可怕的暴风式卷席就像是突然出现，悄然无息的就已经开始，如果没有提前准备等他们反应过来的时候，恐怕早已被尸海堵住了去路，最终只能被吞噬。
<br/>　　段氏基地的速度很快，不多时一架直升机就已经升空朝着北部飞去，南方边境的情况需要立即通知各大基地。
<br/>　　另一批直升机队则是晚了几分钟，带着一批后勤部刚准备好的物资朝着南署基地而去。
<br/>　　两座基地的距离不远，并且也是当下为数不多的十几个大型基地中，相互间距离最近的两座。
<br/>　　所以并没有等很长时间，这些运着物资而来的段氏基地成员，就见到了传说中的大尸潮！
<br/>　　他们驾驶着直升机飞在高空，一开始只觉得远方的地面有些诡异，为何一片漆黑，等到飞近了才发现那是密密麻麻拥挤在一起的丧尸群！
<br/>　　甚至数量多到让南署基地都快看不见了，若非那一圈耸立的高墙，段氏基地的这几架直升机恐怕还得再找一会儿。
<br/>　　“这是多少丧尸？”一名飞行员震惊出声，看着下方越来越近的地面，那种扑面而来的恐怖，就好像一个漩涡将他整个人都拉进去搅碎。
<br/>　　其余人也震撼的无法出声，这时候在视野范围内的一切都是丧尸，死的，活的，亦或是半死不活的！
<br/>　　“降落在哪？”一名飞行员紧张发问：“到处都是丧尸啊！不管是高墙还是地面，根本没有空地给我们降落！”
<br/>　　“往里面飞！”此时这几人中的负责人大声开口：“南署基地改造的时候我来过，中心处就是顿放物资的地方。”
<br/>　　几架直升机立即摸索着朝基地最中心而去，他们不敢飞的太高，因为此时下方一片黑色根本分不清是丧尸还是地面还是作战的人类。
<br/>　　而距离地面低空飞行的经历，却让这几个飞行员永生难忘，恐怕这辈子都不会再想重复感受这一次行程。
<br/>　　那些丧尸群就近在咫尺，作战的狼牙战团成员浴血奋战，整个基地都变成了杀与被杀的世界！
<br/>　　当这些人在心惊胆战的经历这些时，远在段氏基地的高少辉，却被科技部的人拦在了一处空屋里足足半个小时。
<br/>　　“别翻白眼了，我都放下研究手头工作过来陪你一起等了。”旁边的江左开口劝着高少辉道。
<br/>　　高少辉显然越来越没有耐心：“和上那货到底行不行啊？一个东西需要准备这么久？”
<br/>　　江左无奈道：“不是普通的研究，楚涵老大都让你亲自过来了，可见一斑。”
<br/>　　高少辉烦躁的在地上走来走去，紧接着又忽的想起了什么问道：“对了，段氏基地怎么也在改造？”
<br/>　　他在飞进基地的时候，恰好看到了苏行带着工程队在段氏基地的一处地方进行着改造，动工还不小！
<br/>　　江左看了他一眼：“你不知道吗？南署基地是大尸潮第一战，段氏基地就是第二战啊！”
<br/>　　高少辉愣住：“还要打？”
<br/>　　江左想了想皱眉道：“据说是会在将来的好几年内不断有丧尸群从南方过来，这里要建立起一条边境防线。”
<br/>　　高少辉这回不仅是愣住，而是惊呆：“这是要在南方边境建造一条长城啊？！”
<br/>　　江左先是一愣，他还真没想到这一点，但是听高少辉这么说，还真有点像。
<br/>　　“先不说这个。”此时的高少辉又忽然转移话题，从兜里掏出一个用布包裹的严严实实的东西，道：“反正和上也不知道要多久，你先带我去找卢宏盛，看看这玩意儿能不能给我打造一下。”
<br/>　　江左也没想太多，直接带着高少辉就去了卢宏盛他们锻造部的车间。
<br/>　　哪怕是从狼牙基地转移过来的锻造部，这会儿的仪器也应有尽有，最关键的是卢宏盛直接带着人霸占了段氏基地面积最大的大楼。
<br/>　　段江伟一开始建造出来作为礼堂的地方，在卢宏盛到来的短短几天内就早已变了样，丝毫不见之前的繁华，彻底变成了满是黑炭和碎屑乱飞的工厂。
<br/>　　高少辉倒是觉得很正常，没有这点变化怎么打造兵器？
<br/>　　只是当他进入这车间一路走过，渐渐越来越靠近深处的时候，高少辉的心情则就从淡定一下子转变了一百八十度。
<br/>　　他停在一处根本看不出有什么作用的仪器面前，表情有些扭曲，而后实在忍无可忍的大吼出声：“楚涵！你什么时候把白忧整个岛的东西都搬了过来？！”
