　　　　离开这家面馆，涂小安闷闷不乐。
<br/>　　于是乎换了一个思维，他进面馆吃了一碗面，还看了一具极品大美女的酮体，一分钱没花。
<br/>　　这如何算，自己都不吃亏呀。
<br/>　　他没道理心里不爽。
<br/>　　况且不管自己的身体能不能用，他本就不是一个急色的人。
<br/>　　好吧，这样一安慰自己，涂小安心里的不爽也就抵消了，最后回头看了一眼面馆，将落子烟这个名字暗暗深记。
<br/>　　继续顺着这条古街往外走，还没走多少步，涂小安经过了味鲜阁的旁边，只见味鲜阁的大门口有好几辆的警车停着，更有不少警员进进出出。
<br/>　　这咋的一看，不知道的还以为味鲜阁在被查封呢。
<br/>　　很快，涂小安看到了门口的一位熟人，他就是味鲜阁的大老板苏羽。
<br/>　　苏羽西装笔挺，但却一脸阴郁的跟一位警员录口供，看样子好像是店里进贼了，丢了什么东西。
<br/>　　涂小安观望了一会儿，见苏羽跟警员录完口供，他走过去道：“苏老板，出什么事情了”
<br/>　　苏羽看到涂小安有点诧异，但还算和悦，摇了摇头叹气道：“涂兄弟，看来你昨天是白忙活了”
<br/>　　“这话怎么说”
<br/>　　“我店里的龙蛇半夜被人偷走了”
<br/>　　苏羽很气愤，这叫什么事情，有听过偷金偷银的，但这半夜来偷一条蛇是什么鬼。
<br/>　　这龙蛇不单单价值不凡，更是关乎这他苏羽的气运，他是商人，商人最讲气运了。
<br/>　　如今龙蛇没了，苏羽感觉自己这个味鲜阁也可以关门大吉了。
<br/>　　现在虽然报了警，但他也不奢望这些警察能将他的蛇给找回来。
<br/>　　咦...龙蛇被偷？
<br/>　　涂小安闻言，一脸的古怪，一时半刻都不知道说点什么好。
<br/>　　偷蛇？好一个有眼力劲的小偷呀。
<br/>　　看似偷条蛇是很滑稽的事情，但要知道龙蛇可是一条珍稀蛇种，价值五百万，更有特殊的意义，将龙蛇偷到手，比去偷金店更一本万利。
<br/>　　隐晦间，涂小安都有点同情这位苏大老板了，买龙蛇花了五百万，昨日又让自己治蛇，再次掏了五百万，支票现在还在他身上呢。
<br/>　　只是这五百万，算是买他的蛇皮。
<br/>　　但终归，他是竹篮子打水一场空，替别人做嫁衣。
<br/>　　“该死的小偷，连我苏羽的东西都敢偷，千万不要被我找出来”
<br/>　　苏羽精光四射，看来是动了真火了。
<br/>　　涂小安看着他的样子，忽然脑子灵光一闪，想起了之前在面馆吃的那碗肉丝面。
<br/>　　肉丝面用蛇汤底，而且用的不正是一条珍稀品种的蛇吗。
<br/>　　面汤鲜美无比，蕴含一股能量，涂小安一口就喝出来了。
<br/>　　莫非...蛇汤底是用龙蛇熬的？
<br/>　　猛然间，涂小安不禁倒吸了一口凉气，这也太疯狂了吧。
<br/>　　稍稍这么一想，涂小安觉得事情八九不离十就是这样。
<br/>　　珍稀蛇种，一蛇难求。
<br/>　　那边面馆用珍稀蛇种熬汤，这边味鲜阁丢了一条珍稀蛇种。
<br/>　　事情不会那么的巧合。
<br/>　　况且面馆的老板娘之前也跟涂小安说过，说自己昨晚得到了一条不错的蛇种，所以就用蛇汤下面了。
<br/>　　另外，面馆跟味鲜阁之前相差不过十来间店铺的距离，近月楼台先得月，偷起来也方便。
<br/>　　“涂兄弟，你怎么不说话”
<br/>　　苏羽见涂小安在发呆，开口询问。
<br/>　　“哦，没什么，只是替苏老板觉得可惜，多好的龙蛇就这样没了”
<br/>　　如果苏羽知道关乎自己气运的龙蛇有可能是被人做成了面汤，不知道会不会气的吐血。
<br/>　　但涂小安还不是特别的肯定，需要亲自去验证一下事情的真相。
<br/>　　简单的安慰了几句苏羽，在加里他店里又有不少警察找线索，他就找个了不打扰的借口离开了。
<br/>　　一离开，涂小安往回走，再次来到了人满为患排队的面馆前。
<br/>　　但这次，他可没有心情排队，直接来到了队伍的最前沿，准备直接进去。
<br/>　　这一举动，顿时引起了现场排队人的不满跟吃喝。
<br/>　　“喂，你这个年轻人怎么回事，排队懂吗”
<br/>　　“一点素质都没有，还想插队”
<br/>　　“让他插队吧，等下有他好果子吃”
<br/>　　为什么大家都那么乖乖的排队，因为你插队也没用，前面有个高大威猛守门的大汉等着你。
<br/>　　你要是不服从现场的秩序，二话不说，直接给你丢出去。
<br/>　　果不其然，涂小安被守门的大汉给拦了下来，正当大家幸灾乐祸的准备看热闹的时候，只见涂小安张口不知道说了什么，高大威猛的大汉既然神奇的将通往里面的路给让了出来。
<br/>　　涂小安抬头挺胸，光明正大的进去了。
<br/>　　顿时间，现场排队的人一个个目瞪口呆，这暴躁的大汉什么时候变的那么好讲话了？。
<br/>　　太阳打西边出来了啊。
<br/>　　这时，有一个瘦小的男子，也想学涂小安投机取巧，直接冲过去，准备进去。
<br/>　　可没一会儿，他已一个诡异的弧线被高高的抛了出去。
<br/>　　惹的现场一片哗然...
<br/>　　涂小安再次进面馆，只见落子烟再次恢复成了寡妇脸的丑样，正在笑眯眯的跟一些吃面的客人，插科打诨。
<br/>　　故作骚态，媚笑不止。
<br/>　　可惜她现在这副雀斑跟皱纹的丑脸，实在让吃面的客人有苦说不出。
<br/>　　辣眼睛，太辣眼睛了。
<br/>　　这时，老板娘见下一波进来的客人却是涂小安，顿时意外了。
<br/>　　涂小安快步的走过去，将寡妇脸的她拉到了最里面，但不是小卧室，刚想开口，却被对方强先了。
<br/>　　“呦，小哥哥，莫非刚才没得到我的身体，心有不甘？”
<br/>　　涂小安大大的白了一眼，她可别用这副样子跟自己说这些话。
<br/>　　不知道恶心人吗。
<br/>　　“前面味鲜阁的龙蛇被偷了，是不是你所为”
<br/>　　寡妇脸的她，看了眼涂小安：“是呀，怎么了”
<br/>　　呃...
<br/>　　涂小安呆滞。
<br/>　　本以为她会狡辩一下，可随便一问，对方直接承认，反倒将他弄个措手不及。
<br/>　　这神秘的女人，还真是从不按套路出牌呀。
<br/>　　偷人东西，还那么的坦荡，整的跟拿一样。
<br/>　　“你为什么要偷龙蛇，那可是别人的镇店之宝”
<br/>　　寡妇脸的落子烟道：“都是一条街上做生意的，自然要落井下石，偷了龙蛇，味鲜阁的生意不就一落千丈了”
<br/>　　“你以为我会相信你这种鬼话”涂小安冷笑起来。
<br/>　　别人是开大酒楼的，你一个开面馆的，还担心别人抢你生意？
<br/>　　这也太给自己面子贴金了。
