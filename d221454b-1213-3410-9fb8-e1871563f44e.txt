　　杨国威虽然一向行事飞扬跋扈，却也不是完全没有脑子。
<br/>　　他当然知道，那位云州叶神仙的分量有多重！
<br/>　　他们杨家这半年来，之所以突飞猛进，实力跻身东江省前五，甚至可以将其他各大家族不放在眼里。
<br/>　　就是因为得到了紫金仙泉的销售权，攀上了这尊大佛！
<br/>　　相比之下，他杨国威不过就是一个小小蝼蚁，人家叶仙人就算把他踩死了，他大哥杨国强也不敢多说一个字。
<br/>　　一念及此，杨国威之前心头的怒气，瞬间便消失的无影无踪，取而代之的，是无尽的懊悔和恐慌，
<br/>　　“大、大哥，你说那位叶神仙，会不会秋后算账？我要不要亲自去云州一趟，去登门请罪？”
<br/>　　杨国强深吸了一口气道：
<br/>　　“早知今日，何必当初？不过，叶仙人是何等身份？既然他说饶了你，那便是真的饶了你，应该不会再寻你的麻烦了！不过……”
<br/>　　杨国强说到这里，微微一顿，
<br/>　　“那个庄菲菲，还真是不容小觑，没想到她居然攀上了这颗大树，以后该怎么做，就不用我教你了吧？”
<br/>　　杨国威连忙点头如啄米，
<br/>　　“大哥放心！我这就召集公司高层，传令下去，今后一定把她当祖宗一样供着！”
<br/>　　……
<br/>　　却说叶尘骤然离开后，庄菲菲心里就一直空落落的，一直回到住处，依旧有些闷闷不乐。
<br/>　　就在这时，手机忽然响了，
<br/>　　庄菲菲拿起一看，居然是华艺天皇娱乐的副总裁打来的。
<br/>　　庄菲菲顿时眉头一皱，心中寻思着，
<br/>　　杨国威被叶尘废掉了一手一脚，此时对自己只怕已经恨之入骨了，看来自己是没法在华艺天皇娱乐继续待下去了，不如我识趣点，主动离开，也省的以后再被人算计。
<br/>　　一念及此，庄菲菲接通电话，直接开口道：
<br/>　　“徐副总，是杨总让你打电话过来的吧？请你转告他，我打算和贵公司解除合约，违约金我也会照价赔偿，请他不必再费心思对付我了！”
<br/>　　说完这话，庄菲菲便要将电话直接挂断。
<br/>　　不料，
<br/>　　电话那头的徐副总顿时急了，
<br/>　　“别呀！菲菲小姐，您这是说的什么话！您是不是和杨总有什么误会？”
<br/>　　“的确是杨总让我打电话来的，不过他老人家可说了，今后我们华艺天皇娱乐的全部资源，都由菲菲小姐任意支配！”
<br/>　　“这，这是什么意思？”
<br/>　　庄菲菲顿时愣住了。
<br/>　　徐副总继续道：
<br/>　　“意思就是，只要菲菲小姐您愿意，您可以调动公司的一切力量为您服务！”
<br/>　　“之前您不是一直想发行专辑吗？公司会聘请最好的音乐团队，为您量身打造最适合您的音乐专辑！”
<br/>　　“您要是想拍电影，但凡公司旗下的影视资源，您可以随意挑选，公司旗下的所有艺人，您也可以随意指派给你搭戏！”
<br/>　　“对了，咱们公司最近刚接了一个，和米国好来坞合作的大片，只要您愿意，女主角就是您的，你看这样可好？”
<br/>　　徐副总似乎生怕庄菲菲和公司解除合约，一口气说了一大堆，说话的语气态度，也没有了平日里的高冷，恭敬之中带着恳求，甚至都有些卑微！
<br/>　　庄菲菲又哪里知道，就在不久前，杨国威紧急召集公司所有人，召开了一场电话视频会议，直接对众人下达了死命令。
<br/>　　今后华艺天皇娱乐，要不惜一切代价力捧庄菲菲，谁要是敢惹这位姑奶奶不高兴，就立马卷铺盖滚蛋。
<br/>　　杨国威的这一命令下达之后，顿时遭到几位高层的反对和质疑。
<br/>　　然而，
<br/>　　让众人万万没有想到的是，杨国威根本连听都不听，直接将质疑之人全部罢免，甚至这其中，还有两位公司元老级的人物！
<br/>　　一时间，
<br/>　　整个华艺天皇娱乐人心惶惶，简直都要炸锅了。
<br/>　　没有人知道，杨国威为什么会突然下达，如此荒谬的决定。
<br/>　　可是杨国威的态度，却异常的坚决，甚至容不下丝毫的质疑！
<br/>　　无奈之下，公司众高层，只能选择接受这一事实。
<br/>　　听完徐副总的话之后，庄菲菲足足愣了好一会，方才反应过来，不过却仍然有些难以置信。
<br/>　　毕竟，徐副总所提出的条件，实在是太过离谱，即便是那些背景通天的超级巨星，也远远没有如此优厚的待遇啊！
<br/>　　‘难道这一切，都是因为他……’
<br/>　　庄菲菲脑海之中，突然跳出了叶尘的身影，
<br/>　　虽然她知道叶尘的身份不简单，却也万万没有想到，竟然让杨国威畏惧到如此地步。
<br/>　　电话那头，徐副总见庄菲菲迟迟不吭声，顿时急的不行。
<br/>　　虽然他并不清楚，杨国威到底在发什么疯，可是他已经接到死命令，倘若庄菲菲执意要离开公司，他肯定也要跟着完蛋了！
<br/>　　“菲菲小姐，我的姑奶奶嗳！我知道我之前对您态度不好，您大人有大量，就不要和我一般见识了好不好？只要您不离开公司，今后您让我干什么都成！”
<br/>　　平日里高高在上的徐副总，此时态度简直卑微如奴才，就差要给庄菲菲跪下了。
<br/>　　庄菲菲的自尊心得到了极大满足，但还是觉得不太真实，一时间脑子有些反应不过来，于是便随口道：
<br/>　　“你容我考虑一下！”
<br/>　　说完，便直接挂断了电话。
<br/>　　电话那头徐副总，顿时气得差点吐血。
<br/>　　然而，
<br/>　　让庄菲菲万万没有想到的是，大约半个小时之后，徐副总竟然亲自赶了过来。
<br/>　　一见面，徐副总直接“噗通”一声，就跪倒在了地上，庄菲菲和红姐两人全都看傻了眼。
<br/>　　“徐副总，您这是做什么？”
<br/>　　徐副总哭丧着脸道：
<br/>　　“菲菲小姐，您就当是行行好，千万不能和公司解约啊，否则杨总肯定饶不了我的！我求求您，求求您……”
<br/>　　庄菲菲和红姐两人顿时哭笑不得，同时心头又无比震撼，
<br/>　　这徐副总何许人也？
<br/>　　那可是华艺天皇娱乐的真正高层，娱乐圈巨头级的人物，就算是那些超级明星见了他，也得客客气气的，不敢有丝毫得罪。
<br/>　　如今居然跪在地上，哀求庄菲菲这个娱乐圈的新人，不要和公司解约？
<br/>　　这事说出去，恐怕任谁也不会相信吧！
<br/>　　而这一切，全都是因为那个男人！
<br/>　　恍然间，庄菲菲的脑海之中，再次跳出那道瘦削的身影，美眸深处泛起一抹甜蜜。
