<br/><br/>　　    落款写着嘉靖丙申秋七月二十三日书，徵明。后面跟着一个红色篆刻徵明两字的小章。<br/>　　<br/>　　    册页上的字不大，行书体，很有王羲之行书的风格，温润秀劲，法度谨严而意态生动。虽无雄浑的气势，却具晋唐书法的风致，也有自己的一定风貌。<br/>　　<br/>　　    这竟然是明朝四大才子之一的文徵明的作品，文徵明号称诗书画三绝，他的行书作品历经几百年能够流传下来自是价值不菲。<br/>　　<br/>　　    徐正笑着将册页递还给苏南，推辞说：“苏董玩笑了，这是文徵明的字，我就再没有见识，也知道是价值不菲之物，这我可不能收。”<br/>　　<br/>　　    苏南笑了，说：“这徐市长就有所不知了，这个册页是不是文徵明的真迹还很难说，专家说存疑，清朝有名的书画著录书《石渠宝笈》对这个册页没有丝毫记载。我带它过来，不是说他值多少钱，只是取其正气二字。试问这正气二字不正贴合您的心境吗？您如果不收，那就是嫌弃它不够贵重了？”<br/>　　<br/>　　    徐正看得出来这个册页古意盎然，字迹精妙，绝非假货，即使不是文徵明亲笔所写，可能也是跟文徵明时代相近的后人的摹本。他心中也是很喜欢，不过他还是有些不好意思就这么接下来。<br/>　　<br/>　　    徐正又把册页往外推了推，说：“这不好吧？”<br/>　　<br/>　　    苏南听徐正的语气弱了下来，便知道他还是有些心动了，就又将册页推了回去，说：“我刚才听徐市长一番慷慨陈词，觉得我带这份礼物还真是带对了，也只有徐市长才衬得起这正气二字，您如果再推辞，就是看不起我了。”<br/>　　<br/>　　    徐正笑了，说：“叫苏董这么一说，我还真是不好意思推辞了，那我们就以正气这两个字共勉吧。”<br/>　　<br/>　　    苏南笑笑说：“对，对，共勉。”<br/>　　<br/>　　    两人相视一笑，这个册页就放在了徐正面前，再也没有互相推来推去了。<br/>　　<br/>　　    两人又闲聊了一会儿，看看时间已经是中午了，徐正就留苏南一起吃饭，苏南正要趁机跟徐正混熟，假意推辞了几句，就答应了下来。<br/>　　<br/>　　    徐正领着苏南去了海川的西岭宾馆，一进门正好看到吴雯在大堂里吩咐服务员做事，徐正笑着说：“吴总，今天没去工地啊？”<br/>　　<br/>　　    吴雯笑着说：“您好啊，徐市长，工地上已近收尾阶段，不用我时时看着了，今天就没过去。”<br/>　　<br/>　　    徐正笑着说：“怎么样，销售状况如何？”<br/>　　<br/>　　    吴雯笑着说：“基本上都被订走了，销售状况良好，这还应该多谢徐市长您的支持啊。”<br/>　　<br/>　　    徐正笑着摇了摇头，说：“”是你的房型适合了市场的需求，与我无关的。吴总，你不是在北京呆过吗？今天我邀请的这位客人也是北京来的，振东集团的董事长苏南先生，不知道你是否认识？<br/>　　<br/>　　    吴雯笑笑说：“北京大着呢，我哪能每个人都认识啊？欢迎你来做客，苏董。”<br/>　　<br/>　　    苏南虽然是振东集团的董事长，不过他向来很少出席风月场所，因此对这个曾经艳名满京城的花魁并不认识，他笑着跟吴雯握手，说：“想不到在海川能够见到这么美丽的老板娘，幸会了。”<br/>　　<br/>　　    吴文笑笑，说：“苏董真会说笑。”<br/>　　<br/>　　    徐正笑着说：“现在你们算是认识了，一会儿你可要过来多敬苏董几杯啊。”<br/>　　<br/>　　    吴雯笑着说：“行啊，徐市长你们先过去，我把这边布置完了，就过去。”<br/>　　<br/>　　    徐正就领着苏南去了雅座，双方这是第二次坐在酒桌旁，少了很多拘礼，气氛活跃了很多。<br/>　　<br/>　　    喝了一会儿，吴雯来了，进门就赔不是说：“不好意思，徐市长、苏董，我酒店的杂事太多，忙到现在才过来。”<br/>　　<br/>　　    徐正笑笑说：“来了就好。”<br/>　　<br/>　　    吴雯说：“来，我先给各位满上。”<br/>　　<br/>　　    吴雯就给众人把酒斟满，然后笑着说：“还不知道苏董这一次来海川是做什么？”<br/>　　<br/>　　    苏南笑笑说：“我是顺路来看望徐市长的。”<br/>　　<br/>　　    吴雯就端起酒杯，笑着说：“来这杯酒欢迎苏董这位朋友来我们海川市。”<br/>　　<br/>　　    众人喝的是白酒，苏南面有难色，他不想一口就干掉这一大杯白酒，吴雯笑了，说：“苏董啊，你不会害怕我这个女流之辈吧？”<br/>　　<br/>　　    苏南笑笑说：“现在是巾帼不输须眉，女人在酒桌上能抬起杯来的，都是好酒量的。”<br/>　　<br/>　　    徐正笑着说：“苏董啊，被人敬的酒你不喝我不好说什么，我们这么美丽的女主人敬的酒你也不喝，也太说不过去了。”<br/>　　<br/>　　    苏南见这架势，知道这杯酒逃不过，便笑着说：“那我舍命陪君子了。”<br/>　　<br/>　　    苏南就和吴雯碰了杯，吴雯一口就杯中酒干掉了，苏南也没示弱，跟着也喝干了杯中酒。<br/>　　<br/>　　    吴雯笑着说：“苏董真是痛快，来我再给你满上。”<br/>　　<br/>　　    吴雯又将酒斟满，说：“苏董，我曾经在北京呆过几年，我的第一桶金就是在北京赚取的，我对北京有着很深的感情，今天见到你这个北京来的贵客，感到特别亲切，可以这么说我算半个北京人，也就跟您算是半个老乡，老乡见老乡，两眼泪汪汪，我怎么说也要在跟你干一杯。”<br/>　　<br/>　　    苏南笑了，说：“吴总，我真是佩服你这敬酒的口才了，不过，我可真是不能再一杯全干掉了，这样，我们到中央。”<br/>　　<br/>　　    吴雯笑笑说：“苏董不愧是北京来的贵客，一说就是到中央，不过您今天来了海川，不是到了中央，而是屈尊到了地方，这杯酒没说的，一定要干掉。”<br/>　　<br/>　　    苏南笑着看了看徐正，说：“徐市长啊，您今天真是找了一个好地方啊，你们这里的女将厉害啊，让人抵挡不住啊。”<br/>　　<br/>　　    徐正笑笑说：“没办法，吴总确实很能干，再说这里是吴总的地盘，这酒我也帮不了你。”<br/>　　<br/>　　    吴雯笑笑说：“对啊，我的地盘我做主，苏董，不会这么点面子都不给我这个主人吧？”<br/>　　<br/>　　    苏南有些无奈，只好再次跟吴雯干掉了杯中酒，不过，吴雯再要给他添酒，他说什么也不肯了，坚持说再这么喝他就要出洋相了。<br/>　　<br/>　　    最后徐正说和，苏南才肯让人再添了半杯酒。<br/>　　<br/>　　    酒桌上加入了女人，又是这么漂亮的女人，气氛就更活跃了起来，吴雯在其中敬这个，敬那个，大家喝得不亦乐乎。<br/>　　<br/>　　    喝完酒，苏南就跟徐正说要离开海川回北京，徐正笑着说：“苏董，好不容易来一趟，多住几天吧？”<br/>　　<br/>　　    苏南笑着摇了摇头，说：“我也想多玩几天，可是集团那边很多事等着我处理呢。”<br/>　　<br/>　　    徐正说：“那我就不好再留了，保持联系。”<br/>　　<br/>　　    苏南笑笑说：“保持联系。”<br/>　　<br/>　　    吴雯也出来送苏南，笑着跟他握手告别，说：“一路平安啊，苏董。”<br/>　　<br/>　　    苏南笑笑说：“谢谢吴总盛情款待了。日后会北京别忘了去找我啊。”<br/>　　<br/>　　    吴雯笑笑说：“怎么，苏董的酒还没喝够吗？”<br/>　　<br/>　　    众人都笑了起来，苏南就上车离开了，徐正也跟吴雯说了再见，回了市政府。吴雯送走了这些人，回到了办公室，给刘康拨了一个电话，刘康接通了。<br/>　　<br/>　　    吴雯笑着说：“干爹，这一向身体还好吗？”<br/>　　<br/>　　    刘康笑笑说：“还行吧，你那边工程进行的怎么样了？”<br/>　　<br/>　　    吴雯笑笑说：“进展顺利，已经快收尾了。”<br/>　　<br/>　　    刘康笑笑说：“那就好。今天打电话找我干什么？”<br/>　　<br/>　　    吴雯说：“是这样，干爹，北京有个振东集团你知道吗？”<br/>　　<br/>　　    刘康说：“我知道啊，挺大的一个公司，很有实力。怎么了？”<br/>　　<br/>　　    吴雯笑笑说：“他们的董事长刚刚从我这个西岭酒店离开，这个人真是好风度啊，喝了那么多酒，行为举止一点都不乱，一副贵公子的架势。”<br/>　　<br/>　　    吴雯对苏南印象不错，因此才会向刘康说起他。<br/>　　<br/>　　    刘康惊讶的叫了一声：“苏南去海川了？”<br/>　　<br/>　　    吴雯愣了一下，说：“干爹认识苏南？”<br/>　　<br/>　　    刘康说：“我知道这个人，你说他一副贵公子的架势，一点没错，他就是那样，走到哪都是众人瞩目的焦点。他没说去海川做什么吗？”<br/>　　<br/>　　    吴雯说：“是徐正请的客，苏南自己说是顺路来看徐正的。”<br/>　　<br/>　　    刘康迟疑了一下，说：“这家伙跟徐正搭上关系了？”<br/>　　<br/>　　    吴雯说：“是，看样子，俩人还很亲近。”<br/>　　<br/>　　    刘康说：“这家伙，这么早就了。”<br/>　　<br/>　　    吴雯笑笑，说：“干爹啊，什么这么早就了？我看苏南就是路过而已，喝完酒之后他就赶回北京了。”<br/>　　<br/>　　    刘康笑了，说：“他是振东集团的董事长，每天忙得不得了，才没闲工夫去路过你们海川呢。”<br/>　　<br/>　　    吴雯说：“干爹，你别打哑谜了，究竟怎么回事啊？”<br/>　　<br/>　　    刘康说：“我想苏南肯定是在打海川新机场项目的主意，他振东集团旗下有一家机场建设公司，在机场建设方面很有实力。”<br/>　　<br/>　　    吴雯说：“海川新机场项目我知道，听不少来吃饭的官员们聊起过这个项目，不过好像还没正式立项。”<br/>　　<br/>　　    刘康说：“这个项目涉及几十个亿的规模，是一个很大的项目，所以就把苏南这样的贵公子也引到了你们海川去。现在是项目立项在即，苏南跑去海川，就是为了争取这个项目预作布局的。哼哼，这家伙想得到美。”<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
