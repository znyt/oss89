　　江行泽没想到自己也有一天能成为人类觉醒者。
<br/>　　什么是觉醒者，他其实并不是很清楚，但他清楚一点就够了，他师父蛇博士陈默就是一位人类觉醒者。
<br/>　　原本江行泽不断的吸食吸血蛇的蛇血，气力日渐增强，当他一次意外跟吸血蛇发现了这种红彤彤的小果实，他吃下一颗，既然打开了身体的桎梏，直接成为了人类觉醒者。
<br/>　　身体被淬炼，耳聪明目，整个人都完全变的不一样，全身充满了力量。
<br/>　　更为惊奇的是，江行泽发现吃了这种红彤彤的小果实，体内的蜂毒好像被解了不少，就算几天没喝吸血蛇的蛇血，也不会发作。
<br/>　　震惊，兴奋，他以为这辈子都解不了的蜂毒，有了痊愈的机会。
<br/>　　另外吸血蛇吃了那种小果实，体型也得到暴涨，现在都是二十五米的巨蟒了。
<br/>　　对于蛇类来说，身体每成长一米，都是重大突破。
<br/>　　于是乎，江行泽跟吸血蛇朝着蛇沟深处探索，为了就是得到更多这种小果实。
<br/>　　渐渐的，它们发现这种小果实会发出一种异香，非常吸引蛇类，简直就是蛇类一脉的圣果，吃了能脱胎换骨。
<br/>　　然后就将这种称之为蛇莓。
<br/>　　“吸血蛇，想必蛇莓就要这蛇谷的里面了，快进去，不能被别的毒蛇给抢走”
<br/>　　蛇谷虽美，但也是危机四伏，有很多散发强大气息的毒蛇，它们也都想得到蛇莓。
<br/>　　沿途前进，很快，江行泽目瞪口呆的看到前方一片区域居然被数万条毒蛇所覆盖，那一条条不管大小，摸样都非常狰狞恐怖，特别是由于种类不一，颜色各异，瞬间便把蛇谷染成了一片炫彩海洋……
<br/>　　整个蛇谷的气温更是低的可怕，数万冷血蛇种汇聚在一起，而且还不是一般的凡蛇。
<br/>　　因为能从充满死亡气息的沼泽进来，绝非泛泛之辈。
<br/>　　不够资格的，都淹没在沼泽之口。
<br/>　　它们就算不是觉醒类蛇种，也都是有一定的灵智了。
<br/>　　数万毒蛇的前方还有一座小山丘，高高从地面凸起，蛇莓独特的异香就是从这小山丘散发出来。
<br/>　　江行泽现在是独眼龙，但就算只有一只眼睛，也迸发明亮的光彩。
<br/>　　而他座下的吸血蛇更是凶光大亮，对蛇莓也是充满贪婪。
<br/>　　有时候动物对一些天材地宝更为的敏锐，一般有宝物出世，肯定有什么强大的生物在旁守护，不让人觊觎。
<br/>　　忽然，前方有一条庞大的黑色巨蟒拦住了它们的去路。
<br/>　　这黑色巨蟒穷凶极恶，蛇眸之中闪烁着寒光，看着一人一蛇，似乎在说，想过去，就必须先过我这关。
<br/>　　“吼~！”
<br/>　　吸血蛇何等凶煞，一声怒吼，天地都要为之变色，火红鳞片似有火焰在窜动，一股恐怖如斯的气息席卷出来。
<br/>　　霎时间，前方数万毒蛇被这道嘶吼声震惊了，不少蛇种都开始诚惶诚恐起来。
<br/>　　在看这条黑色巨蟒，穷凶极恶的样子瞬间消失不见，庞大的蛇躯开始颤栗，它立刻朝着吸血蛇，低下了自己高傲的蛇颅，静静俯在大地之上，等待着强者的宣判。
<br/>　　在生物界，并不是都要打上一架才知道谁强谁弱，本身的气息已经能说明很多问题。
<br/>　　吸血蛇的气息一散发，黑色巨蟒不由的就畏惧起来。
<br/>　　那么就高下立判。
<br/>　　“黑蟒，还不给我滚开，敢挡小爷的路，你是活腻了”
<br/>　　在吸血蛇身上的江行泽对着黑色巨蟒破口大骂，嚣张的不可一世，自打进入蛇沟，就是遍地毒蛇。
<br/>　　很多毒蛇强大的他都想象不到。
<br/>　　但他不怕，因为他有吸血蛇，碾压一切。
<br/>　　黑色巨蟒匍匐在地，不敢动弹。
<br/>　　吸血蛇动了起来，经过黑色巨蟒身旁之时，它的一条蛇尾骤然出手，瞬间将这条已经臣服的黑色巨蟒给抽的血肉模糊。
<br/>　　敢拦它，就是死。
<br/>　　传闻中的变异吸血蛇，刚出世就无比凶残，更别说现在了。
<br/>　　前方几万毒蛇骚动起来，因为它们明白，又来了一个强敌来争抢天材地宝。
<br/>　　“吸血蛇继续走，谁敢阻拦，就杀了它”
<br/>　　蛇莓在前，江行泽整个人都兴奋起来，这些畜生还想跟他争夺宝贝不成，找死。
<br/>　　吸血蛇二十五米的庞大蛇躯目空一切的前进，身上恐怖的阴冷气息，让一些毒蛇自动的让路，躲得远远的，根本不敢冒犯。
<br/>　　整整二十五米，加上摄人恐怖的气息，但凡不傻的蛇，都不敢主动找吸血蛇的麻烦。
<br/>　　这何止是巨蟒了，在人类眼中，或许十米以上的早就称为一个巨了。
<br/>　　况且吸血蛇一身火红鳞片，就宛如钢铁铸就的一样。
<br/>　　它现在太强大了，有遇神杀神，遇佛杀佛的气势，睥睨一切，任何蛇种都不放在眼中。
<br/>　　不过数万毒蛇里面，也不是没有二十多米的巨蟒存在，也不是谁都被吸血蛇的气息吓到，它那么嚣张，背上既然还拖着一个人类，非常的吸引注意力。
<br/>　　这里是蛇谷，岂能出现人类。
<br/>　　“吼”
<br/>　　一条犹如蛟龙般的二十四米巨蟒飞射了过来，张开血盆大口，简直可怕之极，直接朝着吸血蛇吞噬而去。
<br/>　　江行泽看到此一幕，吓的差点从吸血蛇的背上掉下来。
<br/>　　他现在虽然成为人类觉醒者，但要是没有吸血蛇保护，他在这个蛇谷中，活不过一分钟。
<br/>　　二十四米的巨蟒气势滔滔如海，很多强大的蛇种开始看热闹，甚至很多对江行泽这个人类很感兴趣。
<br/>　　这座蛇谷中别的动物也都见到，但唯独人类百年难得一见。
<br/>　　人肉的味道应该是极美的。
<br/>　　有蛇出来挑战它，吸血蛇一双蛇眸凶光乍现，流露出兴奋，火红色的蛇尾席卷出去，空气层层爆响，一尾之力不知道蕴含多少力量，这条二十四米的巨蟒还腾在半空中就被吸血蛇的一尾给活生生抽飞十几米。
<br/>　　轰然倒地，地面震动，它倒在地上发出哀鸣，既然半分动弹不了，生死不知，看样子出气多进气少，凶多吉少了。
<br/>　　这一尾的威力彻底镇住了数万蠢蠢欲动的蛇种，二十四米的巨蟒在它们之中可谓是强者的存在。
<br/>　　可一招就败了，看样子完全不是一个级别的。
<br/>　　可怕，太可怕了。
