蛇并没有投降或者地头认输，给人的感觉就是要使有机会，还是会偷跑的。<br/>　　<br/>　　但是就凭借它那有限的智商，还真的是有些搞不明白自己身处的地方。另外就是整天都是昏迷中，即使清醒，也会被陈默扔进灵兽养殖区接受~精神洗礼，所以打蛇能够坚持到现在，还真的是韧性十足。<br/>　　<br/>　　也可以说这样的韧性，也是靠地下空间阵法的培养，要不是时不时的刺激一下打蛇的精神识海，它还真的不可能开窍进入修炼！<br/>　　<br/>　　不过这条大蛇的修炼有没有什么脑子，也没有什么修炼秘籍，就是靠着动物的本能在修炼。<br/>　　<br/>　　所以上千年下来，也就这个样子了，不过对于普通人来说，绝对是非常可怕的存在，但是对现在的陈默来说就不足了！<br/>　　<br/>　　陈默其实想收复这条大蛇来着，就是想收个看家护院的存在。再说这次地下空间的事情，也让他感觉到地球没有那么普通。<br/>　　<br/>　　以前还以为地球没有修真者，却没有想到不是没有，而是灵气匮乏后修真者都离开了！<br/>　　<br/>　　因此现在的修炼体系，都是修真者后代中，一些人另辟蹊径开发的。不过陈默猜想，可能随着时间的流逝，也随着科技的不断发展，很多修炼功法都渐渐丢失了。<br/>　　<br/>　　剩下的，也就可能再次经过改良等等，才产生了武者这个概念。从地球上其他国家，修炼者不同的情况来看，可能自己猜测的八~九不离十！<br/>　　<br/>　　不过不管如何，自己实力高才是最主要的。所以现在趁着在家里的这些天，先将神识给拓展上去，巩固自己练气十层的巅峰状态。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
