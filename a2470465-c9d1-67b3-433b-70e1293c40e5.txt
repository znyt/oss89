　　毛小方道：“不说这些了，这姑娘……你打算怎么安排？”
<br/>　　叶少阳瞥了云儿一眼，大声笑道：“大妹子，你自由了！”
<br/>　　“自由……”这个词对这个时代来说，还是一个非常时髦的词，云儿转过脸来看叶少阳，突然哇的一声，趴在他怀里放声哭了起来。
<br/>　　曾经以为这辈子都没希望了，实在没有想到，这个陌生人的出现，突然挽救了自己，改变了自己的命运……
<br/>　　三年的委屈和酸楚，在这一瞬间爆发出来，云儿哭成了泪人。
<br/>　　“这……”叶少阳不知道怎么安慰，有点手足无措，看看毛小方，说道：“那个，我不太会安慰人，不然你趴他怀里哭一会？”
<br/>　　“不不不，”毛小方连忙摆手，“安慰姑娘这种事，适合你这种年轻人，我这么老，就算了。”
<br/>　　云儿哭了好一会，总算缓和下来，慢慢地止住了哭泣。
<br/>　　叶少阳这才问道：“云儿妹子，你现在自由了，你有什么打算，想要回家吗？”
<br/>　　云儿听他这么问，又怔怔地发了会儿呆，说道：“我不知道，我不想回家了……”
<br/>　　“为什么？”叶少阳脱口而出，立刻就想到了，当年她的父母把她送给五通神，虽然对他们来说这可能也是一种牺牲，不过，这三年来她所受到的摧残，对自己的父母肯定恨得要死，自然也不想回家去了。而且她家肯定就在附近镇上，离五通神的山神庙很近，就算她回家，肯定也会担心五通神再把她抓回去。
<br/>　　“我，我不想再在这地方生活了，只想走得越远越好……”
<br/>　　叶少阳点点头，从包里摸出一把银元，塞给她，说道：“给你点钱吧，现在外面不太平，不管你去哪，身上有钱总要好一点。”
<br/>　　云儿呆呆地看着手里的钱，说道：“你让我去哪？”
<br/>　　“这……我哪知道啊。”叶少阳挠了挠头。
<br/>　　云儿把钱又放在他手里，说道：“不然，你们带我走吧。”
<br/>　　叶少阳愣了一下，看看毛小方，对云儿说道：“这怎么行，我们两个大男人，带着你一个姑娘算怎么回事，再说我们自己也不知道将来去哪。”
<br/>　　“你们去哪，我都可以跟着。你们是好人，既然能救我出来，我对你们完全信任的。我可以一路上服侍你们。”
<br/>　　叶少阳想到之前被她压在身下强吻的样子，忍不住有点尴尬。
<br/>　　云儿似乎也看出他在想什么，脸上一红，说道：“不要误会，我意思是，可以做你们的丫鬟，一直跟着你们……”
<br/>　　叶少阳冲毛小方挑了挑眉毛，道：“毛哥你带她走吧。”
<br/>　　“开什么玩笑，我一个道士，带着一个漂亮姑娘在身边，像什么样子。”毛小方有点尴尬地摸了摸鼻子。
<br/>　　云儿听到“道士”两个字，想到什么，问道：“我可以当道姑吗？”
<br/>　　叶少阳一愣，问道：“你怎么会有这种想法？”
<br/>　　云儿惨然一笑，道：“说出来也不怕你们笑话，我现在……已经是残花败柳，就算有人愿意娶我，我也觉得愧对人家，我想我还是一个人好了，当道姑最合适不过，那种平静的生活，也能让我的心更平静。”
<br/>　　想起她这三年来的遭遇，叶少阳在心中暗暗叹了口气，也不知道怎么劝她，突然有了一个想法，告诉她自己有个姐姐，是个寡妇，也是一个人，暂时来说，她可以去找翠云，两人也算做个伴，将来也可以一起开个饭馆之类的，至于成家这种事，将来如果遇到合适的男人再说，遇不到就自己生活也好。当道姑的话，叶少阳觉得她这么年轻，不应该那么清苦，而且她本心来说也不信道。
<br/>　　云儿听了他的提议，想了一下也同意了。磕头感谢他们的恩情，叶少阳连忙将她扶起来，决定先离开这个地方，于是一起赶路，回到了镇上，住进了客栈里。
<br/>　　叶少阳先要了一桌饭菜，三人饱餐一顿，然后叶少阳向云儿打听起她的经历和有关五通神的事。云儿从头开始讲起。
<br/>　　关于五通神的来历，云儿所说的，跟之前叶少阳二人打听到的差不多，当地的山民出于对五通神的敬畏，从几十年前起，除了常规的祭祀之外，每隔三年，就会挑选一位十六岁的少女，送到山神庙里，供五通神享用……
<br/>　　三年之后，这女子到了十八岁，对五通神来说就没什么吸引力了，会让家人领回去，寻一个人家嫁了。这男人一般都是五通神的信徒，对这种事不会怎么觉得屈辱，至少能够接受，五通神还会给他们一笔钱，让他们生活无忧……
<br/>　　五通神从没有杀过人，将伺奉自己的姑娘还安顿下来，这乍一听似乎还不错，不过五通神这么做可不是为了姑娘本身，而是为了自己的神位，尽量少制造业力。这一点，叶少阳看得很清楚。
<br/>　　提到自己受辱的这三年，云儿泣不成声。那种痛苦，光是想一下，叶少阳也是略有体会，更不要说忍辱三年……这简直就不是人过的日子。
<br/>　　这几十年来，五通神不知道祸害了多少妹子，不管阴司和那位李纲怎么看，至少在自己心中，这个五通神非杀不可。最重要的是，五通神不杀，他将来还会祸害更多的姑娘，无休无止……尽管他没杀人，但是对于这些姑娘身心的摧残，可能会影响一生，更不要说在这些姑娘嫁人之后，他还会偶尔去找她们……
<br/>　　云儿听了他跟毛小方的交流，得知他们想杀了五通神，顿时吓得不轻。“万万不可，两位大哥，那五通神神通广大，而且他是神灵，根本就不是你们能对付的！再说，他也的确保佑了这一方水土，让这里五谷丰登……”
<br/>　　叶少阳哼了一声说道：“没有山神保佑，我不信老百姓就活不下去了？有很多地方穷山恶水，当地百姓不也照样生活？再说，就算这里真的不适合人居住，大不了去到别的地方，人是活的，不应该被环境所困，再说了，你们这些都是当地的女儿，这什么镇子，靠着卖女儿过上好日子，这样的日子，不过也罢！”
