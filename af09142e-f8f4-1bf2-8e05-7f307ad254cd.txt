        “好。”落以柔礼貌性地接过童筱颖手中的保温饭盒。
<br/>
<br/>        “谢谢了。”童筱颖谢道，然后毫不犹豫地转身进了电梯。
<br/>
<br/>        而童筱颖那与世无争的姿态，倒是让落以柔感到诧异。
<br/>
<br/>        她既然怀疑她对傅夜沉有爱慕之心，那为什么不把她当情敌防着？
<br/>
<br/>        殊不知，童筱颖压根就没想过要防着她落以柔。
<br/>
<br/>        落以柔要是有本事把傅夜沉给抢了去，那就让她去抢好了。
<br/>
<br/>        童筱颖早就做好了随时卸下傅家大少奶奶这个身份的准备。
<br/>
<br/>        落以柔回到自己的秘书室里后，打开了童筱颖送过来的保温饭盒，看到里面是一荤一素一汤一点心后，将里面的荤菜和汤，换成了自己做的带过来的荤菜和汤。
<br/>
<br/>        只是，她不明白，童筱颖做的点心为什么会是一只“天鹅酥”和一只“青蛙酥”？
<br/>
<br/>        落以柔不想否认，童筱颖的手艺确实不错，这“天鹅”和“青蛙”，她都做得惟妙惟肖。
<br/>
<br/>        傅夜沉从会议室里走出来，路过秘书室的时候，落以柔连忙起身叫住了他。
<br/>
<br/>        “傅总，这是夫人给你送过来的午饭。”
<br/>
<br/>        落以柔将保温饭盒提了出去。
<br/>
<br/>        傅夜沉的目光随之看了过去，见是筱颖经常提的那个保温饭盒，于是毫不犹豫地伸手接了过来，
<br/>
<br/>        “她人呢？”他问。
<br/>
<br/>        落以柔心虚地回答道：“已经走了。”
<br/>
<br/>        其实，是被她这个当秘书的给“赶走”的。
<br/>
<br/>        傅夜沉微蹙了下眉头，默不吭声地提着保温饭盒去了自己的办公室。
<br/>
<br/>        保温饭盒里，底层是汤，然后从下往上是米饭、荤菜、素菜、点心。
<br/>
<br/>        所以，傅夜沉一打开饭盒，最先入眼的是一只“天鹅”和“青蛙”。
<br/>
<br/>        看到这里，他嘴角情不自禁地微微扬了扬，而那天她跟自己的弟弟傅昼景私奔，还谎称怀孕的事情，他瞬间抛诸脑后。
<br/>
<br/>        傅夜沉随之将保温饭盒的盖子重新盖上，提起保温饭盒，拿起车钥匙，只身离开了总裁办。
<br/>
<br/>        刘婶今天请假回老家去了，连华生这两天也不见人影，偌大的私宅里就只剩下童筱颖一个人。
<br/>
<br/>        她从傅氏集团大厦回来后，自己给自己做了午餐，一个人坐在餐厅里慢慢地享用。
<br/>
<br/>        此时，澳洲那边的宠物幼儿园又给她发邮件过来了，告知她，如果她和傅昼景再不将小爱领回去，他们将会把小爱卖给其他家庭抚养。
<br/>
<br/>        其实，她迟迟不去接回小爱的原因，是怕睹物思人。
<br/>
<br/>        童筱颖深吸了口气，给那边回了一条电子邮件，把私宅的地址和联系号码发给了对方，支付了费用，让对方把小爱空运过来。
<br/>
<br/>        就在这时，家里的大门开了，将她吓了一跳。
<br/>
<br/>        当她从餐厅里起身跑出来，看到进来的人是傅夜沉，心里暗自松了口气。
<br/>
<br/>        傅夜沉的这座私宅很大，以前还有保安巡逻，现在却没有请保安了，全靠私宅里新装的智能安防系统守着。
<br/>
<br/>        所以，陌生人压根就进不来。
<br/>
<br/>        “你忙完了？”童筱颖讷讷地问道。
<br/>
<br/>        傅夜沉换了鞋，提着保温饭盒走到餐桌前坐下，见童筱颖也正在吃午饭，不冷不热地说：“一起吧！”
<br/>
<br/>        “好。”童筱颖回到餐桌旁，在傅夜沉的身边坐下，然后替他打开了保温饭盒。
<br/>
<br/>        当她把每层菜拿出来后，看到荤菜和汤菜被人换掉了，脑海里第一个想到的人便是落以柔。
<br/>
<br/>        她还真没想到，落以柔竟然会跟她来这招。
<br/>
<br/>        童筱颖忽然灵机一动，将自己刚炒的两个菜一并轻轻地推到了傅夜沉的面前。
<br/>
<br/>        “这些菜你都尝尝，然后给排个序，最爱吃最好吃的排第一，以此类推。”童筱颖一手五指轮流敲击着桌面，一手托腮，看着傅夜沉，一脸俏皮地说道。
<br/>
<br/>        傅夜沉睐了童筱颖一眼，拿起筷子，第一个夹的就是“天鹅酥”。
<br/>
<br/>        看到这里，童筱颖想都不用想就知道，大哥一定会把她做的“天鹅酥”和“青蛙酥”排第一位。
<br/>
<br/>        傅夜沉都试吃完后，果然把“天鹅酥”和“青蛙酥”排在了第一位，然后是她自己炒的两个随意菜，接着是保温饭盒里的素菜，再是荤菜和汤。
<br/>
<br/>        这回，童筱颖莫名其妙地感到心情特别好。
<br/>
<br/>        在大哥不知情的情况下，大哥还是认为她炒的菜才是最好吃的。
<br/>
<br/>        傅夜沉看着童筱颖脸上的笑容，心里也跟着乐开了花，然后拿起碗筷开始自顾自地吃饭。
<br/>
<br/>        他忙了一上午，现在又能看到她脸上的笑容，在又累又饿，心情又好的情况下，他的胃口大开。
<br/>
<br/>        “是什么事情让你如此开心？”傅夜沉按捺不住地问。
