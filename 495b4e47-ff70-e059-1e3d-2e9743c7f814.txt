　　这些年轻人，统一穿着淡蓝色衣衫，另外，在胸膛左上方，皆是佩带着一枚蔚蓝色徽章，徽章中刻着一把沾染殷虹血液的匕首。
<br/>　　确实是迦南学院的标志，慕清霄打量他们的同时，后者们也都打量着他，眼底有一丝淡淡的戒备与排斥。
<br/>　　上前一步，慕清霄道：“听说迦南学院正在举办招生选拔赛，我们是来参赛的。”
<br/>　　“参加选拔赛？”
<br/>　　话语刚落，周围纷纷投来错愕的目光，中年男子望着两人格外出众的容貌，满脸怀疑道：“你们来自何处，怎么不见引荐导师与你们一起过来？”
<br/>　　“我们来自加玛帝国，半个月前错过了迦南学院的招生大会，所以并没有引荐的导师，而是长途跋涉来到此处。”
<br/>　　“你是说，你们两个是穿过黑角域来这里的？”
<br/>　　中年男子一愣，眼中满是惊诧，心中却是信了，半个月前迦南学院的招生队，确实经过了加玛帝国，而且这支招生队才刚回来没多久。
<br/>　　只是让他不敢置信的是，两人居然是横穿黑角域来到此处。
<br/>　　而且，似乎并没有受到一丝损伤，要知道黑角域这种吃人不吐骨头混乱之所，他们当导师的，都不敢说横穿就横穿。
<br/>　　“真是令人意外，我是头一次遇上穿越黑角域来报名的学生，学院确实还在招生，审核要求也很简单，十五岁前达到八段斗之气。”
<br/>　　话语刚落，一股不弱的气息就从慕清霄与小医仙身上扩散而出，中年男子一惊，道：“斗师，八星斗师，七星斗师，你们的年龄...”
<br/>　　待慕清霄谎报年龄后，中年男子微微点头，眼中有一抹疑惑，按道理来说，只有八星斗师的实力，怎么可能横穿黑角域？
<br/>　　更何况，身边还有这么一位倾国倾城的女子，不招人惦记才怪。
<br/>　　不过，疑惑归疑惑，两人的天赋都非常的不错，绝对算得上天才，学院就是需要这种新鲜血液。
<br/>　　至于执法队其他人，纷纷投来震惊的目光。
<br/>　　没想到，两人竟都是斗师修为，放在迦南学院中虽算不得最高，但也算是不错的天才了，就冲这份勇气，前途不可估量啊。
<br/>　　“我叫霍德，迦南学院执法队二小队队长，也是学院的一名黄阶导师，你们先随我去填写一份信息吧。”
<br/>　　微微点头，慕清霄与小医仙就在周围异样的目光中，跟在霍德身后离开。
<br/>　　......
<br/>　　......
<br/>　　几分钟后，迦南学院临时驻地，一间宽敞的屋内。
<br/>　　霍德拿出两张信息表格，随后又拿出一枚蓝色的水晶，道：“将信息表格填好，这是记录水晶，能够专门录下学生的容貌。”
<br/>　　慕清霄微微点头，眼中闪过一丝奇异，原理简直跟地球上的照相机一模一样。
<br/>　　当两人将信息简单的填写完毕，霍德将信息表格拿在手中瞧了瞧。
<br/>　　“由于你们来的比较晚，正好遇到我在这里执法，我将你们的信息递上去，你们暂时就只能在我的班级中修炼，你们有没有意见？”
<br/>　　其实，在哪个班级，慕清霄都觉得无所谓，他的目的是陨落心炎，不过看霍德期待的表情，显然导师间的竞争也是蛮大的。
<br/>　　至于小医仙，更是如此，只要有慕清霄的地方，四海八方皆为家。
<br/>　　“既然如此，就按照霍德导师的意思。”
<br/>　　闻言，霍德给慕清霄投去一个赞赏的目光。
<br/>　　同时，心中也无比的激动，二十岁的八星斗师，十七岁的七星斗师，两人都是天才，或许他也有晋升玄阶导师的资格也说不定。
<br/>　　“今日的天色有些晚了，迦南学院外有一片极大区域的原始森林，其中高阶魔兽不少，夜晚穿行风险极高，明日一早，我就命人驾驭狮鹫载你们过去。”
<br/>　　“多谢导师。”
<br/>　　“没事，你们来我班上，对我也有很大的帮助。”
<br/>　　霍德微微一笑，挥了挥手，叫来一男一女两名执法队员，道：“他们是入学新生，给他们准备两间屋子。”
<br/>　　慕清霄道：“准备一间就可以。”
<br/>　　“呃...！”
<br/>　　话语刚落，不仅是霍德，连两名执法队员都有些愕然，可看到小医仙娇羞的模样后，瞬间秒懂。
<br/>　　看来两人也是郎情妾意啊，倒也是，两人天赋都非常不错，而且郎才女貌，确实非常的般配。
<br/>　　“咳咳，那就给准备一间吧。”
<br/>　　霍德干咳一声，挥了挥手，看向慕清霄，道：“虽然学院不反对恋爱，但你们年纪还小，记住要以学业为重。”
<br/>　　微微一笑，慕清霄就拉着小医仙，跟在女执法队员身后离去。
<br/>　　......
<br/>　　......
<br/>　　随着时间的推移，黑暗笼罩大地，皎洁的月光透过云层倾洒而下，给昏暗的世界带来了一丝的光亮。
<br/>　　迦南学院一间干净整洁的卧室中，窗口处，望着和平镇安详的夜晚，慕清霄思绪万千。
<br/>　　抬起臻首，小医仙忽然问道：“我们为什么要用如此麻烦的方式入学，直接进去不是很好吗？”
<br/>　　“迦南学院是斗气大陆上最古老的学院，随着岁月的变迁，依旧能屹立至今日，自然有它的底蕴。”
<br/>　　慕清霄微微摇头，他何尝不想直接进去，他的实力虽已经不弱，但远没有达到硬闯迦南的程度。
<br/>　　迦南学院入学标准虽然简单，但水却极其的深，其中老怪物更是不知凡几，硬闯后能不能活着出来都是问题。
<br/>　　暂且不是迦南学院半圣级别的院长，内院大长老，苏千都是三星斗宗巅峰，外院斗皇级别强者无数。
<br/>　　更何况，隐藏在暗中的外院院长，谁也不知道有多强，但根据以上信息推测，最起码也是斗宗级别。
<br/>　　而且，他寻找吞噬陨落心炎也需要时间，冒着得罪迦南的风险，硬闯绝对是最愚蠢的选择。
<br/>　　“明天就要入学，先在这里修炼一段时间，等你实力稍微强大一些，我们就离开此处，前往中州，彻底解决你的体质问题。”
<br/>　　小医仙静靠在他怀中，跟在慕清霄的身旁，看遍人情世故，一路走下来，她的心境已经比半个月前成熟了许多。
<br/>　　可厄难毒体，依旧是最令她担忧的问题。
