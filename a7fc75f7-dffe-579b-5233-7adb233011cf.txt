　　一座孤崖之上，两道身影有些萧索，一道抬头望天，眼见夕阳西落，暮色沉沉，晚风萧瑟，充满凄凉悲哀的意境。另一道身影却仿佛在沉思，在领悟什么，但却始终抓不到，她的面上充满无奈。
<br/>　　这两人正是三生道人与后土祖巫。
<br/>　　“一路走来，你可有悟到什么？”
<br/>　　良久，三生道人看着有些无奈的后土，出声问道。
<br/>　　后土闻言，微微一怔，脸上满是悲切之色，口中轻轻一叹，缓缓应道：
<br/>　　“我循着天机而来，一路走来，似有所明悟，我已隐隐感觉到自己的大道所在，但还是不太明确，你能为我指点迷津吗？”
<br/>　　不知为何，后土心中对于眼前这个刚刚认识不久的三生道人并没有任何防备，想什么，就当口问了出来。
<br/>　　三生道人闻言，却是轻轻一叹，沉默良久，不知道该说些什么。
<br/>　　“怎么，是有什么难言之隐，不便多说吗？”
<br/>　　后土看出了三生道人的沉默，疑问道。
<br/>　　“唉……”
<br/>　　三生道人感叹一声，道：
<br/>　　“你可知道，若是我讲了出来，你之道也到了尽头了？”
<br/>　　后土沉默了片刻，坚定道：
<br/>　　“朝闻道，夕死可矣，为了天地众生，牺牲我个人那又何妨？盘古父神当年一样牺牲了自己，造化天地洪荒。若是能够有助于天下苍生，我等后辈更应效仿盘古父神，舍己为人。”
<br/>　　“哎，你可知道，我平时最不屑的就是那些舍己为人之辈。天下熙熙，皆为利来，天下攘攘，皆为利往！看着那些不顾自己死活，为了帮助别人的人，我心中就非常不爽。
<br/>　　你愿意为他人而牺牲自己，你可曾问过其他人的意见没有，你问过你自己的亲朋好友的感受没有，你这样做，一样是自私自利。
<br/>　　但，同时，我最敬重的也是这样的人，因为他们拥有强大的勇气，能够舍己为人。不过，虽然敬重，但我绝对不会喜欢那样的人，因为做这样的人的朋友，太累了！他时时刻刻将天下苍生放在心中，这样活着，实在是莫名其妙……”
<br/>　　三生道人唠唠叨叨说了一大堆，也同样骂了一大堆。对面的后土面色不变，相反还有些淡淡的微笑，这让她觉得心中安宁。
<br/>　　“我知道。但我还是要去做，只因为我就是我，而你，终究只是你！”
<br/>　　听了这话，三生道人沉默了，是啊，她就是她，若她不这样做，还会是日后那个被人敬仰万古的后土娘娘吗？而我终究只能是我，所以我是自私的，我做不到她那般伟大。
<br/>　　“哎，罢了罢了，既然你都如此说了，更何况天机已现，早晚都会出现那一幕，是早是晚，又有何区别呢？”
<br/>　　三生道人叹息一声，幽幽道。
<br/>　　“盘古开天，劈开苍茫，分化阴阳，塑立三才，演化万道，造化万物。但，其实这天地也并不完美。”
<br/>　　“不完美？这是为何？”
<br/>　　“是啊，不完美。盘古自是实力强大，能够开天辟地，演化洪荒万物，孕育轮回。但，完美的世界应该是天、地、人三才并列。而洪荒当初刚刚开辟之时，只有天道，人道与地道并不全。后来，女娲造人成圣，人族出现，补全了人道文明。故而，如今这天地还差地道。”
<br/>　　“地道？何谓地道？”
<br/>　　后土似有所悟，心中那道明悟也越来越深，气机也越来越浓厚。
<br/>　　“所谓地道，自是应该与天道相平。天道管理天下万物运行的规则，保持世界运行轨迹，使得万物平衡，而地道则是管理大地生灵，生死幻灭，轮回不休。这其中最重要的，便是六道轮回！”
<br/>　　三生道人字字如机，如同讲道一般，对着后土说道。
<br/>　　“六道轮回本为生灵造化之转处，但有生灵死亡，鬼魂出现，当归此六道。似六孔拱桥之势，分合于六道，轮回不息，鬼魂不绝！”
<br/>　　三生道人知道后土已经有所领悟，便也看开了，不再犹豫，更何况，死亡又何尝不是意味着另一种新生呢，所谓的生生死死，地道天道，在他成为三生石，斩出三生三世之身后，便已经明了。
<br/>　　“何为六道？”
<br/>　　后土只觉得自己内心深处最后一道迷雾也被三生道人拨开了，只觉得有一种拨开云雾见青天的感觉，身上的气息更加浓郁了，她知道，自己成道就在当下。
<br/>　　三生道人抬眼，看了看不远处的血海，缓缓道：
<br/>　　“所谓六道者，又分有形道与无形道。有形道为人道、阿修罗道与畜生草木道，无形道为天道、饿鬼道和地狱道。
<br/>　　有形之道皆能得道体，人称肉身，畜生与草木成本身，以此轮回的生灵，无论人妖，皆能修炼元神之术，死后亦会有鬼魂，循着天机，投入下一道。无形之道，除却天道，皆是只有元神鬼体，虽能幻化，却无实体。
<br/>　　天道转世者，多为大富大贵之辈，亦或者是修道资质上乘者，天地为其父母，一出生即为先天。地狱道乃众鬼魂受苦受罚之地。是故，六道之中，以天道居最上，地狱道为最下。”
<br/>　　后土一听，居然还有如此多分法，真是稀奇，便开口问道：
<br/>　　“入六道又要以何区分？莫非是要如那流水一般，择道而入？”
<br/>　　“择道而入？那岂不是天下大乱。所谓前世因，今生果，来世果，今世因。鬼魂欲如六道，须得以前世因果为凭，享后世之福。
<br/>　　因果重者，须入那地狱道，受尽刑罚，洗去自身因果业力方能重新转世；因果业力沾染稍轻者，亦须投入饿鬼道，阿修罗道，洗去业力方能重新轮回；因果业力最轻者，当如畜生道，转世轮回，洗去因果业力，重入人道。
<br/>　　生灵生而皆有因果，只要不沾因果业力，便能入人道轮回，以求天道转轮，脱去凡体。如若世代都行事积善，有大善于世，亦或者有功德余荫庇佑，当入天道，享得人间诸般福禄恩泽，荣华富贵，或者修仙得道。”
<br/>　　后土闻言，不由为之一愣，之后出声问道：
<br/>　　“敢问道友，可有脱离六道之法？”
<br/>　　“超脱六道之法？”
<br/>(本章完)
