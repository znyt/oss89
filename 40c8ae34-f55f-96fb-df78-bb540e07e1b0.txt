　　已经踏入阴谷入口的白家五十人，经历了正常的天旋地转后，便统统落在了那处沙滩上。
<br/>　　这里依然是好似与外界没有隔阂的碧海蓝天，金色的细沙在海风中形成层层浪花，沙滩上的多种贝壳铺了满地，显然已经有段时间无人踏及。
<br/>　　五十人与所有进入阴谷的人一样遭遇了昏迷，白忧第一个醒来，瞬间的警惕让他一跃而起，紧接着便是一个个白家人的依次苏醒。
<br/>　　面对这一完全陌生的地方，白家所有人都下意识聚集在一处。
<br/>　　“没人？”一名白家人脱口而问。
<br/>　　白忧淡淡开口：“有人才不正常，大家注意不要分散，阴谷很大。”
<br/>　　“罗家那帮人怎么还没进来？”一名白家成员略带嫌弃的开口。
<br/>　　“一帮胆小鬼。”有人冷笑。
<br/>　　白忧眼神闪了闪，道：“集体行动注意言辞。”
<br/>　　“明白。”
<br/>　　“是。”
<br/>　　白家人纷纷点头表示心里有数，只是在罗家人还未进入的时间里，也不禁会想到这次时间刻意错开的行动。
<br/>　　“话说大小姐她……”有人试探性问道。
<br/>　　白忧面色瞬间难看起来，语气也冰冷：“随她去，不要动她。”
<br/>　　一名白家旁支成员苦笑：“想动也动不了吧？大小姐的战力实在是恐怖。”
<br/>　　此话略带一丝倾斜态度，也彰显了白允儿在白家的地位，可不仅仅是因为她有嫡系血脉与白忧同母。
<br/>　　其他白家人在听到这番话后，大多面色变了变，少数人看向了白忧的神色，白允儿刚来白家时可是被囚禁在那处花园里，虽说只是软禁一切优待，但也至少没有自由。
<br/>　　可自从她失忆让白忧放松警惕，并且阴差阳错的赴了高少辉的约去了一趟阴阳谷，整个人的战力就无限飙升，返祖更是让所有白家人都始料未及。
<br/>　　眼下的白允儿别说被白家人抓·住针对，恐怕只要白家人靠近对她出手，下一秒便是被她反压制。
<br/>　　血脉的高度只是其一，返祖才是真正让所有白家成员发自内心畏惧的根本，每一名神秘家族的返祖者，都有着某种神奇的强大气场，能让这个家族的成员都难以抑制的赶到恐惧。
<br/>　　只是白允儿虽强大到让这些白家成员害怕，却依然战不过白忧，两人并非没有爆发过战斗，白家大宅后方的那山谷里，就有一大·片地带是废墟。
<br/>　　那边是两名白家嫡系交手的结果！
<br/>　　在那之后白允儿便离开白家，谁也抓不住，还带走了白羽。
<br/>　　这也是白忧有心继续将白允儿软禁，却毫无办法的原因，白允儿斗不过他，可不代表跑不了。
<br/>　　想到这里的白忧眼神更冷，开口道：“去狼牙的人都安排好了吗？”
<br/>　　“大少请放心。”一名白家旁支立即回答：“这次楚涵必死无疑。”
<br/>　　白忧看了他一眼，眼底的杀意浓郁：“我要的可不仅是楚涵死。”
<br/>　　“狼牙也将彻底消失。”那名白家人再次保证，而后道：“此外我们也已经调查清楚，叶永阳当下的确就在狼牙。”
<br/>　　听到这一消息的白忧面色缓和了一些，嘴角也渐渐勾起了弧度：“抓·住后直接带进来。”
<br/>　　“自然，我已交代清楚。”
<br/>　　谈话到此间，一阵海浪翻滚的声音响起，紧接着便能看到几十个人被巨浪冲上沙滩，他们都处于昏迷之中，正是随后而到的罗家人。
<br/>　　白家众人立即闭口不再多话，安静的站在一旁等待这些人苏醒，白忧也只是淡淡的扫了一圈，他对这些罗家人可不感兴趣，之所以让两家族人一同行动，也不过是想让罗家当炮灰。
<br/>　　而就在这段时间内，白忧等人却并不知道在他们身后的不远处，正有一名人身鱼尾的女子静静的看着这里。
<br/>　　水族守护着阴谷入口，在入驻水族旧址之后，任何人进入阴谷他们都会第一时间知道，而水族圣女更有着超乎常人的感应，甚至不用靠近便能在水族旧址的内部感应到沙滩上的情况。
<br/>　　包括来者的数量，大致战力等……
<br/>　　当白家这五十人到达的一瞬间，她便立即从小溪游了过来，恰好听到了刚才的对话，也了解了这批人的身份。
<br/>　　“有意思。”鱼柔女面上浮现了一个极淡的笑容，灵动的眼眸里不知道在想什么。
<br/>　　鱼柔女并没有惊扰到任何人便悄悄离开，她来去自如，在这入口处谁也不可能发现她。
<br/>　　这时候的罗家人也依次苏醒，在经历了与白家人一样的警惕惊乍之后，也很快适应了此处的环境。
<br/>　　罗杰醒的最早，在看到不远处的白忧等人后，就走上前开始商量接下来的行动，而他身后的那些罗家人由于战力水平不均，醒过来的时间也不同，尤其是并非神秘家族成员的楚云天，若非有着罗家几名成员刻意伪装，恐怕他长时间的昏迷都会被白家人察觉到异样。
<br/>　　白忧与罗杰的商议很细致，即将商量妥当后他看了眼沙滩，嘴角倾斜的嘲讽道：“真够慢的，早知道该让你们先进，浪费时间。”
<br/>　　罗杰也不恼，直接道：“比不上你们白家喜欢用偏门法子留住血脉浓度。”
<br/>　　白忧直直的盯了他一眼，而后一言不发的转身，朝着蔓延到沙滩上的那处小道走去。
<br/>　　这是阴谷入口唯一的道路，指引很明显，对于本就是这里居民的神秘家族来说，没有必要再去开辟新的道路，阴谷内的禁止也不允许他们这么做。
<br/>　　白家人全部跟上白忧的脚步，罗杰也在给了楚云天一个眼神后，带着罗家人跟在白家人后方，而他自己则是先一步与白忧并肩而行。
<br/>　　两个家族一百来号人的队伍浩浩荡荡，在这并不宽的小道上排的有些长，走了相当长的一段距离，尾端的人依旧还未踏上小道的石子路。
<br/>　　“直接去阳谷。”白忧对着身旁的罗杰开口。
<br/>　　罗杰有些不甘心：“不找找旧址？”
<br/>　　“你找得到么？”白忧声音带着嘲弄，目光看向了远处好似屏障一般的浓雾：“只有一条道，下方也有着拼图磁场。”
