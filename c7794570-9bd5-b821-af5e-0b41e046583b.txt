<br/>				<br/>　　    白二爷瞪大了眼睛看着那缓步而去的青色身影，有些不可思议。不是说鬼医喜穿红衣吗？不是说鬼医性格阴晴不定手段狠辣吗？<br/>　　<br/>　　    怎么、怎么会是这个浑身散发着尊华气息的翩翩贵公子？这一路走来，他见过她出手，也知她的实力深不可测，却没想过，这救了他们，护了他们一路的人，居然就是鬼医凤九……<br/>　　<br/>　　    “是的，她就是我主子。”白倾城站了起来，看了那已经往阁楼走去的身影一眼，这才对她二爷爷道：“二爷爷，我们到后院去说吧！”<br/>　　<br/>　　    几人往后院而去。而冷华等人则跟着凤九上了阁楼，混元子则到了这里后又去酒肆买酒了，并没跟着进来。<br/>　　<br/>　　    “主子，你是从哪里抱来的孩子？”杜凡看着凤九怀里的孩子，好奇的问着。<br/>　　<br/>　　    “路上捡的。”凤九脸上露出笑容来，逗弄着怀中的孩子，一边对他交待着：“你等下找个奶娘，方便以后照顾灏儿。”<br/>　　<br/>　　    “灏儿？”<br/>　　<br/>　　    “对，我给他取的名字是轩辕灏，以后，他就是我和墨泽的儿子。”她轻点着怀中孩子的小嘴，见他咯咯的笑了起来，便也轻笑也声。<br/>　　<br/>　　    旁边几人听了，不由的相视一眼。主子和阎主的儿子？阎主要是回来看到这么大一个儿子，估计准会吓一大跳。<br/>　　<br/>　　    冷华沉思着，道：“主子，我们府里的人手足够了，帮忙带小主子也没什么大问题，我觉得奶娘可有可无，而且，主子不是说孩子也可以喂养羊奶吗？”<br/>　　<br/>　　    闻言，凤九挑了挑眉：“最近又不是很太平了？”<br/>　　<br/>　　    “有一些势力的人想趁机混进我们里面来，主子不在这一段子，揪出了两个。”冷华说着。<br/>　　<br/>　　    见此，凤九点了点头：“也好，那就让府里的人都帮忙照顾着吧！府里的人都是信得过的，交给他们我也放心。”<br/>　　<br/>　　    “是，我等下就让人去买两头羊养在后山。”冷华说着。<br/>　　<br/>　　    “对了主子，那个白倾城那里又是怎么回事？我怎么隐约听见说什么族里出事了？”杜凡握着手中的扇子疑惑的问着。<br/>　　<br/>　　    “据说是白家出事了，被人灭了门，也就剩下他们几人逃了出来。”凤九说着，将怀中的孩子递给一旁的冷华，继续道：“我在路上遇见他们三个，得知他们是白倾城的家人，便顺路将人带回来了。”<br/>　　<br/>　　    她站了起来，负着手走到窗口处，道：“路上我已经让人去查白家被灭门一事了，到时消息传达回来时记得拿来给我。”<br/>　　<br/>　　    “是。”他们应了一声，相视一眼，有些意想不到。那白家也是一个有底蕴的家族，没想到居然就这样被灭门了。<br/>　　<br/>　　    “冷华，你先把灏儿带回凤府去照顾着，我晚点会回去。”她转身对抱着孩子的冷华交待着。<br/>　　<br/>　　    “好。”冷华应了一声，便抱着孩子先告退。<br/>　　<br/>　　    后院，在听到她二爷爷和两个弟弟将事情说了一遍后，白倾城脸色苍白，眼中尽是无法置信，浑身的力气仿佛被抽干了一般，险些晕了过去。<br/>
