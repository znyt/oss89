　　那豹哥早已经被郑同宇这一巴掌扇懵了，
<br/>　　“宇少，您……”
<br/>　　“跪下！”
<br/>　　郑同宇再次一声暴喝，语气毋庸置疑。
<br/>　　普通人或许不清楚，可是他身为高干子弟，又岂会不知道，云州叶先生，是何等恐怖的存在！
<br/>　　便是在当初，他刚得知叶尘的身份之时，就已经位列少将，连军方的巨头级人物君老，都到机场亲自迎接。
<br/>　　更遑论如今，此人据说已经位列元帅，是华夏国数十年以来，唯一一位在和平年代取得此殊荣的存在。
<br/>　　至于这一年来，此人的所作所为，更是可以用空前绝后来形容。
<br/>　　先后将高丽、岛国、米国一一踩在脚下，前不久更是以一人之力，灭掉了西方教廷，如今已然是名副其实的全球第一人！
<br/>　　此等人物，莫说是他，就算是他的父亲，和此人一比，也显得微不足道。
<br/>　　然而，
<br/>　　豹哥显然还处于懵逼状态，满脸的不甘和难以置信之色，
<br/>　　“宇少！您这……到底是为什么呀！”
<br/>　　也难怪豹哥想不通，以郑同宇的身份地位，即便对方是那些超级大家族的公子哥，也不至于让他畏惧到如此程度啊！
<br/>　　郑同宇见他如此冥顽不灵，顿时急的差点吐血，低喝道：
<br/>　　“你若是不想死的话，就赶紧给我跪下，请求叶先生的原谅！”
<br/>　　豹哥感受到郑同宇身上的杀气，顿时吓得浑身一颤，也不敢再继续啰嗦，当即老老实实跪俯在地上，咬着牙，一脸不甘道：
<br/>　　“叶先生，对不起！”
<br/>　　叶尘却犹若未闻，端起手中的酒杯，又缓缓品了一口，方才低头瞥了一眼那豹哥，缓缓道：
<br/>　　“你刚刚如此挑衅于我，又那般羞辱我的兄弟，一句对不起就完了？”
<br/>　　豹哥心头原本就十分不忿，跪下磕头道歉，也完全是被郑同宇所逼，并非出于自愿，就这已经感觉到莫大的羞辱，如今听到这话，哪里还能忍得住，顿时没好气道：
<br/>　　“那你还想怎样？”
<br/>　　叶尘两眼微微一眯，眼眸中闪过一抹寒光，冷冷道：
<br/>　　“辱我者，死！”
<br/>　　轰！
<br/>　　等到叶尘那个‘死’落下的同时，豹哥顿时全身一颤，感觉自己仿佛一下子坠入了修罗地狱，一股强大的令人心悸的恐惧感，涌上他的心头。
<br/>　　“啊！！！”
<br/>　　豹哥忽然撕心裂肺的惨叫了一声，同时两样瞪得老大，如同看到了世上最恐怖的事情一般，继而身体猛然一挺，然后便软绵绵的倒在了地上。
<br/>　　“啊！！”
<br/>　　周围众人看到如此诡异的场面，顿时一个个吓得纷纷倒退，均不知道发生了什么事。
<br/>　　“什么情况？”
<br/>　　“豹哥好像……死了？”
<br/>　　“怎么会这样？明明没人对他动手啊！”
<br/>　　……
<br/>　　众人全都疑惑不解，唯独一旁的郑同宇，似乎想到了什么，双眸之中顿时露出浓浓的惊骇之色，浑身忍不住瑟瑟发抖，
<br/>　　“叶、叶先生……此人对您不敬，死有余辜，但是与我无关，还请您……”
<br/>　　郑同宇早就听说过云州叶先生的手段，如今亲眼看到，早就吓得尿都快出来了，连忙撇清自己的关系，生怕落得和豹哥一样的下场。
<br/>　　没等郑同宇说完，叶尘轻轻一笑，
<br/>　　“郑公子这话何意？莫非你认为，是我害死的他？”
<br/>　　郑同宇顿时再次一颤，连忙摆手，解释道：
<br/>　　“不不不！我不是这个意思……此人平日里作恶多端，多半是被冤魂索命，所以才会突然暴毙，自然和叶先生没有任何关系……”
<br/>　　他不解释还好，这么一说，周围众人反而越发觉得此事诡异。
<br/>　　之前明明还安然无恙的豹哥，在叶尘说出死字之后，就莫名其妙的真的死了，而且没有任何征兆！
<br/>　　难道真的是被眼前这个神秘少年所杀？
<br/>　　可是他明明什么也没有做啊？
<br/>　　莫非这少年会妖术不成？
<br/>　　一时间，众人望向叶尘的目光，顿时充满浓浓的畏惧和警惕，全都不由自主的纷纷向后方倒退而去。
<br/>　　就连李虎、唐威和陈东三人，眼眸深处也露出一抹敬畏之色。
<br/>　　“唉！”
<br/>　　叶尘看到众人的反应，顿时也有些意兴阑珊，不由得轻叹了一口气，随手将酒杯放回桌上，回头冲着李虎三人道：
<br/>　　“时间不早了，回吧！”
<br/>　　说完，便直接朝酒吧外面走去，周围人物顿时纷纷散开，主动让出了一条通道，而李虎三人也连忙快步跟上。
<br/>　　郑同宇看到叶尘离开，顿时松了一口长气。
<br/>　　……
<br/>　　四人出了酒吧之后，才刚刚没走多远，徐倩倩忽然从后方追了上来，上气不接下气道：
<br/>　　“李虎，你等等，我有话跟你说！”
<br/>　　然而，李虎却连头都没有回，冷冷道：
<br/>　　“你都说和我们不熟了，还有什么好说的？”
<br/>　　徐倩倩顿时满脸尴尬之色，
<br/>　　她之前为了自保，所以才极力撇清和李虎等人的关系，却怎么也没有想到，一直被她看不起的那个小白脸，竟然有着如此强大的背景和实力！
<br/>　　别人或许不知道郑同宇的身份，她可是清楚的很，那可是比三大家族的公子哥还要牛逼的存在。
<br/>　　可是在这个叶尘面前，却如此的卑躬屈膝，这绝对不是一般人能办到的。
<br/>　　之前她之所以一直吊着李虎，那是因为李虎虽然也算是一个富二代，可是其貌不扬，而且比起那些真正的贵公子，终究还是要差上不少，所以一直把他当备胎。
<br/>　　可是如今不一样了，李虎有着一个如此牛逼的室友，将来前途肯定不可限量！
<br/>　　如果早知道这层关系，她肯定早就毫不犹豫的答应李虎的追求了。

<br/>　　徐倩倩说着，竟然主动凑到李虎的身上，一脸妖媚道：
<br/>　　“李虎，你之前不是一直想让我做你的女朋友吗？我答应你了！”
<br/>　　叶尘听到这话，顿时眉头一皱，
<br/>　　他没有想到，这个女人竟然还有脸追上来，倘若李虎被她的美色所迷，此事他也不太好插手。
<br/>　　不过很快，叶尘就发现自己的担心多余了
<br/>　　李虎冷冷一笑，直接将徐倩倩的胳膊一把甩开，
<br/>　　“你答应，我还不答应呢！我之前瞎了眼，才会看上你这样的女人！今天如果不是尘哥在，你此刻恐怕早就躺在别的男人怀里了吧！像你这种水性杨花的女人，我李虎才不会要！滚开！”
