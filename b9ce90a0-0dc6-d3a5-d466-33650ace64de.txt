　　第三、第四条主脉已被炼化。
<br/>　　萧逸立刻催动体内冰鸾剑武魂和控火兽武魂，吸收天地灵气。
<br/>　　澎湃的灵气波涛汹涌般涌来，快速地填充着新炼化的两条主脉。
<br/>　　大半个时辰后，单第三第四条主脉被灵气填满，代表着他已经彻底稳固修为，达到了后天四重。
<br/>　　后天境的修炼，最难的莫过于炼化主脉。主脉中的杂质堵塞得非常严重，需要靠极庞大的真气去冲击和洗刷。
<br/>　　按照武者普通的日常修炼速度，完全洗刷一条主脉，起码需要数年乃至更久的时间。
<br/>　　当然了，萧逸拥有紫色武魂，就算没有妖血炎心果，他完全洗刷一条主脉的速度也比其他武者快上许多倍。
<br/>　　像萧家的执事，十年都难以见到一次修为上的突破。
<br/>　　但若是萧逸，别说十年炼化一条主脉了，恐怕数年时间就能突破到先天境，这就是紫色武魂的恐怖之处。
<br/>　　只是，现在有妖血炎心果，倒为他减了不少麻烦。
<br/>　　正当他准备吞服第二颗妖血炎心果时，房门外响起了‘扣扣’的敲门声。
<br/>　　敲门声只出现了一次，而后便没有再敲。
<br/>　　萧逸站起身来开门，发现竟是四、六两位长老。
<br/>　　两位长老也算有心，猜到萧逸可能在修炼，所以只敲了一次门。
<br/>　　武者修炼过程中，忌打扰，若是在突破过程，受到严重的干扰，甚至会出现反噬而重伤的情况。
<br/>　　两位长老也怕出现这种情况，所以只敲一次门，若萧逸闲着，自然会来开门，若是没有开门，便代表萧逸正处于紧要关头，他们会立刻离去。
<br/>　　“两位长老，有事？”萧逸淡淡地问道。
<br/>　　“那个...呵呵。”六长老干笑一声，道，“如果我没有猜错的话，你应该是在吞服妖血炎心果突破修为吧？”
<br/>　　“嗯。”萧逸点点头，淡漠地问道，“怎么？家族里还有其他丹田被毁的子弟需要我去救吗？”
<br/>　　“不是，当然不是。”六长老连忙摆手。
<br/>　　四长老笑道，“你小子，是在生我们的闷气吧。别那么小气，我们是来送东西的。”
<br/>　　“送东西？”萧逸微微疑惑。
<br/>　　两位长老各自从怀中拿出了两个玉瓶，递给了萧逸。
<br/>　　四长老笑道，“两瓶是先天丹，各五粒；两瓶是后天丹，各十五粒。小子，别嫌少，我们毕竟在陨星山脉，也要留一些给自己平日的修炼。”
<br/>　　“嘶。”萧逸倒吸一口凉气，两位长老也太大手笔了吧。
<br/>　　这四瓶丹药加起来，价值起码在一万多两以上。
<br/>　　“两位长老这是何意？”萧逸接过玉瓶，问道。
<br/>　　四长老道，“你忘了妖血炎心果的功效吗？它能炼化主脉，焚烧杂质，这个时候服用丹药，将无后顾之忧。我们作为长辈嘛，能帮你的自然多帮。”
<br/>　　六长老撇撇道，“小子，老四他是怕你生我们的气，以后做了家主后给他小鞋穿。”
<br/>　　四长老怒瞪了他一眼，愤愤道，“六长老，你不说话没人当你是哑巴。”
<br/>　　.....
<br/>　　四六长老送过丹药后，没有多留，免得打扰萧逸修炼。
<br/>　　萧逸回到房间，继续修炼。
<br/>　　他原本就是打算服用第二颗妖血炎心果时，顺便服用丹药的。
<br/>　　平日服用丹药，会有杂质残留在体内，无法连续服用。但妖血炎心果的功效，却能焚烧杂质，自然免了这个后顾之忧。
<br/>　　最重要的是，这不会影响炼化主脉的效果，只是焚烧主脉杂质时顺带着把丹药杂质也焚烧了罢了。
<br/>　　不过，他原本没有太多丹药，所以无需在服用第一颗妖血炎心果时就服用丹药。
<br/>　　第一颗妖血炎心果，权当试试炼化主脉的效果了。
<br/>　　想罢，他从乾坤袋里把之前就拥有的丹药拿了出来。
<br/>　　算上五长老‘送’的，杀掉的三位慕容家执事，还有五天前从慕容千军乾坤袋中获得的。总共三粒先天丹，四十粒后天丹。
<br/>　　加上现在两位长老的，总共是十三粒先天丹，七十粒后天丹。
<br/>　　萧逸毫不犹豫地服下了第二颗妖血炎心果。
<br/>　　妖异力量再次出现于体内，焚烧着体内主脉的杂质。
<br/>　　同一时间，萧逸大口大口丹药地服下。
<br/>　　丹药的杂质，顺带着被妖异力量所烧毁，丹药化作了精纯力量。
<br/>　　萧逸认得这股感觉，跟之前吞服九转炼体丹时一模一样，完全是精纯力量；只不过现在丹药的品级更高，数量也更多。
<br/>　　这股庞大又精纯的力量，萧逸用作冲击和洗刷另一条主脉。
<br/>　　当妖血炎心果整颗吞服完，所有丹药全部消化掉，已是数个时辰之后。
<br/>　　夜幕降临，萧逸才停下了修炼，感受着自己体内澎湃的真气，一时豪情万丈。
<br/>　　“竟然又足足炼化了四条主脉，让我的修为达到后天八重，再遇到慕容千军，我一巴掌就能废了他。”
<br/>　　“就算遇到先天武者，恐怕我也能斗上一番。”萧逸脸上战意十足。
<br/>　　第二颗妖血炎心果，同样为他炼化了两条主脉。
<br/>　　而那十三粒先天丹，七十粒后天丹，也为他完全冲击洗刷了两条主脉。
<br/>　　加起来便是四条，算上第一颗妖血炎心果炼化的两条，今天足足炼化了六条主脉，让他的修为达到了后天八重。
<br/>　　今日早上还是后天二重，晚上之时已是后天八重，一天连跳六重，这速度传出去，恐怕会让人惊骇得无法言表。
<br/>　　当然了，这归功于两颗妖血炎心果。
<br/>　　难怪慕容千军当初不惜冒险守护在密林，更高价雇佣铁刀猎妖队的人，使尽方法都想得到那三颗妖血炎心果。
<br/>　　妖血炎心果，对后天武者的作用是在太恐怖了。
<br/>　　可惜，他费尽心机，最后还是落到了萧逸手上。
<br/>　　两位长老送的丹药也帮了萧逸很大的忙。
<br/>　　“难怪说武者修炼是一份非常烧钱的职业，啧啧，今天服用的丹药加起来要二万两吧。”萧逸笑着摇了摇头。
<br/>　　萧逸看了眼窗外的夜色，吹了吹这矿脉中特有的萧瑟凉风，再次盘膝坐下修炼，稳固今天跳得太快的修为。
<br/>　　.....
<br/>　　同是那夜色，同是陨星山脉中的清风。但在距离萧家矿脉数十里之外的慕容家矿脉中，夜色是那么的阴沉如水，清风又是那么的冰冷。
<br/>　　慕容家矿脉，议事厅。
<br/>　　夜色深沉，族人们却没有睡去。反而，慕容家的两位长老和数位执事，竟在密切地商量着什么。
<br/>　　“千军现今的伤势如何了，可重？”说话的是慕容家在矿脉中两位长老的另一位，名叫慕容槐。
<br/>　　此人在慕容家的地位高于慕容蝎，更是有名的狠辣之辈，计谋多端，心思阴冷。
<br/>　　慕容蝎道，“伤势有点重，但不至于致命，好好休养一段日子便能痊愈。”
<br/>　　“那就好。”慕容槐点点头，而后将目光看向议事厅的众人。
<br/>　　“今日深夜还唤大家过来，想必也知道所谓何事了。”慕容槐永远是一副冷冰冰的脸，目中那阴鸷之色，让人不寒而栗。
<br/>　　“长老息怒。”一位执事连声道，“我们已经尽力了，可是萧家的人委实不好对付。”
<br/>　　慕容槐冷冷地盯着了这执事一眼，“如果萧家好对付，今夜我们还需要商议吗？”
<br/>　　这位执事听着冷冰冰的声音，顿时一惊，面露惧色。
<br/>　　一旁的慕容蝎出言道，“也怪不了执事们，萧家的人确实不好对付，而且他们的运气也太好了。”
<br/>　　慕容蝎边说着，叹了口气。
<br/>　　“十年前，萧家好不容易走了个萧晨枫，萧家群龙无首，正是最薄弱之时。可偏偏萧离火那个老家伙力挽狂澜，又借着一身威望，生生震慑了那些想打萧家主意之人。”
<br/>　　“如今，那老家伙老了，没魄力了。我们想靠那萧家五长老和他儿子夺了萧家，萧逸却又横空出世，生生坏了我们所有计划。”
<br/>　　“这些天，我们连家主女儿的声誉都用上了，想借机吞了萧家的矿脉，又是被这萧逸生生给坏了。”
<br/>　　“哎，十年了，萧家没了萧晨枫，他儿子却又是个比他还耀眼的绝世天才，这该如何是好？”
<br/>　　慕容蝎连叹了好几声。
<br/>　　慕容槐脸色变得更加阴冷，“萧逸，萧逸，最近我听得最多的就是这个萧逸，就是他把千军给打伤了吗？”
<br/>　　“正是，还把妖血炎心果也夺了。”慕容蝎道。
<br/>　　“哼。”慕容槐冷哼了一声，“这种天才，不能让他成长起来。”
<br/>　　慕容蝎摇了摇头，道“他人在萧家矿脉那边，有萧家的长老保护....”
<br/>　　其他执事也个个面露为难之色。
<br/>　　“闭嘴。”慕容槐冷喝一声，“昨日家主已经遣信给我，务必尽快拿下萧家矿脉。大长老也已经震怒了，萧家那些人要尽快解决。”
<br/>　　“大长老？”慕容蝎和执事们听到这话，顿时脸色一紧。
<br/>　　慕容槐忽然从怀中拿出一物，此物一出，顿时整个议事厅都被一股阴冷所笼罩，所有人都打了个冷颤。
<br/>　　“这...这是大长老的...”慕容蝎看着那东西上不断环绕的黑气，明显瞳孔一缩。
<br/>　　慕容槐冷声道，“我没时间与你们废话，时间紧迫，我已有办法，你们照办即可。”
<br/>　　“是。”众人连声应是。
