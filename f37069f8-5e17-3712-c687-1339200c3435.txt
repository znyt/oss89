<br/>
<br/>        当她看到明司寒突然出现在房门口时，先是一愣，下一秒后，她便回过神来迅速地掀起被子裹住自己。

<br/>        明司寒回过神来后，立即转过身去，但好看的俊脸已经红得像个苹果，只觉心脏跳动的速度都像是快要从胸腔里蹦出来了一样。
<br/>
<br/>        “我去给你买水，你多喝点水，就没事了！”明司寒讷讷地说。
<br/>
<br/>        连星儿也没反对，只是说了声：“谢谢。”
<br/>
<br/>        当明司寒刚走出房门，背后便传来“哐”地一声，又是一阵冷风袭来。
<br/>
<br/>        他出门后，回来时，是扛着一箱矿泉水回来的。
<br/>
<br/>        明司寒走到房门边，再次叩响了房门，轻声轻语地问道：“星儿，我帮你把水买回来了，你……”

<br/>        “咔嚓”一声，房门开了，直接打断了明司寒接下来要说的话。

<br/>        她此刻的眼神已经浑浊迷离，双颊十分红润，整个人的状态处于一种醉后微醺的摇摇欲坠。
<br/>
<br/>        但是，当连星儿看着明司寒时，莫名其妙地发现，这空气里好像全是这男孩身上的特殊味道。
<br/>
<br/>        那是一种很淡很淡像蜜桔般充满诱惑的味道，在连绵不断地窜进她的鼻息。
<br/>
<br/>        好想吃啊……

<br/>        连星儿开始上上下下的打量着明司寒。

<br/>        “我、我把水放地上，你、你自己拿！”明司寒见连星儿如狼似虎般盯着自己，不禁打了个寒颤，说话时也跟着不太流利。
<br/>
<br/>        这个时候，他没想过要乘人之危，反倒是担心自己会不会被这丫头给“吃”了。
<br/>
<br/>        “明司寒，你……”连星儿似笑非笑的唇瓣，逸出有气无力地呼唤。
<br/>
<br/>        正当他以为她要命令他转身出门时，她却忽然两眼一闭，晕了过去。
<br/>
<br/>        明司寒见状连忙上前一步，一把将她拦腰横抱了起来。
<br/>
<br/>        她浑身像个火炉一样，烫得过分吓人。

<br/>        明司寒心中一颤，连忙将她抱到床上后，从衣柜里拿出睡袍裹住她的身子，再重新抱起她离开了总统套房。
<br/>
<br/>        好在他将她送医院送得及时，四十摄氏度的高烧可不是闹着玩的。
