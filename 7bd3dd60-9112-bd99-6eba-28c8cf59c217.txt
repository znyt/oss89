　　“什么办法？”李毅恒听到黄浦说有办法，立刻来了精神。
<br/>　　黄浦这个时候倒也没有再卖关子，脸上露出一抹得意的笑容说道：“找人带我们混进去。”
<br/>　　李毅恒原本还以为黄浦能有什么有办法，结果一听之下立刻失望的摇头说道：“带我们混进去？这怎么可能！”
<br/>　　“为什么不可能，事在人为，能进到皇宫里的人有很多，我们完全可以找到机会混进去。”黄浦反问道。
<br/>　　“就算是可以混进去，又怎么出来，皇宫里面一定守卫禁严，你不会认为在这样的情况下我们能混进去几百人吧？”李毅恒有些抱怨的说着，显然他并不认为黄浦的主意是一个好主意。
<br/>　　“可是除了这个办法之后，我们没有任何其他的途径能够进去了，如果想要杀进去别说现在这些人，就是再多十倍，百倍估计也很难办到。”黄浦的提议一再被否决也有些急了，声音不自觉得大了起来。
<br/>　　“我觉得我们可以对独孤玉凤下手，只要我们能够抓住她，其实去不去皇宫里救她的男人都不重要。”一个声音突然从外面传进来，吓了房间中李毅恒两人一跳。
<br/>　　但是黄浦很快便冷静了下来，伸手推开窗子：“‘喂’先生既然来了，怎么不进来？难道还要老夫出去请你？”
<br/>　　“呵呵……，某家不请自来，也不知道主人家欢迎与否，哪敢擅自进入。”外面那人正是之前与黄浦见过面的‘喂’先生。
<br/>　　不过此人虽然口中说的客气，但其行为却没有任何一点客气的地方，在他的身后此刻正站着五个黑衣人，而在那五个黑衣人的脚下，则躺着七、八个已经晕厥过去的壮汉。
<br/>　　黄浦只扫了一眼便已经看清了外面的情况，冷冷一笑说道：“想不到，‘喂’先生竟然如此热情，来到主人家便先将主人家的护卫给绑了，难道这就是高句丽特有的礼节？”
<br/>　　“嘿嘿……，鬼面先生何必如此斤斤计较呢，某家只不过是怕他们打扰了我们的谈话罢了。”所谓的‘喂’淡淡一笑，随意的一挥手，他身后的五个黑衣人便弯下腰将地上的躺着的几个人拖了起来，几人合力全都拖了出去。
<br/>　　黄浦眼睁睁看着自己的几个手下被人拖走，却没有任何异样的表现，只是冷冷的哼了一声，略有些不屑的说道：“是么？那不知道‘喂’先生想要谈些什么。”
<br/>　　“合作，不知道鬼面先生敢不敢！”‘喂’先生独身一人在在院子里面，隔着打开的窗子与黄浦地视着，胸有成竹的说道。
<br/>　　刚刚他在外面的时候已经把房间里李毅恒与黄浦的对话听了清楚，知道他们现在面对的是什么样的困难，所以在他看来对方根本没有什么选择的余地。
<br/>　　但意外的是在黄浦还没有开口的时候，李毅恒已经出现在他的身边，用十分不屑的语气说道：“不好意思，我们从来不跟藏头露尾的人合作，‘喂’先生请回吧。”
<br/>　　被李毅恒拒绝之后，那‘喂’先生显然有些意外，想了想之后说道：“名字不过是一个人的代号而已，叫什么真的那么重要么？如果你们一定想要知道，某家完全可以现在就编一个名字出来。”
<br/>　　“名字叫什么的确并不重要，但却代表了一个人是否有诚意，你连名字都是编的，连自己的父母留给你的东西都不珍稀，你还想让别人相信你？还想让别人跟你合作？”李毅恒摇了摇头，深吸一口气：“所以你还是走吧，看在有过一次交易的份上，这一次你擅自偷听的事情本公子便不追究了。”
<br/>　　‘喂’先生显然被李毅恒的话刺激到了，脸上露出狰狞的笑容说道：“笑话，这位李公子，请你看清楚眼下的形势，你的人现在已经全都被我控制了，这个院子里也只剩下了你和鬼面修罗，你觉得你还有选择的余地么？”
<br/>　　不知是出于自信还是自大，李毅恒像是根本没有听到‘喂’先生的话一样，满不在乎的对他说道：“那你可以试试看，只要你的人能进来这间屋子，本公子便一切都听你的安排。”
<br/>　　局面似乎一下子就僵持住了，虽然那个所谓的‘喂’先生很有把握可以将黄浦与李毅恒两人抓住，可是他眼下需要的却是合作，而不是将两个人都杀了。
<br/>　　黄浦似乎也看出了他的犹豫，在暗处给李毅恒打了一个手势，示意他稍安勿躁，然后发出一阵如同夜枭般的笑声：“桀桀……，‘喂’先生，不如这样好了，只要你敢只身进来这间屋子，咱们便继续谈谈合作如何？”
<br/>　　“好啊！劳烦开一下门。”自认已经吃定李毅恒和黄浦的‘喂’先生没有任何犹豫，轻轻一点头，抬步便向着房间的门口走了过来。
<br/>　　不得不说，此人胆气的确是有些，因为虽然他相信李毅恒和黄浦两人是瓮中之鳖，但是独身进入房间还是有些风险的。同样的事情如果放在李毅恒身上，他相信自己决不会去冒这个险，哪怕是不能完成合作！
<br/>　　“啪嗒，啪嗒”脚步声不停，很快‘喂’先生便已经来到了房间的门口。
<br/>　　“嘎吱”一声，房间的门被人从里面打开，黄浦肃手而立：“请！”
<br/>　　“鬼面先生客气了！”‘喂’先生点头为礼，脚下不停，数步间已经进了房间之中，并且还找了一个位置坐了下来。
<br/>　　“哼，先生好大的胆子，竟然真的敢独自进来。”李毅恒看着神态自若的‘畏’先生，不屑的哼了一声。
<br/>　　‘喂’先生耸耸肩膀，做了一个那又如何的表情：“李公子还是不要说这些没用的东西了，现在某已经进来，我们是不是可以继续谈一下合作的事情？”
<br/>　　“好啊，那就说说你的打算好了，让我们听听你打算怎么合作，合作的目的又是什么。”黄浦此时已经关好了房门，回到了房间之中，与李毅恒并肩站到了一起。
