　　苏莫离开战界之后，便回到了火元峰，身形落在了自己的洞府门前。
<br/>　　抬头看了一眼山峰上方，那火元圣师居住的宫殿，他面现沉吟之色。
<br/>　　这次出去了这么久，回来了要不要去向对方见礼？
<br/>　　不过，略作沉吟，他还是摇了摇头，抛却了这个想法。
<br/>　　以前他拒绝火元圣师，应该是让对方心生不满了。
<br/>　　对方虽然让他居住在火元峰，但没说收他为徒，也没说教授炼器，完全就是将他晾在一边。
<br/>　　如此的话，他也不需要向对方见礼了，就这样也挺好。
<br/>　　他住在火元峰，还能借助对方的虎皮，不至于认人欺辱。
<br/>　　随即，苏莫叹了口气，走进了自己的洞府中。
<br/>　　而后，苏莫拿出了太史轩的储物戒，查探了起来。
<br/>　　太史轩相当富有，储物戒内下品灵晶足有两万多块，兵器倒是不多，只有三件，全部是长枪。
<br/>　　唰！
<br/>　　苏莫意念一动，将一杆银白色的长枪取了出来。
<br/>　　长枪入手温润，通体白光流转，其内蕴含磅礴的气势，赫然是一杆尊级下品宝枪。
<br/>　　“不错！”苏莫面上露出了一丝笑容，这是他得到的第二件尊级宝器。
<br/>　　虽然是长枪，不是长剑，不太适合他使用，但也很值钱。
<br/>　　苏莫在天冥星这一段时间，杀了无数的巫族之人，却并没有得到一件尊级宝器，可见尊级宝器的珍贵。
<br/>　　当然，巫玛和巫幽兰可能有尊级宝器，但大战之下，储物戒都烟消云散了。
<br/>　　还有巫天御，苏莫记得，当初对方去苍穹世界之时，曾经所用的便是尊级兵器。
<br/>　　但他杀死了巫天御之后，在对方的储物戒里搜寻了一番，却是并没有。
<br/>　　或许，那只是巫天御去苍穹世界之时，巫族临时赐予的兵器。
<br/>　　打量了一番这杆宝枪，苏莫郑重的将其收了起来，随即他又从储物戒中取出了六枚玉简。
<br/>　　“希望能有尊级的功法武技！”苏莫望着六枚玉简，暗暗期待了起来，太史轩身份不凡，修为也不低，说不定有尊级的功法武技。
<br/>　　而后，苏莫便开始查探了起来。
<br/>　　“天阳心经！”苏莫查探之下，第一个玉简之内，便是一门尊级功法。
<br/>　　这门功法是尊级下品，至刚至阳，威力非凡。
<br/>　　不过，却并不适合苏莫修炼。
<br/>　　随即，他继续查探，但接下来几枚玉简，却不再有尊级武学，全部是帝级上品武学。
<br/>　　这让苏莫有些失望，天阳心经不是五行属性的功法，并不适合他修炼。
<br/>　　随即，苏莫收起了几枚玉简，心中沉吟了起来。
<br/>　　他心中有改修功法的想法，而所修对象，正是混天魔功。
<br/>　　但是，混天魔功虽然强大，修炼的不够精深的话，所能展现的战斗力，却是并不如大混沌五行神诀。
<br/>　　只不过，随着修为的不断提高，他大混沌五行神诀融合的五种功法，等级也会渐渐的跟不上了。
<br/>　　“也不清楚大混沌五行神诀，能不能融合尊级的功法！”苏莫心中暗暗猜测，若是能融合尊级的功法，他绝对不会改修混天魔功。
<br/>　　此事，还需从长计议，等到有了五行属性的尊级功法之后，实验一番才能知道。
<br/>　　嗖嗖！
<br/>　　这时，外面响起了两道破空声，聂乾和韩盈身形落在了苏莫的洞府门前。
<br/>　　“苏兄，不好了，太史世家派系的人，都在叫嚣着要杀你！”聂乾大步的走进了苏莫的洞府，满脸担忧的说道。
<br/>　　“无妨，我就不信他们敢来火元峰杀我！”苏莫摇了摇头，心中丝毫无惧。
<br/>　　他与太史轩是在战界公平一战，双方自愿，没有任何人能说出他的罪责。
<br/>　　太史世家派系的人，就算再恨他，也不敢来火元峰杀他。
<br/>　　“可是……这样下去也不是办法啊！难道你就一直在火元峰待着？”聂乾皱眉说道。
<br/>　　虽然待在火元峰安全，但总不能一辈子龟缩在此吧！
<br/>　　“天无绝人之路之路，神武学府之内，又不是只有太史世家派系！”苏莫淡淡的说道。
<br/>　　“苏兄，你的意思是加入其它派系！”随后走进洞府的韩盈，闻言立刻眼前一亮，明白了苏莫的意思。
<br/>　　聂乾也不傻，转念一想，便也明白了。
<br/>　　难怪苏莫敢杀太史轩，原来是早有应对之策啊。
<br/>　　“不错！神武学府由十三个大势力组建而成，学府内有十三个派系，各大派系之间明争暗斗，争夺天才弟子！”
<br/>　　苏莫点了点头，眸中精光闪烁，继续道：“我展现了这么强的天赋，我就不信没有人来邀请我！”
<br/>　　苏莫心中早有想法，上次他斩杀冷江之后，其它派系的人便纷纷来拉拢他，只不过他没有答应而已。
<br/>　　现在，他展露了更强大、更恐怖的天赋战力，其它派系的人怎么可能坐视不理？
<br/>　　太史世家派系或许非常强大，但其它的派系同样强大，可不会惧怕分毫。
<br/>　　苏莫面上露出了淡淡的笑意，如果他所料不错的话，要不了多久，就有人来邀请他了。
<br/>　　“苏兄，没想到你心思如此缜密，我还以为你是一时冲动呢！”聂乾脸上露出尴尬之色。
<br/>　　他只顾着担忧了，完全没想到这一茬。
<br/>　　“苏兄天赋无双，心机不凡，真是让我汗颜！”韩盈摇了摇头，她感觉自己与苏莫之间，差距真是不小。
<br/>　　其实，这也不是什么高深的计谋，只不过她和聂乾两人，一想到得罪了太史世家派系，有些慌了神，完全不够冷静。
<br/>　　“聂兄、韩姑娘，这十三个派系之中，除了太史世家派系，其它派系你们了解多少？”苏莫向两人问道。
<br/>　　既然要加入派系，自然不能随便加入，因为加入了某个派系，将来离开学府之后，便要加入这个派系所在的势力。
<br/>　　不然的话，便会面临所有势力的绞杀，成为公敌。
<br/>　　所以，此事非同小可，必须仔细斟酌一番。
<br/>　　“这个我比较了解！”
<br/>　　聂乾点了点头，继续道：“十三个派系，乃至背后的十三个势力，大致都差不多，并没有某个很强，或者某个很弱的情况……！”
<br/>　　而后，聂乾详细的向苏莫讲解了一番十三个派系，以及十三个派系背后所在的势力。
<br/>　　聂乾在神武学府待了十年，虽然没有加入任何派系，但对于这些情况还是相当了解的。
