<br/>				    家，在国人的概念中，就是一个避风的港湾，一个安全的窝，能够让自己在身心疲惫的时候能恢复的地方。<br/>　　<br/>　　    所以国人对于家，有着执着的。也因为这个原因，所以很多人才会将一辈子都搭上，也要买个房子，努力去营造自己的小窝，一个温馨、温暖的小窝！<br/>　　<br/>　　    和家里人一起吃过饭以后，本来姐姐陈萍要让陈默一起去酒厂的工地转转，但是最后却没有去成。<br/>　　<br/>　　    陈默回来后，想先去看看德林叔和四叔，所以给姐姐说过后，也让陈萍点头答应。在她建设酒厂期间，德林叔和四叔也是跑前跑后的，因此是要过去感谢一下的！<br/>　　<br/>　　    陈默将乾坤珠内以前买的好烟等，弄些些出来。村里人窜个门，平时并不会送礼什么的。只要人过去就成，只有过节过年的时候，买些礼品盒或者牛奶作为访客礼物，甚至有的时候，送方便面的也是有的。<br/>　　<br/>　　    这也是过年过节的时候，农村中一些礼盒什么的，会有很多种类，一些盗版的形似而非的什么七个核桃，康帅傅等大行其道的原因！<br/>　　<br/>　　    好在陈默倒是不会因为钱的原因，买些便宜的东西。他买的都是比较贵的烟等礼品，专门用来送礼的！<br/>　　<br/>　　    而且前文也是说道，在农村，真的要是走亲戚送礼等，香烟也没有必要送多贵的，村里人抽烟只要差不多就成。<br/>　　<br/>　　    但是对于德林叔，自然是诚心感谢，所以就拿出来两条好烟，然后又拿出两瓶市面上卖的比较好的酒，直接就转身去了村委办公室。<br/>　　<br/>　　    “德林叔，你老忙着呢？”一进去，就看见德林叔正戴着老花镜在看报纸，所以就调侃道。<br/>　　<br/>　　    “去！听声音就知道是你，怎么，出去了好多天，终于回来了？”德林叔放下报纸和眼镜后，也是笑着问道。<br/>　　<br/>　　    “嗯！今天早上才回来的，刚才在家吃过中午饭。这不走了十几天了，想着过来看看你！”陈默说道。<br/>　　<br/>　　    “我有什么好看的，还不是一个嘴巴两个眼睛的！倒是你，这次出去做什么去了？上次问过你达，竟然还不告诉我！”德林叔说道。（达，方言，指父亲。）<br/>　　<br/>　　    “其实我爸他也是不知道我去干什么了，咋能说给你呢？我是去见朋友去了，在他们家里玩了几天，这才回来的。”<br/>　　<br/>　　    陈默出去给人看病，但是却不想给其他人说，所以就给所有知道的人，都交代了一遍，不要说自己是出去给人看病的，要不然以后自己不忙死也会累死！<br/>　　<br/>　　    所以打个哈哈也就过去了！至于说德林叔如果有病什么的，那么自己义不容辞，但是如果是其他人找来了，要是要求陈默出诊，那还不是类似啊！<br/>　　<br/>　　    “德林叔，这是我出去后带回来的烟酒！”说着将烟酒放到了办工作上。至于说有没有影响，呵呵！是不存在的。农村里送个东西什么的，大家都认为很正常。<br/>　　<br/>　　    德林叔笑着看了看，也是点点头，对于陈默来说，本来就有本家的关系，而且以前就很照顾陈默，所以现在送个烟什么的，也就没有什么好说的，手下就成。<br/>　　<br/>　　    “咦？德林叔，你上回不是说上面要安排村长什么的，怎么到现在整个村委还是这些人啊？”陈默问道。<br/>　　<br/>　　    “哈哈！还能怎么说，还不是镇上还在开会讨论，虽然人已经确定并且报上去了，不过却在县里给卡住了，说是在研究研究！”德林叔也是有些郁闷，以前也不是没有安排过村长什么的，都是一句话的事情。<br/>　　<br/>　　    现在倒好，既然弄了这么长时间，却依然没有丝毫的动静了。但是德林叔也是很有见地的，在村委做了这么长时间的基层工作，看的很透彻。<br/>　　<br/>　　    不是上面不给力，主要是现在陈默哪里似乎有些不平静，而且在他哪里居住的人，都是一些有来头的人，所以上面的人想要安排人员过来，自然也就受到了牵制。<br/>　　<br/>　　    谁都想将自己的关系户安排过来，只要在这些个老头面前有了个印象以后，自然以后有些路途也能顺畅一些不是。<br/>　　<br/>　　    甚至有些领导自己都想担任村长，所以这样一来，就造成了大家都在争，但是倒是安排谁，也是个问题，因此这个村长什么的，就这么拖了下来。<br/>　　<br/>　　    陈德林无所谓，反正安排谁过来，至少在这个一亩三分地里面，谁来都要看着他的脸色不是，所以来也就来吧，不来也就算了！<br/>　　<br/>　　    他这个行为，倒是能和那种无欲无求相同了。至于说上层领导想询问他的意见，也是被他推了回去，爱咋地咋地！得罪人的事情，他可是不能做的！<br/>　　<br/>　　    陈默和德林叔聊了一会后，就告辞出来，有拿着东西去了四叔哪里！虽然四叔帮助他的事情也没有多少，但是作为后辈，自然还是要对长辈恭敬。<br/>　　<br/>　　    另外就是无论是在建设酒厂事情上，还是以前家里比较困难的时候，四叔也是帮助了不少！所以人要长怀感恩的心思！<br/>　　<br/>　　    另外就是，现在酒厂还没有建成的时候，四叔还是每个月都要烧些酒，然后给陈默送过去，也让陈默这些时间来，能够存下上万斤自酿白酒，都被他放到乾坤珠内，这些酒，可以用来送礼，也可以给自家亲属喝，都是不错的！<br/>　　<br/>　　    去四叔哪里，被四叔拉着又喝了二两酒，然后才告辞出来。四叔一辈子酿酒，也是非常爱喝酒的人，所以整天都是一身的酒味，从小的印象中就是这样！<br/>　　<br/>　　    陈默现在的身体素质，即使这样的自家酿酒，喝上个几斤都不会醉，所以二两酒对他来说没有什么问题。<br/>　　<br/>　　    出来后，就看到大黄和大灰正在门口位置守着，一直等待着陈默。见到他一出来后，就立刻像是保镖一样，一前一后的跟随上来。<br/>　　<br/>　　    也让村里其他人见到以后，也是羡慕的不成。自家养的土狗，根本就没有这样的行为，所以都是见到陈默以后，称赞不已！<br/>　　<br/>　　    甚至有的人，还说道等以后下狗崽了，能留一只给他！<br/>　　<br/>　　    陈默笑呵呵的回复，第一窝狗崽早就被预定完了，根本么就没有了！要等也要等以后！<br/>　　<br/>　　    说起来也是，包括大海，德林叔等等，都是预定了的！所以小狗狗是不够的！<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
