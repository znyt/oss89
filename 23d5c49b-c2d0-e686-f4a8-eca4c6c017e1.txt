　　苏莫和褚彦、沁儿、骆妤三人一起，落在了时空之城下方的大地之上。
<br/>　　这时，骆妤向苏莫看了过来，冷冷的道：“段惊天是吧？我们不是同类人，你就不要跟着我们了！”
<br/>　　苏莫闻言一怔，不过，既然别人不欢迎，他也不会死皮赖脸的跟着。
<br/>　　“既然如此，那在下告辞了！”苏莫抱了抱拳，随即转身快速离去。
<br/>　　“这……！”沁儿见此，顿时有些无语了。
<br/>　　毕竟，他都答应同行了，骆妤居然还把人家赶走了。
<br/>　　随即，她看向骆妤，面带幽怨的道：“骆妤姐，我都答应人家了，你这不是让我很没面子嘛？”
<br/>　　骆妤闻言，有些无奈的摇了摇头，告诫道：“沁儿，你就算交友，也要分清对象，不同的身份地位、不同的天赋实力，都有不同的圈子，你们不是一类人！”
<br/>　　“沁儿，你太天真了！”
<br/>　　褚彦亦是点了点头，道：“此人就是个蝼蚁般的人物，我看他不怀好意，若不是这里是天池神朝，我早就对他不客气了！”
<br/>　　褚彦心中暗叹，沁儿太年轻了，又从来没有出过远门，根本不清楚人心险恶。
<br/>　　在他看来，这个段惊天，很可能是看上了沁儿和骆妤的美貌，借此结交。
<br/>　　又或者是，早就知晓他们的身份，想要攀龙附凤，这都很有可能。
<br/>　　“你们也太谨慎了！”
<br/>　　沁儿有些无奈，随即摆了摆手，道：“好吧好吧，都听你们的，我们现在就回人王宫吧！”
<br/>　　“嗯，我们走！”骆妤和褚彦点了点头。
<br/>　　随即，三人不再逗留，立刻就离开了极空城，向轮回神朝而去。
<br/>　　……
<br/>　　苏莫在极空城之中转了一圈，将身上的一些杂七杂八的宝物，全部都售卖给了一家商铺。
<br/>　　他身上的宝物并不多，也就卖了七千下品源石。
<br/>　　不过，神域的下品源石，比神路最好的源石都要好。
<br/>　　当然，因为修炼五行之力，所以，苏莫索要的源石，全部是五行源石。
<br/>　　随后，苏莫便也离开了极空城，向轮回神朝所在的方向而去。
<br/>　　为了不浪费时间，他御剑而行，一边修炼一边赶路。
<br/>　　之前，他打听了一番，神域各朝之间，也有传送阵，只不过价格相当的高。
<br/>　　而且，乘用传送阵，也有风险，他不能留下任何自己去轮回神朝的线索。
<br/>　　苏莫心中清楚，他去轮回神朝，暂时只能隐匿身份，绝对不能露出丝毫的破绽。
<br/>　　还有就是，他不能距离帝一魂和拓跋青云太近，要保持足够的距离。
<br/>　　反正，就是有拓跋青云和帝一魂的地方，他就决不能出现，哪怕是隐藏身份出现都不行。
<br/>　　苏莫急速而行，脑中依旧在参悟吞噬奥义，他要将吞噬奥义参悟道圆满，参悟吞噬法则，才好晋升到真神之境。
<br/>　　与此同时，他的手中，不时的出现一枚枚源石，吸纳源石中的源气。
<br/>　　壮大圣丹，增强混沌之力。
<br/>　　时间一日又一日，匆匆而过。
<br/>　　转瞬之间，便过去了两年。
<br/>　　两年的时间，苏莫倒是一路顺风，没有遭遇任何的危险。
<br/>　　“应该快了！”
<br/>　　苏莫睁开了眼眸，望着前方一望无际的辽阔大地，低声自喃。
<br/>　　天池神朝和轮回神朝接壤，而时空界域，距离轮回神朝，也算是比较临近。
<br/>　　苏莫估计，再有大年的时间，基本上就能到达轮回神朝了。
<br/>　　苏莫微微的叹了口气，他的一切资源都耗尽了，连灵石都被用完了。
<br/>　　他已经很久，没有这种资源枯竭的感觉了。
<br/>　　但是，在这神域之内，他又不敢动用吞噬元神，大规模的吞噬天地源气。
<br/>　　不然，一旦被人发现他的元神，很可能就会让他陷入绝地。
<br/>　　苏莫继续飞行，赶往轮回神朝。
<br/>　　这一日，他的前方出现了三道身影，正是褚彦三人。
<br/>　　苏莫睁开了眼眸，眸中露出意外之色，这么久了，居然还能遇见这三人，看来他们走的路线差不多。
<br/>　　而苏莫发现褚彦三人之时，褚彦三人，也发现了后方的苏莫。
<br/>　　“呀！这个段惊天居然在后面？”沁儿面带惊讶之色。
<br/>　　“冥顽不灵！”骆妤俏脸一沉，面目冰冷了下来。
<br/>　　她们一路之上，走走停停，速度不算快，没想到这人居然追了上来。
<br/>　　“岂有此理！”
<br/>　　褚彦回头看向苏莫，眸中冷光一闪，这个蝼蚁居然一直在跟踪他们。
<br/>　　“沁儿、骆妤，你们先走，我去教训教训他！”褚彦停下了身形，转头等待后方的苏莫。
<br/>　　“不行，褚彦哥，你怎么能这样？”沁儿一听，顿时也停了下来。
<br/>　　在她看来，这个段惊天并不像恶人，对方只是要去轮回神朝历练而已。
<br/>　　“我们一起吧！”骆妤同样停下了身形，她倒要看看这个段惊天要耍什么花样？
<br/>　　如果此人心存歹意，她们绝不会留情。
<br/>　　她们之前也发现了，苏莫的修为，达到了虚神境四变。
<br/>　　但是，虚神境四变的修为，在她们的眼中，依旧不值一提。
<br/>　　三人伫立在半空之中，静等苏莫到来。
<br/>　　后方，苏莫见此，速度不减，向三人径直而去。
<br/>　　片刻之后，他身形停在了三人前方万丈之处。
<br/>　　“三位，不知又何贵干？”苏莫收起了灵剑，长身而起，向三人抱拳问道。
<br/>　　“小子，你居然跟踪我们，你这是在找死吗？”
<br/>　　褚彦冷喝一声，身上气势涌动，仿佛立刻就要出手。
<br/>　　“给我们一个理由，不然，今日别怪我们不客气了！”骆妤冷着脸说道。
<br/>　　沁儿没有说话，只是静静的望着苏莫。
<br/>　　苏莫闻听两人之言，眉头一皱，这两人有病吧？
<br/>　　他们只是走的同一条路线而已，对方两人居然认为他在跟踪？
<br/>　　这简直是可笑！
<br/>　　“难道你们不知道我要去轮回神朝？”
<br/>　　苏莫环视两人，淡淡的说道，随即，他继续道：“我只是飞的慢而已，才落在了你们的后面，你们若是不赶路，那我可要赶路了！”
<br/>　　言罢，苏莫不由分说，绕过三人，径直向前方飞去。
<br/>　　“嗯？”
<br/>　　褚彦和骆妤两人见此，顿时微微一怔。
<br/>　　这人倒是坦然，难道真的只是同路？
<br/>　　“褚彦哥、骆妤姐，我们别为难别人了，他上次在极空城，不就说要去轮回神朝吗？”沁儿对两人说道。
