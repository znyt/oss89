        &lt;h3&gt;第三卷第二百一十章  闫欢准备去甸城&lt;/h3&gt;
<br/>
<br/>        闫欢在家里陪着家人过了一个新年，心里谋划着怎么重返甸城，妹妹回来了更是坚定了她去甸城的心思。
<br/>
<br/>        她带着闫芈参观了她们所有的超市乃至西城的产业，虽然大股东是陈子昂和陈子悦，但是借助陈家姊妹的力，她现在也是股东之一，每天源源不断的进项让她越来越富有。
<br/>
<br/>        站在八楼看着楼下川流不息的人群，看着来来往往的消费者，闫芈说：“我们的这个大楼目前是西城最大的一个集购买休闲吃饭为一体的商圈，这里面涵盖了儿童、成人的各种品牌的奢侈品，每天人流量源源不断，最大的流量能达到十万次。”
<br/>
<br/>        “姐，你们太厉害了。”闫芈感慨着，姐姐失踪了几年，回来就到了超市上班，短短几年已经跃为西城屈指可数的人物。
<br/>
<br/>        “是机遇，我们抓住了机会，也是社会发展的必然规律。你看，我去甸城开这样的一个超市怎么样？甸城有吗？”
<br/>
<br/>        “没有，只是你这还带着吃饭的地方。甸城有超市却没有这种模式，不过最近几年也发展了一些大型的超市，都是谷家的产业，你要是想去甸城，也只能是和谷家合作，而谷家和秦家其实是一体的。”闫芈当然知道谷家和秦家的关系不一般，虽然她没有接触到核心的东西，但是只要是谷家的项目都是直批，就拿有一次她驳回了谷正娟的项目，可是不久，谷正娟改了申请，项目还是和原来一样最后都给批了。
<br/>
<br/>        “我认识谷强，我可以找谷强合作，谷家不用出资就有钱可赚，他们不会不答应吧？”闫欢笃定的说。
<br/>
<br/>        “我都要走了，你何苦要去趟那趟浑水？”闫芈叹口气。
<br/>
<br/>        “为了你的幸福，我必须得去趟这趟浑水。”闫欢主意一定，她要和闫芈一起去甸城，她等不及陈子昂了。
<br/>
<br/>        闫芈这次回家其实也是一个锲机，她可以以送闫芈为借口，和秦家人拉上关系，然后在慢慢接触谷强，按照闫芈所说，甸城经济还是很落后，对于他们来说，落后的地方才有钱赚，因为当地要发展，在社会前进的步伐中，在改革的浪潮的推动下，每一个地方都在进行着轰轰烈烈的改变，有改变就有机遇，有机遇就有机会，有机会就要抓住，闫芈这一次回家就是最好的机会，比陈子昂去甸城的时机还要好，还要合适。
<br/>
<br/>        闫芈说了，秦家和谷家是一体的，陆玉和秦少卿是青梅竹马，那么陆家和秦家也是密不可分的，秦家、谷家、陆家，她被送到那个黑暗的地方，一定和他们都有关系！
<br/>
<br/>        闫欢拨开了迷雾，她的心被提起来了，谷强肯定是知道这个情况！谷强你还是原来的陈子寒吗？
<br/>
<br/>        “姐，你不要过去了，我回去找一找秦少卿出轨的证据，他们原本都各自有家庭，你也说了，张家也是一个大家庭，陆玉怕是也不敢明目张胆的胡来，只要我找到他们的证据，我能够安全的带着皓儿离开就行了。”闫芈叹了口气，她就不应该给姐姐说那些话。
<br/>
<br/>        “我过去投资做事情啊，再说了我们原本就打算过去，陈子悦的妹妹陈子昂今年也会过去，秦少卿搞的那个信息化工作将来是要和陈子昂合作的，他们都已经商量好了，而我们现在的产业最大的股东也是陈子昂，所以没有你，我们今年也是去定了。”闫欢知道妹妹的忧虑。
<br/>
<br/>        “那好吧。”闫芈知道了姐姐过去投资不完全是为她，心里松了一口气，她和秦少卿肯定会闹掰的，只是结果到底是什么样子，她也不清楚。
<br/>
<br/>        闫欢将她的妹妹介绍给陈子悦；“我妹妹闫芈，甸城的一个部门老总，掌管甸城经济，另外还是秦副总的老婆，她就是我过去的依仗和靠山。”
<br/>
<br/>        “闫芈好，很高兴认识你，呵呵，你还需要依仗和靠山？这么几年也没有见你靠谁？”陈子悦笑着招呼闫芈并且挤兑了闫欢一句。
<br/>
<br/>        “你不是担心我嘛，你看看我有这么样的一个妹妹，你担心什么？我这几天就会收拾收拾和闫芈一起过去，先做好前期工作，等子昂过去我们就可以一起并肩战斗了。”闫欢笑眯眯的说，其实只有一提起陈子昂，陈子悦必然会答应的。
<br/>
<br/>        “行吧，行吧，你先过去，到时候我也可以过去看看。”陈子悦不再多说什么，他们都一门心思往甸城跑，她阻止不了。
<br/>
<br/>        “你哪里都不要去了，就在这里坚守阵地，如果你还想发展的话周边的省份你可以去跑一跑，去甸城投资也不小，我想做一个样板工程，一举拿下那边的市场。”
<br/>
<br/>        闫芈听着姐姐和陈子悦的互动，突然就有些羡慕俩个人的相处方式，虽然闫欢和她是姊妹，但是有时候说话还客客气气的，看看他们俩个人，配合默契，似乎她才是一个外人。
<br/>
<br/>        “好的吧，那我就在这里坚守阵地，你有什么需求了告诉我，我是你后方坚强的堡垒。”陈子悦也不再坚持，既然他们都想去甸城发展，多个人总是多一份力量吧，闫欢和陈子昂算是一路人，坚强不屈，越挫越勇，不像她优柔寡断。
<br/>
<br/>        “行，我这就收拾收拾，随闫芈一起走，家里的钥匙交给你了。”
<br/>
<br/>        “这就要走？”陈子悦想到这几年和闫欢携手前进，俩个人可以说是不离不弃，闫欢要走了，她的心突然就空落落的，不自觉的就疼起来。
<br/>
<br/>        “闫芈要回去了呢，我随她一起。”闫欢心里也不舍，但是想到谷强，想到身边的闫芈，想到她在甸城那一段黑暗的日子，她义无反顾，现在的日子确实很安逸，也很风光，但是还有人在黑暗中挣扎着。
<br/>
<br/>        陈子悦不知道闫欢所想：“我们还是和子昂商量一下，毕竟你过去也算是投资，子昂这几年从来没有过问我们的收益，但是你要拿出去投资就不一样了，这不是小资金。”
<br/>
<br/>        “可以，你给子昂打电话说一下，万一不行了就将我的股份撤出来，我先过去考察市场，回头再具体商量投资的情况。”
<br/>
<br/>        去甸城是陈子昂提出来的，陈子昂曾经问过谷强的事情，按照陈子昂那聪慧的头脑，一定是会发现端倪的，要不然她不会坚持要去甸城，有些事情应该让陈子昂知道。
<br/>
<br/>        经过多年的沉寂，闫欢再次踏上了去甸城的路，第一次是被人强迫过去的，这一次是因为她完成心中的梦想！
