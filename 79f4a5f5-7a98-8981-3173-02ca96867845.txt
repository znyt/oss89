“既然你不喜欢她，那这么多年你还宠着她干什么，整个四九城都知道你变态的宠着她！”
<br/>
<br/>        顾夜瑾看着顾夫人，轻敛了一下俊美的眼睑，然后低声道，“毁了一个人最好的方式难道不是将她捧上天，然后再让她摔在地上么，这么多年我一步步的让她爱上我，18岁成人礼那一天在她满心欢喜的将自己全部托付的时候，我再让她知道，其实她什么都不是，我爱的是别的女人，妈，我这样做，你难道还不满意？”
<br/>
<br/>        被这么一说，顾夫人当然满意了，她也很开心的看着两年前叶翎搬离了顾家，只身一人离开了海城。
<br/>
<br/>        她这么一走，就是整整两年。
<br/>
<br/>        可是顾夫人不满的是，叶翎到底是跟自己的儿子发生了关系，这让她觉得厌恶。
<br/>
<br/>        她的丈夫深深的迷恋着孙怡，孙怡生前跟她闹离婚，孙怡死后让她变成活寡妇，现在她自己的儿子又睡了孙怡的女儿叶翎，这种错综的关系让她深深的不喜。
<br/>
<br/>        “夜瑾，妈这一辈子过得特别的失败痛苦，”顾夫人眼眶发红的看着顾夜瑾，“你爸和孙怡折磨了我一辈子，在我成为废人的那一晚里我曾经抱着一瓶安眠药准备自杀，但是你推门而入了，夜瑾，妈妈舍不得你，妈妈想要陪着你一起长大，看着贵为顾家长子嫡孙的你成家立业，拥有美满的一生，你是妈妈活下去的冲力，全部的希望，所以你千万不要让妈妈失望，如果你也抛弃了妈妈，那妈妈就真的没有活下去的必要了。”
<br/>
<br/>        顾夜瑾走上前，在顾夫人面前半蹲下来，他拿过毛毯盖在了顾夫人残疾的双腿上，低声呢喃道，“妈，我知道了，我不爱她，我是不会爱她的。”
<br/>
<br/>        他不知道说给顾夫人听，还是说给自己听。
<br/>
<br/>        顾夫人到底是心疼自己儿子的，她摸了摸顾夜瑾的头，“夜瑾，那妈妈先回去了，妈不想逼你，你人大了，有自己的想法，这一次叶翎回来，希望你自己处理好。”
<br/>
<br/>        黑衣保镖走了进来，将顾夫人给推走了。
<br/>
<br/>        ……
<br/>
<br/>        顾夜瑾在客厅里站了一会儿，然后他抬眸，看向楼上的房间。
<br/>
<br/>        那扇房门紧紧的闭着。
<br/>
<br/>        顾夜瑾抬脚上楼，推开了房间门，房间里空无一人。
<br/>
<br/>        叶翎已经不见了。
<br/>
<br/>        她去哪里了？
<br/>
<br/>        这扇门一直关着，她怎么出去的？
<br/>
<br/>        顾夜瑾眸色一沉，迅速走进了阳台，阳台的窗户开着，下面的草坪上有摔落的痕迹。
<br/>
<br/>        她是从二楼阳台跳下去的！
<br/>
<br/>        顾夜瑾俊美的眉心里覆上一层阴鹜的寒霜，他拿出手机打电话，人已经下了楼，拿着车钥匙就出了别墅大门。
<br/>
<br/>        叶翎的电话打不通，那端传来着机械的女声对不起，你所拨打的电话暂时无人接通。
<br/>
<br/>        顾夜瑾眼角猩红，这两年她真的变了很多，从乖巧软糯变得明艳带刺，明明那么怕疼的她，现在已经可以从那么高的阳台上往下跳。
<br/>
<br/>        这时有冰冷的雨滴打在了他的手上，下雨了。
<br/>
<br/>        ……
<br/>
<br/>        雨很大，街上的行人打着伞，都在焦急的往家赶。
<br/>
<br/>        叶翎一撅一拐的走在大街上，她腿上摔流血了，鲜血顺着她白腻的肌肤往下滑落，最后没落进了地上的小水塘里。
<br/>
<br/>        她身上的衣服全湿，又痛又冷，像个落汤鸡。
<br/>
<br/>        实在走不动了，应该离的很远了，叶翎坐在一个小花坛边，蜷着双膝，缓缓的抱住了自己。
