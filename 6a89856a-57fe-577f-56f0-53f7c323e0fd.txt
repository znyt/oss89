　　一共四个玉盒，全部排放在了楚风眠的面前。
<br/>　　“这是我们古家所有的，青璃草，琉璃果，献佛花，天雷木。”
<br/>　　古家家主一边开口，一边大手一挥，这四个玉盒全部打开，其中装在的灵药，顿时映入到了楚风眠眼帘之中。
<br/>　　这四件灵药，全部都是难得一见的圣药，刚刚打开，浓郁的药香，便是充斥在整个密室之中。
<br/>　　这种药香，让一些普通人闻到，都足以强身健体，延年益寿，甚至是让一些实力稍低的武者，闻到药香，都有着一种极大的作用。
<br/>　　毕竟这可是圣药，在九域之中都是难得一见，甚至是许多生死境巅峰的人物，都不一定能够拥有圣药，只有那些超越生死的老古董，才有资格拥有圣药。
<br/>　　“这古家的底蕴，果然不错。”
<br/>　　楚风眠看到古家家主拿出的这四株圣药，心中暗自想到。
<br/>　　炼制玉露破命丹的圣药，可都不是寻常圣药，比起那种普通的圣药可以比拟的，在圣药中都算是极为珍贵之物了。
<br/>　　古家一口气居然都拿出四个出来，这已经是在楚风眠的满意范围了。
<br/>　　有了这四株圣药，在加上楚风眠身上所拥有积累的，起码足以是凑齐一半的数量了。
<br/>　　“这四株圣药，一共需要多少灵石？”
<br/>　　楚风眠缓缓开口道。
<br/>　　“以皇品灵石来算的话。”
<br/>　　楚风眠一边开口，一边手心一动，从他的手心中，一枚皇品灵石便是拿了出来，直接递给了古家家主。
<br/>　　这皇品灵石，正是他所凝聚出来的，他倒是要想要看看，他所凝聚出的皇品灵石，现在到底是价值如何？
<br/>　　毕竟是从远古时代之后，就已经很少有武者可以凝聚灵石了，一是无法寻找到那么多的灵气，二是如今这个时代的强者数量，也是远远无法跟远古时代相提并论。
<br/>　　也只有楚风眠，拥有神脉这种得天独厚的优势，才可以大肆凝聚灵石，不过他也想要看看，他所凝聚出来的皇品灵石，成色如何。
<br/>　　“皇品灵石？”
<br/>　　古家家主接过这一块皇品灵石来，便是灵识扫了一下，脸色微微有些惊讶道。
<br/>　　“好精粹的灵石，在皇品灵石中，都算得上是成色不错的了，这种皇品灵石一般都是只有在一些古老宗门的宝库之中，才能存在，楚公子真的想要拿皇品灵石来结算？”
<br/>　　古家家主的惊讶也不无道理。
<br/>　　皇品灵石，就算是超越生死的老祖，都需要的东西，对生死境武者来说，更是极为珍贵之物，一般超越生死的老祖，都是习惯以王品灵石来交易。
<br/>　　只有那些老祖中都极为强横的人物，才有着极大的财富，以皇品灵石直接结算。
<br/>　　古家家主做生意千年来，也就碰到过几位这样的人物，没想到楚风眠居然有着如此恐怖的财富，一出手就是皇品灵石。
<br/>　　不过以古家家主的阅历，很快便是冷静了下来，一想到楚风眠从归墟天中的事，他的心中便是猜测到了一些。
<br/>　　归墟天是什么地方？整个九域的武者都知道，几乎是一个遍地宝藏的地方，楚风眠从归墟天中得到过远古的宝藏，也很正常。
<br/>　　古家家主缓缓的算了一下，这才开口道。
<br/>　　“这四株圣药，价值不一，总价值至少要在一百五十万皇品灵石左右，不过楚公子这皇品灵石其中的精粹程度，还要远远超越寻常皇品灵石，就给你算一百万皇品灵石如何？”
<br/>　　一百万皇品灵石。
<br/>　　换算成最为普通的灵石，便是足足价值一千亿，而且还并没有人会这样换取，一块皇品灵石，放在黑市上至少价值二十万灵石的价格。
<br/>　　这种价格，就算是对一些超越生死的老祖来说，都算得上是一个天文数字。
<br/>　　古家家主报出这个价格后，也是不由的看向楚风眠，他虽然知道楚风眠肯定是得到了一大笔财富，但是这价格，可着实不低了。
<br/>　　“一百万皇品灵石，这四株圣药，我要了。”
<br/>　　楚风眠没有任何迟疑，大手一挥，便是将那四株圣药，全部收入到了空戒之中，同时从楚风眠的面前，一座皇品灵石组成的山峰，也是正好飞了出来。
<br/>　　这山峰上，正是足足一百万皇品灵石。
<br/>　　这一百万皇品灵石，简直是一些一流势力，一些稍微小一点的商会都无法拿出来，不过楚风眠拿出来，却是轻轻松松。
<br/>　　这种圣药，一向都是拿财富都买不到，现在既然碰到，楚风眠自然不会放过，他现在有着神脉，可以源源不断的凝聚灵石，最不缺少的就是财富。
<br/>　　一百万皇品灵石对其他武者，甚至是对一些老祖来说拿出来都是极为肉疼，可是对楚风眠来说，却是轻轻松松，只要是能够收集到炼制玉露破命丹的圣药，再多的财富楚风眠都不会眨一下眼睛。
<br/>　　“一百万皇品灵石，果然是一百万皇品灵石！”
<br/>　　古家家主看到这如同一座山一般的皇品灵石，眼神都是有些难以置信，就连他都很少能够见到，如此之多的皇品灵石。
<br/>　　古家家主拿出一个空戒，打出一个漩涡来，这一百万皇品灵石，便是全部都被他收入到了空戒之中去。
<br/>　　“楚公子果然痛快。”
<br/>　　做成了这样一笔大生意，古家家主也是极为开心，这种生意，做成一笔对他们古家来说，也有着极大的好处。
<br/>　　“无妨，若是古家主，可以寻找到这剩下的圣药，我也可以花费重金购买，灵石不是问题。”
<br/>　　楚风眠笑了笑道，这话一出，简直是如同一个暴发户一般。
<br/>　　不过现在楚风眠的财富，可是比起暴发户还要可怕，毕竟暴发户的财富再多，也不能无穷无尽的挥霍，而楚风眠有着神脉，财富几乎是无限的。
<br/>　　他就算是随意挥霍干净，也可以在短短几日时间之内，在凝聚出更多的财富来。
<br/>　　“我古家的圣药，已经是没有了，不过宁州古城的大商会中，应该会有一些楚公子所需要的圣药。”
<br/>　　古家家主看着楚风眠，笑着开口道。
<br/>　　“还请楚公子在古家小住几日，我去跑一趟各大商会，来为楚公子寻找一下。”
<br/>　　“那好。”
<br/>　　楚风眠点了点头。
