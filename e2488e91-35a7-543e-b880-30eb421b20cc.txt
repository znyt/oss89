　　第672章
<br/>　　这个问题说大不大，说小也不算小。如今程大雷长住在蛤蟆城，不用像从前那般一天之内来回跑，可以在苏樱和樊梨花之间做出选择。
<br/>　　可一碗水总是要端平，程大雷也不好得罪她们其中任意一个。这无形中给程大雷带来很大压力，令他更不想去二人屋中。
<br/>　　程大雷想了想，最后决定去柳芷屋里。柳芷的身份是妾，不是妻，而她自觉身份，总是百般顺承程大雷，不会违抗他的意思。如此，自然能让程大雷轻松许多。
<br/>　　程大雷起身离开书房时，心中长叹一声，古人说妻不如妾，妾不如偷，当真是诚不我欺。
<br/>　　柳芷也没料得程大雷今夜会出现在自己屋里，既有些窃喜，也有些脸红。慌张得手脚都不知该放在何处。忙走过来将程大雷的外套接过去，又端来洗脚水，蹲着身为程大雷烫脚。
<br/>　　到此时，程大雷方才卸去一身疲惫，靠在墙上思考。如今的情况其实对己方不利，崔白玉的到来，证明相府会铁了心的支持宋伯康。如此，给自己带来更大的压力。
<br/>　　朔方城与蛤蟆城两座城池像是自己的两只手，护着一只就要牺牲另外一支，如果正面开战，首尾不能相顾，形势便愈发危险。
<br/>　　如此说来，直捣黄龙，主动攻击凉州城竟成了唯一的选择。可是军策府直到如今也没拿出来具体可行的法子，这令程大雷不禁有些急躁。这些人往事自诩聪明无双，难道遇到正经事时，一个个都变成了猪脑子么……
<br/>　　还有那号称白玉美人的崔白玉，她会不会真如传说中那般神机妙算……也可能是谣言以讹传讹，比如自己身边的徐神机，还有人说他是天下第一军师呢。
<br/>　　脑子里胡思乱想，竟不知何时阖上双目，沉沉睡去。
<br/>　　柳芷蹲着身服侍程大雷，心里七上八下，也是不知说什么才好。正这个时候，忽然耳边响起轻轻鼾声，抬起头来，程大雷竟不知不觉的睡熟了。
<br/>　　柳芷抿嘴笑了笑，也感觉有几分轻松。程大雷只说苏樱与樊梨花带给他很大压力，却不知他对柳芷来说，何尝不是一种压力。程大雷睡着了，她反而会轻松些。
<br/>　　程大雷的确累坏了，他承受怎样的压力，柳芷不懂，但多少也能看出来些。
<br/>　　她直起身，将程大雷放平在炕上，吹灭了灯，黑暗里悉悉索索脱掉衣服，挤进了程大雷怀中。
<br/>　　苏樱房中，苏樱也在床前翻书，她平日没什么消遣。就保管程大雷交给她的各种奇珍异宝，偶尔清点一番，借此打发时间。
<br/>　　小蝶气鼓鼓的走进屋里，故意搞出很大动静，借此引起苏樱的注意。
<br/>　　苏樱看着她，微笑道：“发生什么事了，把你气成这个样子。”
<br/>　　“都怪大当家，他竟去了柳芷房里，枉我百般为小姐说好话。”
<br/>　　苏樱摇头笑笑：“这是他的家，他自愿去那里是他的自由，你又不开心什么。”
<br/>　　“我不还不是为了小姐不服气么。”小蝶道：“难道小姐就没有一点不开心？”
<br/>　　苏樱摇摇头，没再接小蝶的话。程大雷往何处歇，这是小事，还不值得她烦恼。但她也的确察觉到，自己和程大雷的距离越来越远，相敬如宾，却难相濡以沫。
<br/>　　如此，倒也不算什么，诸多大户人家都是这么过来的。程大雷本有资格去荒诞，但他也的确蛮检点。苏樱本该知足，她也的确知足，可终究有几分不开心。
<br/>　　灯火飘忽，在苏樱额头浮起一抹愁容。
<br/>　　有些结，当真不好解。
<br/>　　次日天明，程大雷刚醒来便去了军策府，在路上碰到阿喜，阿喜带来一份极重要的情报。
<br/>　　来自京州的支援要抵达凉州，这次的支援主要是精良兵器，时间大概是三天后。
<br/>　　程大雷快步抵达城主府，表情已经极其不好看。这批兵器一旦抵达，便是宋伯康向自己开战的时候，可自己还没有准备好，去面对这一场战争。
<br/>　　可恼的是军策府，竟然现在还没拿出一个法子，这已令程大雷心情极不愉快。
<br/>　　吴用上前禀报：“大当家，要不要派人去劫了这批兵器？”
<br/>　　“劫，拿什么去劫，我们的兵力本来就不足，再分兵本就是大忌。或许，宋伯康便等着我们出兵，他已经摆好了圈套，要我们自投罗网。”
<br/>　　吴用沉默着，垂手立在程大雷面前。
<br/>　　程大雷白了他一眼，道：“直到现在，军策府还没拿出可行的法子，是不是所有事情都要我想，所有事情都要我去做？”
<br/>　　从程大雷口中说出这样的话，已经算是极重。吴用自从加入程大雷麾下后，的确也没有做过什么事，证明自己的价值。
<br/>　　他沉默半晌，道：“其实也不是没有法子，属下已经想到一个法子，只是需要冒一些险。”
<br/>　　“我是怕铤而走险的人么？”程大雷倾身问。
<br/>　　吴用的眼睛被点亮，他挥挥手，将屋子里的其他人赶出去。众人争吵议论，的确有可能产生正确。但有时候集体也是愚蠢的代名词，正确有时候往往握在少数人手中。
<br/>　　而且，这件事的确需要保密，事先泄密，是一件极危险的事情。
<br/>　　当人都走干净后，吴用摊开地图，手指指着凉州城。
<br/>　　“凉州城城高兵强，外面有一道护城河，若想破城，必须奇袭。时间要短，人数要少，连夜奔至凉州城，用火药炸开城门，大军杀进去。”
<br/>　　程大雷微微点头：“容易办到么？”
<br/>　　“不太容易。”吴用道：“现在宋伯康一定严加防范，不太容易靠近城墙，还有很多事情要做，调查何处容易突破，毕竟，这件事不容有失。”
<br/>　　“是呐，败了这一场，怕就没办法翻盘了。”程大雷道：“但如今已不能不冒犯，如果我真决定要做，你准备的时间要多长，什么时候可以动手？”
<br/>　　“那个，早就开始准备了，如果大当家要动手，今夜就能行动。”
<br/>　　“今夜么？”
<br/>　　“是呐，毕竟，军策府最近一直没在闲着，都在筹划这件事。”吴用道：“当然，是在保密的状态下。”
<br/>(本章完)
