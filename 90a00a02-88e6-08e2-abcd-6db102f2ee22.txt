        得知她和阿景“私奔”，他心痛大于气愤。
<br/>
<br/>        看到她再一次被阿景“抛弃”，他又高兴大于心痛。
<br/>
<br/>        从什么时候开始，他好像不在乎她的心里到底有没有他了，只在乎她还能不能留在他的身边。
<br/>
<br/>        错过她一次又一次，这一次，他不愿意再错过了。
<br/>
<br/>        童筱颖躺在傅夜沉的臂弯里，听着他沉匀的呼息声，睁着眼不停瞧着他的俊美睡颜。
<br/>
<br/>        她忍不住抬起手来，摸了摸他下巴上新生的胡渣。
<br/>
<br/>        三年的时光，让这个男人更有男人味了。
<br/>
<br/>        不得不说，傅夜沉是个很好看也很耐看的男人。
<br/>
<br/>        他刚毅俊朗，剑眉飞扬、鼻梁高挺、唇瓣细薄，无论他穿什么着装，都难掩与生俱来的贵气，整个人像是从电视剧里走出的混血男星。
<br/>
<br/>        忽然间，想到他西装外套内衣兜里的落以柔的那张寸照，童筱颖蓦然收回自己的手。
<br/>
<br/>        而此时，被他放在裤袋中的手机又“滋滋”地振动了起来。
<br/>
<br/>        童筱颖小心翼翼地从傅夜沉的裤袋里掏出他的手机，看到来电显示上是“落以柔”，于是替傅夜沉接听了电话。
<br/>
<br/>        “大哥，你人又去哪里呢？”落以柔声音温柔，透着一丝略撒娇性的抱怨。
<br/>
<br/>        童筱颖抬眸看了傅夜沉一眼，见他睡得很沉，于是捂着自己的嘴巴和手机话筒孔处，小声说道：“他在睡觉。”
<br/>
<br/>        “你是谁？”落以柔毫不客气地质问。
<br/>
<br/>        “童筱颖。”
<br/>
<br/>        “那麻烦夫人，叫傅总起床，好吗？”落以柔语气忽然严肃起来。
<br/>
<br/>        童筱颖再次看了傅夜沉一眼，感觉他似乎是真的很累，不忍心吵醒他，于是决定当一回红颜祸水的苏妲己。
<br/>
<br/>        “落秘书，傅总他今天的行程全部取消，麻烦你再重新安排一下。就这样，我挂电话了。”
<br/>
<br/>        “童筱颖！你知不知集团上上下下，多少人靠大哥吃饭？自从你来了后，大哥屡次上班迟到下班早退。因为你，大哥今天又要错过一个十几亿的合同。”
<br/>
<br/>        “那请问，大哥花钱聘请你是做什么的？”童筱颖霸气质问。
<br/>
<br/>        落以柔被她这一声质问给怔住，顿了一下，才回答道：“当然是辅佐他工作。”
<br/>
<br/>        “为什么我感觉，你不是在辅佐他工作，而是在干扰他的私生活？”
<br/>
<br/>        “你——”顿时，落以柔被童筱颖怼得哑口无言。
<br/>
<br/>        童筱颖二话不说，直接将电话给挂了。
<br/>
<br/>        她心里莫名其妙地生起一种“现任”大战“前任”的错觉。
<br/>
<br/>        这个女秘书，怎么这么多事？
<br/>
<br/>        童筱颖记得，傅夜沉身边好像还有一个特助叫郑忠威吧！
<br/>
<br/>        殊不知，今天一天的行程，特别重要的事情，傅夜沉昨晚就已经跟郑忠威交代清楚了，只是郑忠威并未告知落以柔而已。
<br/>
<br/>        而落以柔刚刚之所以打那个电话，纯粹是她的私心在作祟。
<br/>
<br/>        因为见不到傅夜沉，让她心里痒痒的有些难受，她只是想知道他此时此刻在做什么。
<br/>
<br/>        现在，她知道了，他在陪他的小娇妻睡觉。
<br/>
<br/>        那个童筱颖，还真是红颜祸水！
<br/>
<br/>        “阿嚏——”也不知道是谁在背后骂她，童筱颖打了个喷嚏，振得身子颤了颤。
<br/>
<br/>        傅夜沉在睡梦中，本能地将童筱颖抱紧，像是在给她取暖。
<br/>
<br/>        童筱颖睁着大大的眼睛，盯着傅夜沉，总感觉这男人是在把她当安抚娃娃使。
<br/>
<br/>        不知睡了多久，傅夜沉的手下意识地往身旁揽了揽，想把睡在他身边的童筱颖重新给搂入怀中。
<br/>
<br/>        但是，他的大手触及之处没有预期的温香玉软，有的只是冰凉的床榻。
<br/>
<br/>        筱颖！
<br/>
<br/>        傅夜沉睁开双眼，望向空无一人的室内后，微微皱起了好看的剑眉。
<br/>
<br/>        她又从他身边逃走了吗？
<br/>
<br/>        想到这，他的心口又在隐隐作痛。
<br/>
<br/>        从床上起来后，他四下打量了一下这里，直到门外传来敲敲打打的声音，他才好奇地走出去看了看。
<br/>
<br/>        门口左边的前坪里，只见童筱颖穿着绛紫色粗布麻衣和黑色布鞋，长发在后脑勺上盘成发髻，一脚着地，一脚踩在高椅上，然后一手拿着铁锤，一手捏着木钉，动作娴熟地在钉门板。
<br/>
<br/>        绛紫色衣袖挽到了臂弯处，露出了两条纤瘦而藕白的手臂，她每钉一颗木钉，都像是使了很大的力气。
<br/>
<br/>        这个小女人真的很不让他省心！
<br/>
<br/>        肚子里怀了宝宝，还架起长腿压迫腹部，使劲去做这些粗活。
<br/>
<br/>        她脑子里都在想些什么？
<br/>
<br/>        傅夜沉大步流星地走过去，二话不说，直接夺走了童筱颖手中的铁锤。
<br/>
<br/>        “诶？你抢我锤子做什么？”童筱颖直起身子，双手叉腰，抬眸看着傅夜沉抱怨道。
