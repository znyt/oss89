　　两人规避着尸兄，先后来到一处居民楼里，小胖提议，两人分开居住，这倒也正合刘哔心意。
<br/>　　早从小胖对那些丧尸的称呼中，他就搞明白这是一个什么世界了，是一部很久之前看过的片子，尸兄。
<br/>　　大概是一个生化类型的世界，因为距离当初观看的时间太过于久远，有很多剧情他都忘的差不多了。
<br/>　　不过，这个世界倒是有大量的尸兄可以猎杀，除此之外，也有一些特殊的物品，对他应该有些作用。
<br/>　　例如主角注射的那些强化剂，或者某个博士研制出来的那些药水。
<br/>　　这些东西对他来说，这些东西都是很有用处的。
<br/>　　和小胖分开之后，刘哔转身进了对门的房间，这个房间的主人早就变成了尸兄，钥匙被小胖顺手收集了起来。
<br/>　　本来小胖是想将这个房间当做一个物品储蓄间，现在他来了，正好也就入住进去了。
<br/>　　“砰！”
<br/>　　关上防盗门，刘哔一改先前诚恐诚惶的表情，整个人的气息变得凌厉起来。
<br/>　　之所以要装成这幅模样，是他想要借助小胖找到主角，这个世界的绝大多数剧情，都是根据主角展开的。
<br/>　　如果不跟随小胖的话，在这样的世界中想找到一个人十分困难，况且他本不是原住民，对于这里的一切都十分不熟悉。
<br/>　　甚至他连自己现在在哪里都不知道。
<br/>　　谨慎的检查了一下房间，没有摄像头，也没有任何监听设备，房间里……似乎没什么危险。
<br/>　　疑惑的向房门处看了一眼，他有些搞不懂，小胖难道真正的目的就是为了救人？
<br/>　　可他们之前分明都不认识啊。
<br/>　　真是一个单纯的人，可爱，但不可敬。
<br/>　　曾经他见过很多这样的人，甚至自己也是，但越到后来，这种人就越少了。
<br/>　　要么死了，要么变了，这种性格，怎么可能在末世存活的下去。
<br/>　　嘴角露出一抹嘲讽的笑容，他也懒得去提醒什么。
<br/>　　这样的性格，反倒是有助于他接下来的计划。
<br/>　　……
<br/>　　“哗啦啦……”
<br/>　　清澈的自来水从卫生间的水龙头内流出，但从刘哔的视角看去，很容易能分辨出其中沾染的一丝丝黑色气体。
<br/>　　这种东西，大概就是这个世界的丧尸病毒吧，但这些对常人来说足以致命的病毒，对他来说根本毫无影响。
<br/>　　他的体质注定会免疫所有的病毒，自然也就不会在乎这种东西。
<br/>　　简单的洗了个澡，刘哔穿着一条短裤，随手开启了卧室中的电脑。
<br/>　　深秋的气候有些寒冷，但他即使只穿一条短裤，也很难感受到一丝寒气。
<br/>　　网络还能用，想必这家主人也没有死去太长时间。
<br/>　　蓝色的开机界面亮起，让刘哔有些恍惚的感觉，他都快要忘记，自己上一次接触电脑是什么时间了。
<br/>　　不过，基本的操作大概也还都记得。
<br/>　　坐在电脑椅上，刘哔有些不熟练的点动鼠标，找出了一些有关这个世界的资料。
<br/>　　观察了一段时间之后，他也大致了解了一些，总的来说，这个世界和他原本的世界并没有太大区别。
<br/>　　背景，历史，甚至科技水平都是大致一样的，唯一的区别，就是这个世界的丧尸，似乎还保留着一些智慧以及语言能力。
<br/>　　这让刘哔有些难以理解，但也找不出任何原因。
<br/>　　除了这些东西之外，他还登录了某东，想寻找原著中主角第一次使用的那种强化药剂。
<br/>　　但遗憾的是，有关强化药剂之类的关键词，都被ZF给屏蔽了。
<br/>　　那这样看来的话，主角现在至少已经提取了强化药剂，他刚来这个世界，还没有参与任何剧情，想必这些东西还没有被更改。
<br/>　　了解完这个世界的基本信息之后，刘哔便关闭了电脑，对接下来的事情做了一些规划。
<br/>　　他可不想等小胖慢悠悠的找到白小飞，那样的话，他会错过很多东西。
<br/>　　得想个办法，让这两个人提前相见，但他现在手上的信息太少，时间线根本确认不下来。
<br/>　　躺在柔软的大床上，刘哔简单规划了一番后，进入了浅层修炼状态。
<br/>　　……
<br/>　　第二天醒来的时候，时间差不多是六点多种。
<br/>　　简单洗漱了一番，刘哔敲了敲小胖家的防盗门，随后，睡意朦胧的小胖打开了房门。
<br/>　　站在门口，小胖微眯着眼睛，有些不解的看着刘哔，不知道对方这么早来找他，是有什么重要的事情。
<br/>　　“恩人，我是来向您告别的。”
<br/>　　刻意忽略了小胖幽怨的眼神，刘哔咬了咬牙，做出一副为难的模样。
<br/>　　“哦，好，啊？什么？”
<br/>　　小胖掏了掏耳朵，话语从口罩内飘出，显得有些沉闷。
<br/>　　天知道这家伙为什么睡觉都要戴着一副口罩。
<br/>　　“末世爆发的时候，我哥哥正在住院，我有些担心，想要回去看看。”微微红着眼睛，刘哔语气有些伤感：“虽然我知道机会十分渺茫，但如果万一呢，不回去看看的话，我没办法原谅自己。”
<br/>　　闻言，小胖沉默了下来，他虽然不必担心家人，但也能理解刘哔的心情。
<br/>　　毕竟……他心中也有牵挂的人啊，否则的话，他又怎么能回来这里呢。
<br/>　　“但是这一路上十分危险，你……确定要这样做？”小胖犹豫了一下，看着刘哔有些泛红的眼睛，道。
<br/>　　他犹豫的是，自己要不要和刘哔一起过去。
<br/>　　毕竟这件事情和他没有什么关系，而且他也有自己的事情要做。
<br/>　　但从小养成的观念，又无法让他对这件事情置之不顾。
<br/>　　因为他知道，如果自己没有一起跟去，凭借刘哔的个人实力，是几乎不可能存活下来的。
<br/>　　“恩人，我一定要去，这件事，非常重要。”刘哔点了点头，眼神坚定：“我只是去看一看，小心一点的话，应该也不是什么问题……”
<br/>　　“那行，等我收拾收拾，一起去吧，这样路上也有个照应。”摆了摆手，小胖打断了刘哔的话语，此时已下定了决心，继续说道：“你要去什么地方，我们来好好规划一下。”
<br/>　　脸上露出一抹感动，刘哔推脱一番，‘无奈’接受了小胖跟随他一起同去的事实。
