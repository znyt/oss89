　　吴晓飞看着取证照片上的口红，张了张嘴，憋了半天也没说出一句话来。这么一支口红居然要1200块，这实在是太令吴晓飞难以想象了。
<br/>　　“开会期间，不要谈论其他事情。”
<br/>　　周宋捷瞪了吴晓飞和陆昕两个人一眼，又把第三张照片投放到投影仪上，说道：“照片上的人叫李霄力，在座的各位除了晓飞以外，应该对他都很熟悉，陆昕。”
<br/>　　周宋捷示意陆昕把李霄力的情况向会议室内所有人，再次介绍一遍。
<br/>　　陆昕在面前的笔记本电脑上敲打着，向会议室里的刑侦队员介绍道：“李霄力，男，南鲜国华裔，1985年3月29日生，‘碧海潮生’夜总会老板，除了‘碧海潮生’以外，在杭山市还有五家酒吧和三家健身中心。年轻、多金最重要还是未婚，据不完全统计，杭山市有近百分之三十的未婚女性把他当成理想的结婚对象。”
<br/>　　坐在陆昕身边的吴晓飞瞥了眼陆昕，想要问陆昕是不是也属于这百分之三十的未婚女性中的一个，可是想到现在是案情分析会议，周宋捷非常反感在案情分析会议上谈论跟案情无关的事情，所以话到了嘴边，吴晓飞最终还是没有说出口。
<br/>　　吴晓飞欲言又止的举动，并没有能瞒过周宋捷这个老警察的眼睛，吴晓飞对陆昕有想法，是整个刑侦队都知道的秘密。陆昕作为市局一枝花，对她有好感的可不在少数。尤其是刑侦队这种工作起来很多时候都是昼夜不分，以方便面和速溶咖啡作为夜晚伴侣，就连陪女朋友去看场电影都是一种奢侈的地方，能发展一段办公室恋情自然是最佳选择。
<br/>　　“对于这个李霄力，我们也是早就有所怀疑。”周宋捷把激光笔点在李霄力的照片上，说道：“我们不仅怀疑他跟这次钱鑫和陆岑的失踪案有关，我们还怀疑他跟之前几起人口失踪案有关，但是这个家伙很谨慎，他名下的产业同样也属于合法产业，手续齐全，我们尝试着申请过搜查令但是都被驳回了。除了人口失踪，这家伙很有可能跟境外制毒贩毒组织有关联，极有可能是境外制毒贩毒组织在我们国内的下线，我们也已经跟国际刑警组织进行联系，希望能得到他们的帮助，但目前为止还没有任何的消息。”
<br/>　　“这个家伙要么真的非常干净，要么就是他背后的人能量大到可怕，深不可测。站在我个人的立场，我绝对不相信这个李霄力是个背景干净的正当商人。所以，我们现在要做的，就是从这个李霄力身上撬开口子，这家伙绝对是条大鱼，如果能打开突破口，我相信能挖出一堆秘密来。”
<br/>　　“最近我们的人在‘碧海潮生’发现了一个新的情况。”
<br/>　　刑侦队副队长刘炯这个时候也从位子上站起身来，走到会议室正前方，拿出一个U盘连上电脑，点开一张照片，说道：“这段时间，每周三晚上八点，‘碧海潮生’都会进行一次拍卖会，拍卖的东西是什么呢？就是这个！”
<br/>　　刘炯用激光笔指着投影布上的照片，说道：“这东西叫‘Crystal-Ball’，也就是‘水晶珠’，是这段时间夜场最火爆的东西，而且还是以拍卖的形式限量供应，只有‘碧海潮生’的金卡会员才能进行拍卖，就这么一个小小的东西，起拍价格是2000块，而最终的成交价格都是过万，据说最贵的一次，这一粒东西卖到30000块！”
<br/>　　30000块！？
<br/>　　听到刘炯的话，坐在下面的刑侦队队员眼睛瞪得滚圆，一脸的不可思议，他们实在很难想象，这么一粒比糖丸还要小的东西，居然能卖到30000块！这价格都能在市中心最好的小区买下两个平米的房子了！
<br/>　　“经过多方了解调查，这个‘Crystal-Ball’是一种提升……”刘炯看了眼坐在下面的陆昕，犹豫了一下，最后还是继续说道：“男人那方面能力的保健品，效果奇佳，所以在夜场很受欢迎，但目前整个杭山市只有李霄力的‘碧海潮生’才有供应，而且还是限量拍卖，真正的有价无市。”
<br/>　　“果然男人都不是什么好东西！”
<br/>　　陆昕红着俏脸，小声嘀咕。
<br/>　　“这东西真的只是一种保健品？”
<br/>　　周宋捷盯着照片，眉头紧皱。出自一个老警察的直觉，周宋捷很敏锐的察觉到，这个东西恐怕绝对不只是保健品这么简单，不然不管这东西有多神奇，也不可能造成这么强烈的轰动。
<br/>　　根据刘炯所提供的，现场拍卖“水晶珠”时偷拍下的影像画面，那些参加拍卖的人完全就像是毒瘾发作时的样子。
<br/>　　“老周就是老周。”
<br/>　　刘炯轻笑着对坐在旁边的周宋捷点了点，说道：“这鼻子果然是比警犬还要灵，嗅到了其中的不寻常。我们也怀疑这种叫做‘水晶珠’的东西，可能是一种我们所未知的新型毒品，但事实并非如此……”
<br/>　　“‘Crystal-Ball’，由美国‘维伦恩’生物制药公司最新研制的一种男性保健品。”
<br/>　　在刘炯说话间，陆昕已经通过网络查到了关于“水晶珠”的新闻，“这是一种革命性的男性保健品，有别于以往的同类产品，并且已经通过相关检测得到了北美销售许可，但‘维伦恩’生物制药公司方面表示，由于目前产能不足的原因，可能会推迟‘Crystal-Ball’的上市时间。”
<br/>　　“这‘维伦恩’生物制药公司是什么来头？”周宋捷对陆昕问道。
<br/>　　“‘维伦恩’生物制药公司是一家总部在北美，研制和生产各类保健品的公司，总裁是霍斯曼德·J·维鲁斯。”陆昕回答道。
<br/>　　“怎么总觉着这家公司有问题呢？”周宋捷皱眉道。
<br/>　　“老周，如果李霄力真的跟这家‘维伦恩’生物制药公司有关系，并且这家‘维伦恩’生物制药公司本身也存在问题，那件事情就复杂了。”刘炯对周宋捷提醒道：“这不是我们可以插手的！”
<br/>　　“继续盯着‘碧海潮生’！”周宋捷吩咐道：“只要是狐狸，总有露出尾巴的时候，一旦被我揪住了他的尾巴，就算他娘的是妲己转世，老子也能打得他原形毕露！”
