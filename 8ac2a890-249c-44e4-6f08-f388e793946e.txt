　　神农踏遍千山万水，尝尽世间艰酸辛辣，只为寻得人族解决病魔之法。
<br/>　　数十年的经历，数十年的艰辛，数十年的坚持，数十年的痛苦，终于让他创出了自己的神功，《药神典》。
<br/>　　《药神典》初分为两部分，即药神部分与毒神部分。可以一步步通过食用各种药草或者毒药来修炼，药性或者毒性越强，实力提升自然也就越快。最后药、毒合一，成为绝世药神体，百毒不侵，万药不袭。
<br/>　　当然，这《药神典》如今也只是他草创的，还存在许多不合理之处，需要他不断完善。
<br/>　　就这样，又是数十年过去了，他承受的痛苦也越来越强，当然，他的修为也越来越强了。同样的，他对于各种草药的药理理解也十分详尽了。在神农不断记录、总结和研究之下，他对草木丹石，治病良方等方面的理解，有着深厚的造诣。
<br/>　　又是数年过去了，神农拖着满身腐烂的躯体，满脸疲惫地回到了人族部落，同时，他也带回了百草图。说是百草图，可实际上，上面所记载的药草，何止成百上千，成千上万，简直是数不胜数。上面所记载的草木之药，大到各种奇异圣药，小到随处可见的普通药草，可谓丰富至极。
<br/>　　回到人族后，神农便将自己关闭在一个小屋子内，开始创作，他要编撰一部医经药典。
<br/>　　三个月，不吃不喝，经过不断努力，他终于创作成功，他写出了《神农百草经》。
<br/>　　此经一成，霎时间，苍穹色变，无尽草木药石的虚影，浮现于天，飘香四溢，风云倒卷。道韵流转，道音嗡鸣，天空之上，好似有大德之辈，顿时唱起诵经之音，那正是《神农百草经》的内容，亦是普通人的治病良方。
<br/>　　“啊，成功了，共主成功编撰出《神农百草经》，如此异象必定非同凡响。我人族，以后普通人也可以有治疗身体疾病的方法了，哈哈哈……”
<br/>　　“太好了，自此以后，就不会因为疾病原因，死去那么多族人了，哈哈，真是天佑我人族，共主大德。”
<br/>　　“共主为我人族亲自尝毒草，这次回来，不知道苍老了多少倍，他几乎是在用命为我人族解决疾病问题，这《神农百草经》是共主用命换来的！”
<br/>　　“神农之德，惊天动地，神农之功，堪比日月，其为我人族之心，日月可昭，苍天可鉴！”
<br/>　　“……”
<br/>　　一时之间，有人在大笑，有人在哭，有人大喊，有人愣住了……人生百态，在这一刻尽显，无数人族纷纷朝着神农所在之地扣首跪拜，高呼“神农大德！”
<br/>　　“神农编撰《百草经》，比当初伏羲用河图、洛书领悟先天八卦也丝毫不弱，这动静，地皇证道不远矣！”巨大的动静，自然也惊动了无数洪荒大能，他们纷纷出声，讨论地皇神农氏。
<br/>　　“伏羲是当初的妖族大圣转世，不知道这神农是哪位先贤转世，竟有如此大德？”也有人想到了伏羲，怀疑神农的身世，他是否是哪位上古先贤转世？
<br/>　　“红云，是你吗？”
<br/>　　万寿山，五庄观内，镇元子喃喃自语道，他有些怀疑神农便是红云转世，因为刚才有那么一瞬，他感觉到了红云的气息，不过，那气息只是稍纵即逝，他也不能确定。
<br/>　　“红云道人？”
<br/>　　他不能确定，但元始却确定了，因为元始清晰的感受到了那千分之一瞬间内，出现了一股红云的气息，尽管那气息出现的时间很短暂，但还是被元始捕捉到了。
<br/>　　“当初红云被道祖算计，得到鸿蒙紫气，而后来也因此被无数人惦记上了，最后也因此被妖族与冥河围攻，导致自爆而死。红云自诩洪荒大好人，可笑，这洪荒可是吃人不吐骨头的世界，像他这样的人，能活下来才算奇怪。如今转世神农，希望他能够明白。”
<br/>　　元始呵呵一笑，便不再关注神农的身份了，管他是谁，反正对于元始都没有影响。对于元始来说，他的人生之中，只有两种人，一种是他感兴趣之人，另一种则是路人。显然，如今的神农之事让他稍微敢兴趣而已。
<br/>　　“神农，已经成功了？！”
<br/>　　玄都先是一愣，但紧接着，便是一喜，他这是在为神农高兴，因为只有他知道，神农为了这一天，吃了多少苦。
<br/>　　“嗡嗡嗡……”
<br/>　　而在同时，道韵依旧流转不断，九天之上，诵经声不停，无数草木虚影浮现，像是在阐述天地大道，一时之间，有无数大能因此悟道，也有无数人经过这《涉农百草经》，学会了药理。
<br/>　　如今的神农，便为天下师。他开辟了人族草木药理之道，后来人即便将他超越了，也无法否认他的地位，因为这就如同老子开创了道教，他便是道教的鼻祖，就算后人有人超越了他，也无法动摇老子道教鼻祖的地位。
<br/>　　“我等拜谢神农老师！”
<br/>　　这一刻，无数得到传道的人族朝着神农跪拜道。
<br/>　　“轰隆隆！”
<br/>　　神农身上的气势在不断增强，他那残破不堪的躯体，也在这一刻被一道道乳白色的光芒笼罩，之后以肉眼可见的速度，迅速恢复过来。他的身体之中，闪过黑白两道光芒，两者相互交织纠缠在一起，形成一道巨大的两色旋涡，如水涡一般，将四周不断涌来的灵气之潮尽数吞噬。
<br/>　　白色表示药力，黑色表示剧毒，两者相互交织，就如同阴阳相互转换一般，药生毒，毒生药，是药三分毒，毒亦可作药。
<br/>　　这一刻，神农的绝世药神体彻底大成了，那代表着他所创的《药神典》也彻底完善了。自此以后，神农便会百毒不侵，所有毒药，只能为他增长功力。他，神农，便是一代药神，同时也是一代毒神，更是一代药祖。
<br/>　　……
<br/>　　人族九黎部落，是当初人巫联盟之时生下的族人部落之一，其部落之中，便是身具人巫双重血脉之人，他们不仅具有元神之力，同时也具有强大的肉体。所以，这样的部落，都比较崇尚实力为尊，同样，他们也继承了巫族的好战因子，喜欢战斗。
<br/>　　在这一族中，有一族长，名为蚩尤，他是一位巫族之人，乃是当初三族大战所剩无几的大巫之一。自从那次大战之后，他便回到了人族部落，与人族混居。是有什么阴谋，还是其他原因？那就不得而知了。
<br/>　　蚩尤在部落之中，可是出了名的好战，因为战斗起来，形似疯魔，完全不顾及其他，所以被别人成为“魔神”！
<br/>　　此刻的蚩尤，看着无数人族朝拜神农，心中却是十分不服。因为之前百余年时间中，神农外出尝百草，人族之事交由他人管理，使得周围许多部落蠢蠢欲动，他蚩尤的九黎族便是其中之一。
<br/>　　此刻见神农回归人族，而且还创出《神农百草经》，他心中对此十分不爽。因为他心中已有决定，要征服人族，为巫族报仇，神农的回归，让他的计划有些不好影响。
<br/>　　不过，虽然神农回到了人族，但他心中的决定，不是那么容易改的。
<br/>　　神农只适合发展生产，将人族经济发展起来，而不适合统御人族。这是他最大的一个缺点，也是一个致命的缺点，正是因为这个缺点，导致许多人族部落人心不稳。
<br/>　　治理人族，可不能只用大德，以德服人，道德终究只是适合普通人。只有道德与武力并施，才是王道。而神农显然不知到这些，或者说，他知道，但他却做不到。正是如此，所以他只能是炎帝，而不能是黄帝。
<br/>　　其余各部落，此刻也有心思各异之辈，一些野心家不甘心屈居别人之下，既然神农可为为天下共主，那么自己为何不可？
<br/>　　一时之间，人族暗流涌动，福兮？祸兮？一切都未可知！
<br/>　　……
