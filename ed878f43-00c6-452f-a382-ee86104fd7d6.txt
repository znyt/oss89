<br/>
<br/>        “总裁，你还好吧，要不我留下来陪你吧？”严毅总觉得自家总裁这是极力隐忍着，他又要发脾气了。

<br/>        ……

<br/>        他坐在沙发里，抽出了一根香烟叼在薄唇上，“啪”一声点开了打火机，将跳跃的幽红火苗凑到自己的唇边点燃香烟，他开始吞云驾雾。
<br/>
<br/>        很快，茶几上的烟灰缸里就落满了烟蒂，他抽烟速度太快，一根接着一根的，一包烟眼见着都空了。
<br/>
<br/>        陆寒霆再伸手去拿烟，发现自己就带了一包烟，现在没烟抽了。
<br/>
<br/>        他胸腔里的那股暴躁嗜血的戾气就靠这点尼古丁的味道来麻痹了，现在手头摸不到东西，他将脑袋靠进沙发里，狭长的眼梢里全是猩红。
<br/>
<br/>        睁开眼闭上眼脑海里都是陆子羡进她房间的一幕，现在他们两个人在干些什么？
<br/>
<br/>        他早已经做好了心理准备，将来她会遇到更好的男孩子，投入他人的怀抱，但是这一刻真正来临的时候他竟然觉得这么难以忍受。
<br/>
<br/>        高大挺拔的身躯倏然站了起来，他健步走去打开了房间门，然后来到了夏夕绾的房门前。
<br/>
<br/>        这扇门还紧紧闭着，陆子羡还没有出来。
<br/>
<br/>        陆寒霆抬手就要敲门，但是他的手指突然在半空停了下来，就这么顿住了。
<br/>
<br/>        英俊的眼睑垂落下来，很快他转身就走了，回到了自己的房间里。

<br/>        他拿出了那瓶安眠药，打开了盖子，倒了一堆在掌心里，然后全部吞了下去。

<br/>        他已经离不开安眠药了。

<br/>

<br/>
<br/>        夏夕绾紧绷的神经一松，虽然她相信陆子羡的人品，但是孤男寡女的，他闯入自己的房间里还是吓了她一跳。
<br/>
<br/>        夏夕绾走上前，曼陀罗花依旧娇艳的鲜红欲滴。

<br/>        “你要这个干什么？”这时陆子羡突然问。

<br/>        “夕绾，你还没有放弃陆寒霆，对么？”
