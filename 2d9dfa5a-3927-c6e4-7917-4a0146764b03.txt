　　“放肆。”万剑一冷喝一声。
<br/>　　“何时轮到你这般诋毁夜修正统领？”
<br/>　　萧逸冷声道，“我更希望我是在诋毁，而不是事实。”
<br/>　　“诋毁也好，事实也罢。”夜修沉声道。
<br/>　　“总之，你现在必须退去，这是命令。”
<br/>　　话音落下，一阵浩然气势，凭空而现。
<br/>　　原本禁锢千雄教武者的剑势，瞬间被冲击。
<br/>　　“剑势？”萧逸眉头一皱。
<br/>　　夜修，乃是天元六重武者。
<br/>　　夜空剑之名，绝非浪得虚名。
<br/>　　自然也是拥有剑势的。
<br/>　　“若我不退呢？”萧逸站前一步，恐怖剑势，轰然落下。
<br/>　　两种剑势的碰撞。
<br/>　　瞬间让得千雄教内灵气暴走，高空风云大作。
<br/>　　当然，与萧逸凭借裂天剑魔10成剑道知识所调动的剑势相比。
<br/>　　夜修的剑势，无疑弱了许多。
<br/>　　夜修皱了皱眉头，“好霸道的剑势。”
<br/>　　“这就是传闻中裂天剑魔前辈的剑道手段吧。”
<br/>　　“萧逸副统领，我再说一遍。”
<br/>　　“立刻退去，否则，我们只能强行将你擒回据点。”
<br/>　　夜修话音落下，周遭数百炎武卫，纷纷亮出武器。
<br/>　　武器，清一色指着萧逸。
<br/>　　连原本的六位正队长，也同样如此。
<br/>　　炎武卫，令行禁止，在外执行任务，百分百服从上级。
<br/>　　萧逸脸色难看无比，咬了咬牙，一身气势，陡然消散。
<br/>　　剑势，也同时散去。
<br/>　　他自然不可能真的与周遭所有炎武卫开战。
<br/>　　握了握拳头，萧逸冷冷看向夜修。
<br/>　　“夜修统领，你最好真的是因为不想东边六郡大乱，而来阻我。”
<br/>　　“否则，若我发现你是另有目的。”
<br/>　　“为了包庇某些龌蹉之事而来。”
<br/>　　“我不会就此罢休。”
<br/>　　说罢，萧逸身影一闪，离开了千雄教。
<br/>　　……
<br/>　　夜修三位正统领的到来。
<br/>　　确实让萧逸感到惊讶。
<br/>　　当然，萧逸并无法确定，夜修三人是否知道千雄教的龌蹉事，是否真的在包庇千雄教。
<br/>　　如果是，事情就麻烦了。
<br/>　　而此刻的萧逸，正往星盟而去。
<br/>　　星盟的据点，本就在东来郡内。
<br/>　　不过数个时辰，萧逸便到了星盟，找到了徐星。
<br/>　　“阁下，啊不，萧逸副统领。”徐星见萧逸到来，满脸喜意。
<br/>　　“今日来我星盟，可是有什么事需要我们帮助？”
<br/>　　“若有，萧逸副统领尽管开口，我星盟上下，无不竭尽所能。”
<br/>　　萧逸点点头，道，“倒没有什么大事。”
<br/>　　“只是过来聊聊。”
<br/>　　进入星盟议事厅。
<br/>　　徐星命人奉上香茗，五位半步天元老者正襟危坐。
<br/>　　六人纷纷看向萧逸，等待着萧逸说话。
<br/>　　萧逸抿了口香茗，道，“我这次过来，同样是问星盟的事。”
<br/>　　徐星问道，“不知萧逸副统领具体想问什么？”
<br/>　　萧逸道，“所有。”
<br/>　　“所有？”徐星一愣。
<br/>　　“嗯。”萧逸点点头。
<br/>　　“从星盟创立开始，至今15年来的所有事。”
<br/>　　徐星皱眉道，“这可不是三言两语能说得完的事。”
<br/>　　“那就挑重点的说。”萧逸淡淡道。
<br/>　　“15年来所有大事。”
<br/>　　“额。”徐星愣了愣，随后点点头。
<br/>　　一阵沉默，徐星整理了一下脑海中的思绪。
<br/>　　半晌才开口娓娓道来。
<br/>　　数个时辰后，徐星说到了10年前。
<br/>　　萧逸一直没有出言，只是静静地听着，不时抿一口香茗。
<br/>　　起码至今为止，徐星说了上百件事。
<br/>　　却没有一件让萧逸得到有用的消息。
<br/>　　然而这时，一位半步天元老者却打断道，“慢着。”
<br/>　　“盟主，10年前还有一件大事。”
<br/>　　“哦？”徐星面露疑惑之色，“什么事？”
<br/>　　那位老者说道，“10年前，炎武卫曾经围剿过千雄教。”
<br/>　　“哦？”萧逸眼睛一亮。
<br/>　　徐星则面露惊疑，“有吗？我怎么不知此事？”
<br/>　　那老者摆摆手道，“那时盟主你才几几岁？还未当星盟盟主呢。”
<br/>　　“而此事，是炎武卫秘密行动，而且动作相当迅速。”
<br/>　　“当年，也就少数几个大势力，以及一些上层武者知道。”
<br/>　　“不过，此事后来忽然又不了了之。”
<br/>　　“怎么回事？”萧逸追问道。
<br/>　　“不知道。”那老者摇摇头。
<br/>　　“那次围剿，炎武卫出动得快，行动得快，但撤离得也快。”
<br/>　　“事后，还特地掩藏了此事，没有宣扬。”
<br/>　　“至于原因，谁也不清楚。”
<br/>　　萧逸皱了皱眉头，道，“算了，徐星继续说下去吧。”
<br/>　　徐星点点头，继续叙说这些年的事。
<br/>　　时间，渐渐过去。
<br/>　　一天后，徐星停下了话语。
<br/>　　15年来，发生在东边六郡的大事，都已说罢。
<br/>　　而萧逸，并没有得到他认为有用的信息。
<br/>　　这时，徐星问道，“萧逸副统领两次询问关于我星盟的事。”
<br/>　　“可是有什么不明之疑惑？”
<br/>　　“若有，大可告知在下，或许在下能直接替你解答。”
<br/>　　萧逸摇摇头，淡笑道，“并没有。”
<br/>　　“只是这一两天闲着，过来听听事，打发下时间罢了。”
<br/>　　“啊？”徐星一愣，脸部抽了抽。
<br/>　　五位老者，也是满头黑线。
<br/>　　“呵呵。”萧逸笑笑，“好了，我要走了。”
<br/>　　“哦？这么快吗？”徐星笑道，“本来还想多留萧逸副统领在星盟住几天的。”
<br/>　　萧逸淡淡道，“我准备离开东边六郡。”
<br/>　　“离开东边六郡？”徐星等人脸上的笑容，忽然定格。
<br/>　　那种莫名的戛然而止，让人不忍直视。
<br/>　　“萧逸副统领不是说要对付千雄教的吗？”徐星等人满脸急色。
<br/>　　“抱歉。”萧逸沉声道，“出了点意外，暂时无法对付他们。”
<br/>　　“但，血意丹之事，我会继续调查。”
<br/>　　“另外，徐星盟主，若是可以的话，过些天我可能要邀你前往一趟王都。”
<br/>　　“哦？”徐星急切的脸色中，带着疑惑。
<br/>　　“我现在还缺些证据。”萧逸淡淡道。
<br/>　　“等证据足够了，届时，徐星盟主来王都。”
<br/>　　“我亲自为你们东边六郡讨个公道。”
<br/>　　“当真？”徐星一喜。
<br/>　　“当真。”萧逸点点头。
<br/>　　说罢，萧逸转过身，准备离去。
<br/>　　然而，恰在此时，星盟之外，一道人影飞速袭来。
<br/>　　这是星盟的探子。
<br/>　　“慌什么？不知道有贵客在此吗？”徐星呵斥一声。
<br/>　　那探子连忙道，“盟主，出大事了。”
<br/>　　一旁五个老者怒声道，“说话利索点，什么叫盟主出大事了？”
<br/>　　那探子急道，“不是盟主出事了，是外头出事了？”
<br/>　　“什么事？”徐星问道。
<br/>　　探子快速禀报道，“千雄教遭遇黑魔殿袭击，宗门被毁，教内武者死伤惨重。”
<br/>　　“什么？”徐星一惊，“何时的事？”
<br/>　　探子回答道，“就在数个时辰前。”
<br/>　　“太好了。”徐星满脸喜意。
<br/>　　“黑魔殿，怎会无缘无故袭击千雄教？”萧逸眉头紧皱。
<br/>　　下一秒，萧逸身影一闪，御空飞离，往千雄教而去。
