　　在小惠的带领下，一行人来到一个高档小区内。
<br/>　　途中，几人还去了一趟便利店，收集了一些食物和淡水，为了长远打算，这些都是不可或缺的必需品。
<br/>　　地板上有一层黑灰，这是葫芦娃的骨灰，刘哔出门的时候，动用周围的火元素，将它的尸体烧毁了。
<br/>　　对于这并不明显的痕迹，没有人能在意到，甚至白小飞径直从上面踏了过去，都没有察觉。
<br/>　　都这种时候了，只不过是一层灰而已，又有谁会注意这种东西。
<br/>　　“咔嚓。”
<br/>　　防盗门上覆盖了一层薄灰，晓佳似乎很长时间没有回来过这里了，但从周遭整洁的环境看来，这里似乎没有什么问题。
<br/>　　几人陆续进了客厅，在检查完屋内没有任何问题之后，都是放松了下来。
<br/>　　“我和小惠妹妹先去洗个澡，希望还有干净的水可以用。”晓佳从冰箱里拿出几盒牛奶放在桌子上，笑吟吟的开口说道。
<br/>　　白小飞脸色一愣，有些不解：“可是自来水都被污染了啊，你们洗澡的话，会变成尸兄的。”
<br/>　　“我们家的太阳能刚好坏了，所以为了洗澡，特意买回来一个热水袋，里面储存的那些水，可没感染上病毒。”
<br/>　　给了一个放心的眼神，晓佳拉着小惠，两个女孩子向浴室中走去。
<br/>　　“小鹿也要洗澡，小鹿都好几天没洗澡了！”
<br/>　　听到有干净的水源，小鹿眼睛一亮，咿呀叫着，跟在晓佳和小惠后面。
<br/>　　没过多久，浴室传来了流水声，期间伴随着银铃般的娇小和嬉闹声，坐在客厅沙发上的几个男人，都有点尴尬。
<br/>　　都是血气方刚的年龄，一想到一墙之隔的距离，有两个浑身赤裸的女孩子正在洗澡，心中都不免有些躁动。
<br/>　　“佳佳姐姐，小鹿要喝奶奶~”
<br/>　　“咯咯，小鹿别闹，找你小惠姐姐去。”
<br/>　　“小惠姐姐~”
<br/>　　“别闹了，去找你晓佳姐姐。”
<br/>　　“姐姐，这是什么呀？”
<br/>　　“嗯……不要~讨厌啦！”
<br/>　　浴室中传出令人浮想联翩的声音，一抹金色的血液，从白小飞的鼻孔中缓缓流出。
<br/>　　端起水杯正欲喝水的刘哔一愣，小声提醒道：“白兄，你鼻子。”
<br/>　　“啊？哦……那个，可能是刚刚打出内伤了。”蹭了蹭鼻子，白小飞咳嗽了一声，有些尴尬。
<br/>　　轻笑一声，刘哔没有多说什么，而是拿起茶几上的遥控器，打开了电视。
<br/>　　“H市的疫情，已被控制，请市民尽量不要外出，关好门窗，等待军队的救援……”
<br/>　　新闻频道，主持人一本正经，报道着有关尸兄的最新消息。
<br/>　　现在病毒肆掠，ZF几乎将所有的精力都放在了这上面。
<br/>　　看过原著的刘哔，对此也知道一些事情，虽然同为生化危机，但这个世界的ZF，要比他那个世界中的给力多了。
<br/>　　不说那些ZF雪藏的异人，仅仅是拉开了几个档次的科技，都不是他原本世界可以相比的。
<br/>　　至少，在这样的环境下，ZF机构，还存有一个完整的体系，甚至给困在市里的群众，给出了相应的救援，这是十分难得的事情。
<br/>　　“真不知道，这场混乱多久才能平息。”
<br/>　　赤着上身，白小飞的语气有些感慨，他的上衣，在战斗中已经被撕碎了，无奈，现在也只能暂时这样了。
<br/>　　“谁知道呢，但情况总还算理想，至少，事情也在往好的方向发展，不是么？”
<br/>　　脸上带着微笑，刘哔看了看白小飞充满力量感的古铜色皮肤，黄金比例的肌肉，微微鼓动着。
<br/>　　这是强化过后的效果吧？
<br/>　　看起来倒是不错，只是不知道具体实力会是什么样子。
<br/>　　不过……连博士那种级别都应付不了，想必也不会厉害到哪里去。
<br/>　　“会好的！”一旁，一直闷闷不乐的小胖抬起头来，口罩上的笑容，配合他颓废的双眼，让人感觉有些怪异：“一切都会好起来的！”
<br/>　　话语落音，小胖的双眼恢复了一些神采，看样子，他已经逐渐从那件事情中走出来了。
<br/>　　“唉，你和你的同伴感情，好像很不错啊，一定认识许久了吧。”
<br/>　　刘哔轻叹一声，喝了一口白水，脸上的表情，也有些唏嘘。
<br/>　　“不，实际上，也只认识了两三天而已。”
<br/>　　小胖摇了摇头，语气微微有些颤抖。
<br/>　　“虽然只是三天，但可以看出，他是个很有责任，很有担当，又充满正义的人呐。”
<br/>　　“即使他很弱，可每次面对危险，都不曾有半点退缩，即使面对几乎不可能战胜的八头尸兄时，他都在冷静的寻找逃离办法。”
<br/>　　“最后，在我掉链子的时候，甚至还拖着已经昏迷的我，逃了出去。”
<br/>　　“如果舍弃掉我，像那样的人，应该不会死去吧。”
<br/>　　“他这样，是为了报恩吧。”
<br/>　　“可是，他的哥哥都还没有找到。”
<br/>　　“甚至当初，我自恃学过几年武术，放言要保护他，结果却是他保护了我。”
<br/>　　“我真的，对不起他啊！”
<br/>　　张了张口，白小飞似乎想要安慰一些什么，最终却叹了口气，只是伸手拍了拍小胖的肩膀。
<br/>　　认识了这么多年，他自然知道自己这位好友的性格，这样的事情，对他来说也着实是一个很大的打击。
<br/>　　但这种事情，只能让小胖自己走出来，别人很难做些什么。
<br/>　　“咔嚓。”
<br/>　　就在几人交谈的时候，浴室门突然打开，一阵带着沐浴露香味的白雾喷涌而出，小鹿光着身子，一扭一扭的跑了出来。
<br/>　　“小鹿洗好啦，大哥哥，叔叔，你们快去洗吧。”
<br/>　　看着光着屁股的小屁孩，几人都是微微一愣，停住了本要继续进行下去的话语。
<br/>　　“小鹿，你还没穿衣服啊！”
<br/>　　紧随其后的，是一个有些抓狂的声音，正起身准备倒水的刘哔动作一顿，目光下意识的向下看去，瞥见了两点嫣红。
<br/>　　“喂喂，你也没穿衣服啊！”
<br/>　　雾气缭绕的浴室门口，探出了晓佳小小的头颅，小惠惊叫一声，脸色羞红的跑了回去。
<br/>　　“哗啦啦~”
<br/>　　淡定的从饮水机旁接了一杯白水，刘哔脸色平静，在白小飞和小胖羡慕的目光中，走了回来。
