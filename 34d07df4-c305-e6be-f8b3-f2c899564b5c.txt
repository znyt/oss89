　　萧逸云虽然极度好奇，但也有点不敢问，怕预支了将来的某种意外的惊喜，又怕扰乱了现在的心情，于是先回去了。
<br/>　　叶少阳跟毛小方和道渊真人一起坐在亭子里，继续商量将来的计划。
<br/>　　“叶天师，原来你有这么一件要事要办，那我就不留你了，我自己去调查百鬼夜行。”毛小方说道。
<br/>　　叶少阳摆手道：“我跟你去吧，虽然我一心想着回去，但毕竟现在一时间也回不去不是，我也不能干等着吧，还是跟你一起，尽一下作为法师的责任吧。”
<br/>　　道渊真人询问他们要做什么，毛小方说了之后，问他要不要一起去。
<br/>　　“我最近犯了错，要在这里面壁……”道渊真人有些犹豫。
<br/>　　叶少阳这才想起这件事来，问他：“面壁多久？”
<br/>　　“一直到我认错。”
<br/>　　“那是多久？”
<br/>　　“我要是今天认错，今天就可以下山，但问题是，我觉得我没错。”道渊真人望着远处的崇山峻岭，眼神中透着一股子倔强。
<br/>　　叶少阳看了他一会，小心翼翼地问道：“你到底犯了什么事？”
<br/>　　“杀人了。”
<br/>　　叶少阳跟毛小方面面相觑。
<br/>　　杀人……是要偿命的，虽然这时代是乱世，但也不是乱到完全没有法律的地步。况且江西省在华夏腹地，除了边界在打仗之外，这一代还算是很平静的，杀人肯定是犯法的。
<br/>　　不等他们两个问，道渊真人说道：“前段时间我游历到外地，看到几个土匪抢了几个姑娘上山……一时忍不住，出手杀了一窝土匪，这件事被掌教知道，罚我在这里面壁思过，但我并不觉得自己做错了。”
<br/>　　叶少阳问道：“你杀了几个土匪？”
<br/>　　道渊真人想了一想说道：“就是一窝土匪，百来个吧。”
<br/>　　百来条人命！
<br/>　　叶少阳和毛小方面面相觑。
<br/>　　道渊真人眼中现出狠色，说道：“你是没看到那些土匪把那几个姑娘糟蹋成什么样子……我当时实在按捺不住，本来想杀几个，把别的吓走，结果杀了许多，收不住了。”
<br/>　　叶少阳深吸一口气，道：“土匪都有枪，你一个人怎么杀的？”
<br/>　　道渊真人笑道：“这个容易，我拘来一只恶鬼，让他帮我动的手，趁着那些土匪睡着了，一个个附身过去……”
<br/>　　叶少阳和毛小方更是无语。利用鬼魂杀人，这在法术界来说是大忌。
<br/>　　毛小方拍掌说道：“痛快！以杀止杀，道渊师兄，你这嫉恶如仇的性格，贫道佩服之极！”
<br/>　　道渊真人道：“我原也不想这么做。这时节，到处都是土匪，我一个法师，本不该管这档子事，但既然让我遇到了，便是按捺不住。唉，也不知道什么时候能到太平世界。”
<br/>　　突然想到什么，转头问叶少阳：“叶天师，你是从将来过来的，你说，这种乱世什么时候能够结束？”
<br/>　　叶少阳一下滞住，道：“我不敢说……”
<br/>　　对于大致的历史发展，叶少阳还是知道的，而今是北洋政府时期，军阀混战，之后还有二次革命……国民军北伐，之后就是漫长的抗日战争，然后是内战……光是从战争的角度来说，至少还有将近三十年的乱世。
<br/>　　毛小方和道渊真人都是有悟性的人，也不想知道太多天机，让叶少阳还是别说了。
<br/>　　“那我只能告诉你们，不超过三十年，一定能天下太平。”
<br/>　　两个人听叶少阳这么说，已经很高兴了。
<br/>　　道渊真人对叶少阳说道：“你跟我很熟是吧，我是说跟将来的那个我。”
<br/>　　“不能说很熟……毕竟你比我大了差不多一百岁……”
<br/>　　“那你对我将来的所作所为，是不是知道一些？”
<br/>　　叶少阳点点头，“你做了好几件震动法术界的大事，大家都知道的。”
<br/>　　“那就好。”道渊真人拍了拍他的肩膀。
<br/>　　“可是……”
<br/>　　“你误会了，我不是让你告诉我。”道渊真人目光十分坚定，“对于未来，我的确是很好奇，十分想要知道，但是你必须答应我，无论我将来怎么样央求你，千万不要告诉我，我将来都做了什么。让我对将来的事，抱有一份期待吧。任何事情，凭我的本心去做，我不想提前知道结果。”
<br/>　　叶少阳望定他，点了点头。
<br/>　　毛小方道：“还好你不知道我的人生，我没这方面担心。”
<br/>　　叶少阳笑道：“我在电视剧里看过。”
<br/>　　毛小方哈哈大笑：“那不可能是真的，不过你如果记得的话，倒是可以跟我说说的，我也很好奇，未来的人把我编排成什么样子。”
<br/>　　叶少阳问道渊真人：“那你就一直在这呆着了？”
<br/>　　道渊真人道：“坦白说，按照我犯下的罪，是要逐出师门的，不过掌教是我叔叔，念我有点慧根，不忍心让我流落，因此罚我在这里闭关思过，也算是保护我。”
<br/>　　叶少阳听他这么说，突然想起自己最初来到这个世界的时候，遇到的那个龙虎山的道士，当自己提到道渊真人的时候，表现出来一丝鄙夷，心中很好奇，道渊真人就算是犯了错，毕竟是张天师后人，在龙虎山算是有纯正血统的，再怎么着也不至于被一个外门弟子鄙夷，于是好奇地问道渊真人怎么回事。
<br/>　　道渊真人苦笑道：“我是张天师后人不错，不过我们张家一向人丁兴旺，子弟很多，为了让外姓弟子服气，一向都是通过各种考核，最后的胜出者才能担任掌教，我资质一般，是没什么希望的，加上从小就性格怪异，没人喜欢我，那些人对我不尊重，也是正常的了。”
<br/>　　叶少阳恍然，随即又望着他腰间的松纹古定剑说道：“那为什么这把宝剑会交给你保管，这应该也是一种荣耀吧？”
<br/>　　道渊真人苦笑起来，“你想错了，这把剑给我，偏偏是因为我资质不够好，因为我们历代都有一个规矩，就是执剑人要找子弟中最可靠的一个，而不能给掌教本人，这其中有些缘故，涉及到家族秘史，我就不方便讲了，反正每一代，挑选的都是年轻一代中最为老实的那一个……
<br/>(本章完)
