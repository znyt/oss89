　　在吞噬战魂进入巫尘体内之后，本来已经死亡的巫尘，其手指突然轻轻颤动了起来。
<br/>　　少倾之后，巫尘突然睁开了双眼，缓缓的坐了起来。
<br/>　　噗！
<br/>　　一口鲜血突然从他口中喷出，染红了身前的地面。
<br/>　　“这肉身损坏的也太严重了！”巫尘眉头微皱，口中低声自语。
<br/>　　当然，现在的巫尘已经不是巫尘了，而是变成了苏莫。
<br/>　　苏莫的战魂占据了巫尘的肉身，主导了对方的身体。
<br/>　　只不过，这具肉身损坏的太严重了，全身骨骼碎了一半，五脏六腑几乎都变成了肉泥，非常的凄惨。
<br/>　　这种肉身必须立刻治疗，不然只会变成一副臭皮囊，连战魂都无法寄存。
<br/>　　苏莫看了一眼不远处自己的肉身，取下自己手指上的储物戒，然后便将自己的肉身收进了储物戒中。
<br/>　　因为单纯的肉身，并没有任何生命气息，所以可以存放在储物戒内。
<br/>　　苏莫没有将自己的肉身存放在玉镯空间中，免得遭到巫娴三人的暗算。
<br/>　　随即，他在自己的储物戒中，以及巫尘的储物戒中，找出了很多瓶疗伤丹药，开始吞服丹药疗伤。
<br/>　　他的储物戒内，疗伤丹药奇多，都是以前的战利品，没有时间处理。
<br/>　　时间缓缓流逝，苏莫这一坐就是三日，三日的时间，他这具新肉身的伤势，恢复了不少，可以活动自如了。
<br/>　　不过，伤势依旧严重，短时间内根本难以发挥出多少实力。
<br/>　　而且，这肉身和苏莫的战魂还需要时间契合，两人修炼的功法体系也完全不同，就算恢复了伤势，也不可能发挥出这具肉身的巅峰战力。
<br/>　　苏莫没有在此地过多停留，努力控制体内的玄力，身形一展，向远处飞去。
<br/>　　没过多久，他便来了之前的那座青山，正是聂乾和韩盈藏身之地。
<br/>　　不过，苏莫用灵识查探了一番，却是并没有找到聂乾和韩盈，这两人藏的倒是隐秘。
<br/>　　“聂兄、韩姑娘，我是苏莫！”苏莫低声自喃，他知道，若是韩盈在附近的话，定然能听到他的声音。
<br/>　　嗖！嗖！
<br/>　　果不其然，少倾之后，破空声响起，韩盈和聂乾从不远处的山林中飞了出来。
<br/>　　“苏兄，这就是你夺舍的肉身？”
<br/>　　韩盈和聂乾飞到了苏莫的身边，韩盈上下打量了苏莫一番，不禁心中无语。
<br/>　　实在是苏莫现在这具肉身的模样太过凄惨了，不仅气息孱弱，浑身更是破破烂烂，身上干枯的血液几乎凝结成了铠甲。
<br/>　　“嗯！现在这具肉身伤势严重，我需要静养一段时间！”苏莫说道。
<br/>　　“苏兄，我们到大地之下，足够安全！”聂乾说道，此地距天冥圣城和九大卫城太近，难免会被人差距。
<br/>　　“好！”苏莫点了点头。
<br/>　　随即，三人身形一动，从一片荆棘丛生的草丛中向大地之下钻去。
<br/>　　三人一路下潜，直接向地心深处钻去。
<br/>　　十里！
<br/>　　百里！
<br/>　　千里！
<br/>　　万里！
<br/>　　十万里！
<br/>　　足足在地下潜行了十万里，三人才停了下来。
<br/>　　是苏莫有些坚持不住了，他的肉身伤势太重，地心之下不仅越来越热，还有强大的地底压力，他的肉身快承受不住了。
<br/>　　“两位，我们就到这里吧！“苏莫面色苍白如纸，声音孱弱的说道。
<br/>　　“好！苏兄，你尽快恢复这具肉身的伤势，然后恢复实力！”聂乾点头说道。
<br/>　　“苏兄，我们为你护法！”韩盈道。
<br/>　　“多谢两位了！”苏莫点了点头，便盘膝而坐，再次开始了疗伤。
<br/>　　接下来一段时间之中，苏莫完全沉浸在了疗伤之中，大量的疗伤丹药被他消耗。
<br/>　　时间如流水，转瞬间便过去了一个月。
<br/>　　一个月的时间，苏莫肉身的伤势恢复了七七八八，之所以能恢复的这么快，是因为他的战魂足够强大，对于肉身也有反哺的作用。
<br/>　　肉身恢复的差不多了，苏莫便结束了疗伤，换了一身干净的黑色长衫，而后仔细感受这具新的身体。
<br/>　　“太弱了！”仔细感受了一番，苏莫不禁心中暗叹，这具肉身无论是身体强度，还是体内的神丹，都远不如自己的肉身。
<br/>　　作为武帝境三重巅峰武者，这具肉身体内的神丹，蕴含的玄力浑厚程度，还不如他本体内的一枚神丹。
<br/>　　当然，毕竟是武帝境三重巅峰武者，此人神丹内的玄力，威力确实不弱。
<br/>　　单轮玄力的威力而言，差不多相当于他本体内两种属性玄力融合的威力。
<br/>　　但即便如此，这具肉身的实力，与苏莫本体的差距，也是非常的巨大。
<br/>　　总而言之，此人就是个普通的武帝境三重武者，与战力超强的天才，是完全沾不上边的。
<br/>　　“这是阴煞属性的玄力！”苏莫手掌一抬，手掌中涌出灰色的玄力，煞气冲天。
<br/>　　根据搜魂得来的记忆，这位叫巫尘的巫族人，修炼的功法名为《罗煞诀》，乃是帝级上品功法。
<br/>　　随即，苏莫闭上了眼眸，努力回想罗煞诀的功法要诀，以及巫尘记忆中的修炼经验，然后开始修炼起了《罗煞诀》。
<br/>　　经过这一个月的疗伤，他的战魂和这具肉身，已经逐渐契合，但还远远不够。
<br/>　　他要修炼这门功法，做到纯熟，才能发挥这具肉身的巅峰实力。
<br/>　　霎时间，苏莫全身煞气鼓荡，汹涌澎湃。
<br/>　　他微微催动了吞噬战魂，吸收大地精气与地煞之气，补充神丹内的消耗。
<br/>　　就这样，又过去了十日，苏莫已经完全适应了新的肉身，和新的修炼体系。
<br/>　　这个时候，他虽然不能发挥出这具肉身的巅峰战力，但已经能发挥出九成以上了。
<br/>　　而且，若是配合上他的吞噬战魂，以及大成的大苍穹剑法、本命灵剑阵，众多武魂等等，战力远超这具肉身巅峰之时。
<br/>　　随后，他不再浪费时间，在聂乾和韩盈的叮嘱之下，冲出了地底，向黑耀城飞去。
<br/>　　飞驰在天穹之上，望着远处渐渐清晰的黑耀城，苏莫眸中精光闪烁。
<br/>　　这一次，他要带走宏青璇！
<br/>　　这一次，他要巫天御死无葬身之地！
<br/>　　这一次，他还要巫族付出惨痛的代价，比苍穹世界试炼一役，还要惨痛十倍、百倍的代价。
<br/>　　不多时，苏莫便飞到了黑耀城，身形降落而下。
<br/>　　因为有巫尘的记忆，他对于黑耀城，简直是熟门熟路，仿佛是自己的家一般。
