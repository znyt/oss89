　　如果这转轮和尚是一尊弱者，这转轮和尚根本就没有资格跟小耀王寒夜，凰公子二人联合起来的资格。
<br/>　　像是如果楚风眠的实力，没有足以跟小耀王寒夜，凰公子平起平坐的层次，这二人只怕是就连算计都不会算计楚风眠。
<br/>　　在当时就出手将楚风眠直接轰杀了。
<br/>　　而且最为关键的一点，便是这转轮和尚，可是一尊佛门武者。
<br/>　　就是他刚刚，突然之间催动了佛寺之中的佛门禁制，给楚风眠带来的巨大的麻烦。
<br/>　　在这佛寺之中，残留的佛门禁制，佛门阵法，不知道有多少。
<br/>　　如果这转轮和尚，就连这些佛门禁制，佛门阵法，全部都足以控制的话。
<br/>　　那在这佛寺之中，此人对楚风眠的威胁，比起小耀王寒夜，凰公子两个人加起来，还要大得多。
<br/>　　楚风眠的目光，死死的盯着这转轮和尚，观察着他的一举一动。
<br/>　　楚风眠已经是将身躯之中的力量给催动到了极致，只要是这转轮和尚有着任何的举动，楚风眠就足以随时出手。
<br/>　　“杀人？我佛门中人，向来不滥杀无辜，我只要施主身上的一物，施主只要将万念佛珠交出来，你们两方的恩怨，我就不去插手。”
<br/>　　转轮和尚的目光盯着楚风眠右手手腕之上的佛珠，缓缓的开口道。
<br/>　　“这万念佛珠，本就是佛门之物，施主得到，不过是对其的亵渎，会被佛门惩戒，只有交出万念佛珠来，才能洗刷施主身上的罪孽。”
<br/>　　“洗刷罪孽？想要这万念佛珠？佛门之物？和尚，你不用装出一副道貌岸然的样子，佛门的秃驴，果然是跟记载的一样，一个个都是道貌岸然的小人。”
<br/>　　楚风眠听到这转轮和尚的话，冷笑一声。
<br/>　　这万念佛珠，是楚风眠身上唯一一件可以对抗佛门禁制的至宝，要是没有这万念佛珠，只怕是楚风眠都难以从这佛寺之中逃出去了。
<br/>　　交出这万念佛珠，这转轮和尚随意催动几道禁制的威力，就可以将楚风眠所轰杀。
<br/>　　想要让楚风眠交出万念佛珠，绝无可能。
<br/>　　“是吗？既然施主不识时务的话，那小僧只能是亲自动手了。”
<br/>　　转轮和尚听到楚风眠拒绝了他的话，眼神之中金光闪烁显现出了怒意，在他的背后，古老的佛影显现而出，佛门之力，倾泻而出。
<br/>　　转轮和尚的双手连连催动，突然之间佛门之力，在他的手心之中凝聚，化为了一道巨大无比的佛印，向着楚风眠轰击而来。
<br/>　　“真佛镇杀印！”
<br/>　　楚风眠看到这转轮和尚出手的一刻，一下子便是明白了过来，转轮和尚施展出的这一招，正是佛门之中赫赫有名的镇杀之术。
<br/>　　古老佛门之中，强者如云，许多强大的佛门之术，都被古籍记载下来，在九域之中流传。
<br/>　　这真佛镇杀印，就是其中之一，据说是一旦催动，足以轻易镇杀同境界的武者。
<br/>　　虽然不是仙术，却是媲美仙术，据说是这真佛镇杀印的创造者，便是一名佛门佛祖。
<br/>　　这一招的威力，几乎是足以跟九玄仙术的前三重所媲美，虽不如仙术，却已经是超越了寻常的武技范畴。
<br/>　　这一招在佛门之中，都是至高无上的杀招之一，楚风眠都没有想到这转轮和尚居然足以动用这一招。
<br/>　　看来这转轮和尚，在净土星域之中，都绝对是一位大人物，极有可能是媲美圣龙秘境圣子领袖这般的超级天才。
<br/>　　真佛镇杀印轰击下来的一刻，楚风眠感觉到他的身躯，都是在这一掌的力量之下，开始颤动不以。
<br/>　　一股股的灵力，已经是笼罩在了楚风眠的身躯之上，让楚风眠都难以调动灵力，来进行抵挡。
<br/>　　镇杀之术，先镇再杀！
<br/>　　佛印未落的一刻，楚风眠的身躯都承受了莫大的压力，要被这一股力量所镇压。
<br/>　　“剑出！九玄剑术！贪狼！巨门！禄存！天机！”
<br/>　　楚风眠大吼一声，祖龙至尊剑猛然出现在楚风眠的手心之中。
<br/>　　当楚风眠握住灵剑的一刻，冲天的剑意从楚风眠的身上爆发而出，镇压在楚风眠身上的无数佛光，被全部横扫毁灭。
<br/>　　天空之上，漫天星辰显现而出，星辰之力凝聚在楚风眠手心之中的剑锋之上，猛然一剑，冲天而起。
<br/>　　楚风眠毫无保留，刹那之间打出了九玄剑术的前四式，四式齐出。
<br/>　　耀眼的剑光闪烁，凝聚星辰之力，打出的一刻，顿时化为了星辰长河，无数剑气在其中凝聚，凌空之中跟那真佛镇杀印顿时碰撞在一起。
<br/>　　这真佛镇杀印的威力，几乎是足以媲美九玄仙术的前三式。
<br/>　　但是现在，楚风眠却是已经将九玄仙术的第四式天机，领悟出来。
<br/>　　这九玄仙术的第四式，已经是真正从武技之中脱离出来，踏入了仙术的范畴。
<br/>　　九玄剑术的前四式现在一齐施展出来，跟这真佛镇杀印碰撞的刹那之间，这真佛镇杀印的力量，被当即崩灭。
<br/>　　一招碰撞的力量，将这转轮和尚都给逼退了出去。
<br/>　　转轮和尚被逼退了出去，当即数道剑气，直接向着转轮和尚斩了过去。
<br/>　　只是这一道道的剑气，斩在这转轮和尚身躯之上的时候，从这转轮和尚的身上，突然散发出一道道琉璃色的光芒。
<br/>　　这转轮和尚的身躯，都转变这，化为了一种晶莹琉璃一般的感觉。
<br/>　　一道道剑气斩在这转轮和尚身躯之上，居然是无法撼动这转轮和尚的肉身，反而是这些剑气，都被反震的力量所震碎。
<br/>　　金刚琉璃身！
<br/>　　楚风眠看到这转轮和尚身上显现出的琉璃肉身，也看了出来，这正是佛门之中最为玄妙的锻体秘术之一，金刚琉璃身。
<br/>　　曾经楚风眠杀的一尊魔人傀儡之中，那魔人傀儡生前，就修炼了这金刚琉璃身。
<br/>　　但是那一尊魔人傀儡所修炼的金刚琉璃身，显然是远远无法跟这转轮和尚所媲美，这转轮和尚的金刚琉璃身，已经是显现出了琉璃之色，显然已经是要达到大成的地步。
<br/>　　根据传闻之中，这金刚琉璃身到达大成，几乎足以接近不朽真身的地步。
<br/>　　哪怕是现在这转轮和尚，还没有将金刚琉璃身给修炼到极致，这一道道威力不亚于九劫古帝全力一击的剑气，都难以打破这转轮和尚的肉身了。
