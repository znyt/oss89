　　没有在意到中年男子想要动手的态度，刘哔好整以暇的整了整衣服，轻笑一声。

<br/>　　故意停顿了一下，刘哔继续说道：“但是我这里有一件东西，想必先生应该会喜欢。”
<br/>　　目光微微一闪，中年男子没有回话，脸色却稍稍缓和了下来，他想听刘哔下面会说什么。
<br/>　　作为白日庭唯一的老师，他自然不缺学生，但像有马和宗太这样优秀的，却不知道要多少届才能出一个。
<br/>　　这并不常见。
<br/>　　况且宗太的身份，也十分敏感，虽然宗太的死亡和他无关，但还是对他造成了很大的影响。
<br/>　　这非常麻烦。
<br/>　　他心中已经打定了主意，如果刘哔的理由不能说服他，那他一定会动用最残忍的酷刑来对刘哔进行折磨。
<br/>　　那种痛楚，没人能承受的了。
<br/>　　“如果我没猜错，房间里这些孩子，都不是纯种的人类吧。”
<br/>　　脸上带着自信的微笑，刘哔没有在意中年男子变的阴沉的脸色，继续开口：“据我所知，半人类虽然有着远超常人类的强大体能，但同样有一个致命的缺点，就是身体老化的速度要比正常人快了很多。”
<br/>　　“通俗来讲，就是短命，我说的没错吧。”
<br/>　　停下了口中的话语，刘哔和中年男子对视着，目光中带着一丝莫名的意味。
<br/>　　中年男子脸色更加阴沉，但对刘哔的话语，并没有开口否认。
<br/>　　半人类寿命很短，这在CCG高层并不是一个秘密，很多人都知道。
<br/>　　没什么好否认的。
<br/>　　“这个问题，我有办法解决。”
<br/>　　随着刘哔的话语落下，中年男子的脸色瞬变，双目如电，死死地盯着刘哔。
<br/>　　这是困扰了白日庭许久的问题，即使花费了无数资源研究，也没取得太好的进展。
<br/>　　毕竟，半人类实在是太过于强大了，像正常人类的身体素质，根本没有办法承受这股力量。
<br/>　　喰种之所以可以承受，是因为其强大的再生能力，以及可以容纳大部分能量的赫包，这两样特性，半人类都不拥有。
<br/>　　可以说，像他们在这类人，很难说明白到底是幸运还是不幸，上天给了他们强大的赫眼，却没能给他们容纳这股力量的东西。
<br/>　　英年早逝，这是所有半人类都逃脱不了的命运。
<br/>　　被别人艳羡的同时，他们何尝不在羡慕其他人，能够轻轻松松的活着。
<br/>　　但是现在，刘哔却说可以解决他们这个致命的缺点。
<br/>　　他有怎能不激动。
<br/>　　虽然，自己面对的是一个喰种，虽然，这件消息并不一定属实。
<br/>　　但，如果再不尝试一下的话，他就快要死了啊。
<br/>　　相对其它半人类来说，他活了太久。
<br/>　　他见证了自己一个个学生的死亡。
<br/>　　但他对此无能为力，就连他，也是靠着特殊的药物才能存活下来。
<br/>　　哪些药物，虽然给他延续了生命，却也在蚕食着他的力量，如果一直这样下去，用不了多长时间，他就要变成一个行将就木的普通人了。
<br/>　　他不甘心。
<br/>　　所以他选择相信。
<br/>　　事实上，他也没得选择。
<br/>　　“需要我为你做什么？”看了一眼身后激动非凡的学生，中年男子低沉着声音，语气有些无奈，他拒绝不了这个条件。
<br/>　　说到底，除了那被诅咒的力量之外，他们也是再普通不过的人类了。
<br/>　　只是被命运所迫，长期处于这样一个压抑的环境，周围都是一样的人，所以才会养成这样的性格。
<br/>　　谁生来都不想做面瘫。
<br/>　　“那么，我们现在可以谈一谈了吧，你好，我叫刘哔，是个喰种”
<br/>　　听到中年男子的回答，刘哔伸出了有些苍白的右手，他知道，眼前的这个男人，妥协了。
<br/>　　脸色稍稍犹豫了一下，中年男子叹了口气，和他握了握手：“教皇，白日庭总负责人，CCG特聘特种搜查官培养教官。”
<br/>　　嘴角勾起一抹笑容，刘哔跟随教皇，一起来到别墅二楼的书房中。
<br/>　　……
<br/>　　看着坐在眼前的男人，刘哔眼中泛起一抹沉思，原著中，他对教皇这号人物没有一点印象。
<br/>　　但不可否认的是，这个男人的气息无比强大，就连他，也有些承受不住这股如同山岳一般的压迫感。
<br/>　　不过，倒也不太让人感到意外，毕竟是培养出有马贵将和旧多二福的教官，他的实力，自然高的可怕。
<br/>　　身为一个半人类，能存活这么长时间，也着实让人感到意外了。
<br/>　　刘哔也感受到了他身上那股垂死的气息，如同病重的狮虎一般，虽有余威，却没了那不可阻挡的威势。
<br/>　　或许，是原著中这个教官死于天命，所以才没有在世人面前展露出来吧。
<br/>　　这件事情，也让他提起了警惕，像这样未从原著中出现过的高手，这个世界还不知道隐藏着多少。
<br/>　　如果不小心一些，说不定哪天，就会在阴沟里翻了船。
<br/>　　SSS级，明面上是位面尖端的力量层次，但谁知道实际上会是怎样呢。

<br/>　　但教皇明显不是，否则，又怎么会和一个被CCG通缉了的喰种坐在一起。
