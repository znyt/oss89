　　手指轻轻敲击着膝盖，刘哔暂时将这件事情放在了后面，他心中已经做好了决定，如果什么都不能做，那么即使熟知剧情又有什么用处呢。
<br/>　　任婷婷款款走到桌前，和几人打了一声招呼后，便坐在了任老爷的旁边，一旁的侍应看人已经来齐，忙不迭的走上前来，递上了菜单。
<br/>　　“想喝什么随便点啊，不要和我客气。”任发拿起烟斗嘬了一口，面带笑容的说道。
<br/>　　“我要Coffee。”任婷婷接过菜单，随手放置一旁，她经常出入这种场所，肯定不会感觉陌生。
<br/>　　可是九叔和文才就有些尴尬了，两人都是第一次来这种咖啡厅，对于这种情况不知道该如何应对，更令人蛋疼的是，为了提高档次，餐厅菜单上面都是洋文。
<br/>　　毕竟来他们咖啡厅进行消费的人群都比较洋气，用洋文菜单的话也不会有太大的影响，可是，这就苦了九叔和文才两人了。
<br/>　　看着菜单上如同蝌蚪一般的洋文，九叔有些茫然，在他眼中，这些洋文比道符还要晦涩。
<br/>　　此时文才和任发两人也都已经点完了，文才这个家伙倒是鸡贼，反正他也不懂，就索性跟着任发，点一样的东西就好了。
<br/>　　“你好，请给我和我师父来一杯卡布奇诺。”看到九叔眼中的尴尬，刘哔轻笑一声，侧身向侍应吩咐道。
<br/>　　在末世爆发之前，他平日里也非常喜欢咖啡，据他所知，卡布奇诺是20年代初期，也就是一九零几年的时候研制出来的，按照时间来算，这时候应该已经流入了华夏。
<br/>　　这也是刘哔想要趁此机会，来证实他心中的一个猜测，这个想法也会对他接下来的计划产生一些影响。
<br/>　　“好的先生，您请稍等。”果不其然，在听完刘哔的吩咐之后，侍应并未露出什么异常的反应，当四人全部点餐完成，将菜单收起，恭敬的退了下去。
<br/>　　那这样的话，就没有什么问题了，现在这个世界的时间线，应该是在民国初期，一九二几年这段时间。
<br/>　　按照原本的历史走向，在这段时间里，华夏常有战乱发生，战事多了，自然死人也不会少到哪里去，死人多了，各种妖魔鬼怪也就多了。
<br/>　　这样的世界观，也给刘哔带来了很多好处，要知道积分获得的途径，除了正常完成一些任务，还可以通过猎杀能源体来获得。
<br/>　　那些在寻常人眼中看起来恐怖无比的僵尸鬼怪，或者一些小有法力的能人异士，对他来说却都是一个个行走的积分，只要他有猎杀这些能源体的能力，他的实力就会进入一个飞速提升的阶段。
<br/>　　端起桌子上的咖啡喝了一口，刘哔心中逐渐形成了一个计划，出于对自己家人的担心，他现在迫切的想要提升自己的实力，然后找到离开这个世界的办法。
<br/>　　“你喜欢喝纯咖啡啊？”正在刘哔想的入神的时候，耳边突然传来了一阵黄鹂般的声音，任婷婷手中端着杯子，好奇的看着刘哔。
<br/>　　父亲和九叔正在谈关于爷爷起棺迁葬的事情，她有些无聊，便主动开口，想要和刘哔攀谈。
<br/>　　她平时对于一些洋人的东西都颇感兴趣，也是最近才喜欢上咖啡这种新鲜玩意，但是对于这东西还没有太多的了解，碰到精通此道的刘哔，也有着讨教的想法。
<br/>　　“恩，咖啡还是纯的有味道，虽然苦涩，但也正宗。”看了一眼任婷婷，刘哔放下手中的茶杯，微笑着回应道。
<br/>　　胃部又是一阵抽搐，喉咙间涌上一阵令人想要呕吐的恶心感，刘哔心中苦笑一声，他只是想要实验一下，果然还是不行，自从变成丧尸之后，就连咖啡这种饮品都变得无法入口了。
<br/>　　若有所思的点了点头，任婷婷看了一眼分毫未动的咖啡杯，犹豫了一下，没有像往常一般增加砂糖和奶精，而是直接尝试着喝了一小口。
<br/>　　“哇，好苦啊。”只是尝了一小口，任婷婷便放下了杯子，整张小脸都皱在了一起，那股苦到极致的感受，让她实在无法接受。
<br/>　　看到任婷婷的表情，刘哔嘴角一抽，他之前只不过是随口一说，要知道这可是最原始的黑咖啡，几乎没有任何修饰，如果不加一些配料的话，很少会有人能够接受这种味道。
<br/>　　他之所以那样喝，也是想要试试黑咖啡的苦涩，能不能将那股极为恶心的味道给遮掩住。
<br/>　　“虽然入口的味道不很适口，但只要撑过那股苦涩的味道，你就会品尝出其中醇厚的咖啡香味。”淡淡的笑了一声，刘哔指尖在咖啡碟边缘盘旋着，随口说道。
<br/>　　偏了偏头，任婷婷看向刘哔的眼神中有些好奇，不知道是不是错觉，她总是感觉刘哔身上有着一股和年轻外表完全不搭边的成熟和沧桑。
<br/>　　看到刘哔似乎没有继续交谈的意思，任婷婷也止住了话头，两个人坐在各自的座位上，有些尴尬，而一旁的九叔和任老爷却是交谈甚欢。
<br/>　　“师父，弟子想出去一下，购置一些物品。”感觉在咖啡厅呆的没有什么意思，刘哔附在九叔耳旁，言语了一声。
<br/>　　初来乍到，他对这个世界的了解还是一片空白，他想出去转转，没准还会有一些意外的收获。
<br/>　　“恩，早点回家，可能还有事情要做。”九叔摆了摆手，叮嘱了一句，随后继续和任发交谈了起来。
<br/>　　应了一声，刘哔和任老爷道了别，转身走出了咖啡厅，快要接近午饭时间了，外面大街上还是有些热闹的，只是正午有些毒辣的太阳微微影响到了他的心情。
<br/>　　说来也很奇怪，这个世界的天气实在诡异，明明是快要入秋的季节，却还有这么毒的太阳，早知道出来的时候就不要穿长袖了，也不至于像现在一样这么热。
<br/>　　“喂，正好我要去买一些胭脂水粉，不如我们一起吧？”正在考虑要去哪里的时候，刘哔肩膀突然被人拍了一下，转头一看，是任婷婷清秀的脸庞。
