<br/>    那些身体残破的食人魔蚁尸体，被复活的食人魔蚁扔进了活着的食人魔蚁群中。<br/>　　<br/>　　    李振邦一开始还有些疑惑，这到底是什么意思，不过很快他就看到了那些尸体发生一阵阵的爆炸，将尸体周围的食人魔蚁炸伤。<br/>　　<br/>　　    幽魂的技能并不只有复活，还有尸爆，如果幽魂的等阶提高到六阶以上，它还会使用亡灵召唤术。<br/>　　<br/>　　    守城的人自然看到了米尔森林边缘发生的混乱战斗，他们虽然不知道为什么会这样，但是这对他们来说是绝对的好事。趁着这会儿压力减轻的时候，将食人魔蚁全部从城墙上击退了。<br/>　　<br/>　　    食人魔蚁这会儿放弃了进攻城墙，全部反身朝着复活的食人魔蚁发动了进攻。<br/>　　<br/>　　    复活的食人魔蚁毕竟只是少数，活着的食人魔蚁付出了一定的代价之后，将复活的食人魔蚁全部分尸了。复活的食人魔蚁死亡以后发生的尸爆要比普通的食人魔蚁威力大很多。<br/>　　<br/>　　    在食人魔蚁反身攻击的时候，媪妖和月光被李振邦收入了后天袋中。他可不想冒险，万一它们两个出问题了，自己后悔都来不及。<br/>　　<br/>　　    李振邦和幽魂本身就离战场比较远，此刻都躲藏在一棵巨大的古树树冠之中。李振邦将自身的气息全部收敛，和周围的环境融为一体，幽魂本身就是幽灵，自然没有什么气息外泄。<br/>　　<br/>　　    食人魔蚁并没有发现什么异常，只好反身准备继续进攻翠屏镇。城墙上的人刚松了一口气，还以为食人魔蚁内乱能让他们休息一下，结果就看到食人魔蚁又铺天盖地的爬了回来。<br/>　　<br/>　　    “主人，有些不对劲儿。”这一次幽魂并没有说话，而是通过契约用精神与李振邦进行沟通。<br/>　　<br/>　　    “怎么了？”李振邦皱了皱眉头，现在的情况已经非常糟糕了，他可不希望再听到什么更坏的消息了。<br/>　　<br/>　　    “城墙下死亡的食人魔蚁和城墙上死去的人的灵魂似乎都有些不对劲儿。”幽魂犹豫了一下说道，它毕竟离的太远，还不太敢确定。<br/>　　<br/>　　    “有什么问题吗？”听到幽魂的话，李振邦心底没来由的突突跳了一下，这种感觉很不美妙，往往意味着有什么事情将会发生。<br/>　　<br/>　　    “正常情况下，不论是人还是魔兽，死亡以后灵魂都会升入空中或者渗入地下，再或者四处游荡。可是城墙附近的灵魂都停留在原地，甚至有聚合在一起的倾向，而且我能感觉到他们有很大的怨气，这种情况很不正常。”幽魂思索了一下，谨慎的说道。<br/>　　<br/>　　    “还有其他的异常吗？”听到这里，李振邦反而冷静了下来，他虽然对灵魂没有什么研究，但是他对魔法元素也异常的敏感。<br/>　　<br/>　　    听到幽魂的话，他尝试着将精神力凝聚成一条线朝着城墙方向靠近，他明显感觉到了很多灰色的魔法元素。<br/>　　<br/>　　    平时空气中这种魔法元素很稀少，而且很容易就消散掉，可是现在却很多，而且很活跃。<br/>　　<br/>　　    幽魂在使用魔法时身边就凝聚着这一类的魔法元素，如果自己所料不错的话，灰色的魔法元素应该就是亡灵类魔法元素。<br/>　　<br/>　　    李振邦略一沉吟，想到了一个可怕的事情。有亡灵魔法师混进了翠屏镇，而这一切应该都是亡灵魔法师策划的。他为了收集大量的人类和魔兽的灵魂，偷袭了食人魔蚁的巢穴，带走了蚁后。<br/>　　<br/>　　    虽然这仅仅是一个构想，但是李振邦觉得已经非常贴近现实了。亡灵魔法师，那可是整个大陆的公敌！如果不是没有办法，他也不会让幽魂使用亡灵魔法帮忙，他可不想别人误以为他和亡灵魔法师有什么关系。<br/>　　<br/>　　    李振邦犹豫了一下，还是将自己的想法告诉了欧米伽，毕竟亡灵魔法师可不是易于的角色。<br/>　　<br/>　　    亡灵魔法是比黑暗魔法还要邪恶的法术，黑暗魔法是可以通过修炼成长的，而每一名强大的亡灵魔法师的背后都意味着曾经有无数人被残害，因为亡灵魔法师的成长需要的是大量的灵魂和尸体。<br/>　　<br/>　　    刚才的战斗中，魔法师基本已经耗尽了魔力，脸色惨白，扶着城墙才能勉强站立。不过没有一名魔法师选择退后，除了受重伤的魔法师被百姓们抬下城墙救治以外，其余的魔法师都趁着短暂的休息时间吃下恢复药剂打坐恢复起来。<br/>　　<br/>　　    城墙上的战士们也人人带伤，疲惫不堪，露在外面的胳膊上青筋缭绕，一些人已经出现了脱离的征兆。<br/>　　<br/>　　    年轻力壮的普通壮年百姓也自愿的爬上了城墙，准备做最后的殊死搏斗，他们也清楚，如果城破，他们以及家人也将无法幸免。<br/>　　<br/>
