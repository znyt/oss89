　　白起随即冷笑一声，“没想到救我的却是你，当日我有这一切，确是拜你所赐！”
<br/>　　道风心中也是感慨天道无常，想到自己那时候跟随叶少阳下墓，斩杀白起，当时如果不是机缘巧合，白起连这一抹元神都没机会留存下来，这才过了多久，自己就亲自闯昭狱，拼死再把他的元神救出来……
<br/>　　天道无常，由此可见一斑。
<br/>　　如果斩却三尸，证了混元大道，不知道对于天机，会不会能看透的更多一点，眼下来说，道风觉得自己还是太弱小了，有一种在冥冥之中随波逐流的感觉。
<br/>　　虽然世上绝大多数生灵都是这么过下去的，但是道风不愿意这样。
<br/>　　“道风子，这一饮一啄，岂非天命，你今日救我出来，我们往日的一切，自当一笔勾销了。你无需担心。”白起现在只剩下元神了，但说话还是那么霸气，仍然把自己当成了昔日的杀神。
<br/>　　道风笑道：“就算你仍然是杀神，我又有什么好担心的。”
<br/>　　白起听了，冷笑一声说道：“道风子，你的确也有资格说这话，我早就知道你是谁了……”
<br/>　　徐福道：“住口，此事暂时不提！”
<br/>　　白起听他这么说，也就不说什么了。
<br/>　　道风问徐福，“下一步，你有什么打算？”
<br/>　　徐福道：“我先离开这个时空，寻找机会，将他魂魄复原。”
<br/>　　道风沉吟了一下，道：“元神跟精魄是两码事，想让元神重生出魂魄，比登天还难。”
<br/>　　徐福道：“我自然是有办法的，这件事，还需要你那师弟帮忙。”
<br/>　　“关少阳什么事？”道风眉头一紧。
<br/>　　“这个，且容我日后再说。我近日游荡人间，发现法术公会在人间蠢蠢欲动，意图重现百年前之事……如今人间再无年轻的强者，照这么下去，人间法师，或会失去在人间的主导权。”
<br/>　　道风沉吟道：“这件事，我也有听说，只是我无人间身份，人间法术界对我的印象，比对法术公会还差，我没有办法强出头。”
<br/>　　白起虽然只能出现在镜中世界，不过元神还是完好的，听见他们这么说，大声吼道：“这帮冒充轩辕氏的家伙，为何总是死性不改！徐二哥，快些复活我的魂魄和肉身，让我去屠杀一场！”
<br/>　　道风和徐福都没有理会。
<br/>　　道风想了一下，说道：“这件事，将来再议，我想问你一件事，我小师弟在民国，可是一切都在你掌控之中？”
<br/>　　他最关心的，还是叶少阳的安危。
<br/>　　徐福面露难色，说道：“只有一件事，不在掌控之中。”
<br/>　　道风探寻地望着他。
<br/>　　徐福道：“当日，我用山河印收叶少阳一干人的时候，有一个人意外闯了进去，现在也去了民国。”
<br/>　　“李浩然。”道风立刻说出这个名字。
<br/>　　徐福点了点头，说道：“他才是那个时空中，对叶少阳威胁最大的人，他还没找到叶少阳，一旦找到，必然不会轻易放过他。”
<br/>　　道风想了一下，说道：“他对叶少阳，似乎并没有多深的仇，没有道理杀他。”
<br/>　　“青牛不想杀叶少阳，他只想回来这里。”
<br/>　　“那你为什么不把他送回来？”
<br/>　　徐福嘴角露出一丝不易觉察的微笑，说道：“那个时代，是被法术公会掌控的时代，青牛目下在哪里的所作所为，得罪了法术公会，这样一来，能够分担叶少阳的压力……他迟早也会对上法术公会，有了李浩然，能为叶少阳牵制不少压力。为了通过叶少阳找到我，他不会杀叶少阳，也不会让别人杀他。”
<br/>　　道风盯着徐福看了一会，说道：“青牛，可以对抗法术公会？”
<br/>　　徐福笑了笑，说道：“你在轮回中磨去了记忆，不知道这青牛的来历。我却是在人间呆了两千年，对于他的历史，我是知道一些的。说起来，他也是我一位老朋友了。”
<br/>　　道风道：“我只知道，他是当初老子化胡、出函谷关时候乘坐的坐骑，不知真假。”
<br/>　　“当然是真的，不过你只知其一，不知其二，青牛得道之后，曾显圣三百年，开坛讲道，门生遍布天下，就算是吕纯阳、张果，也都曾听他讲道，也算是他的弟子，吕纯阳……”他看了道风一眼，想到什么，道，“吕纯阳就不说了，就说张果，作为青牛的弟子，都斩三尸证道了，他却一直没有证道，你道为何？”
<br/>　　道风道：“我不喜欢卖关子，有什么就说。”
<br/>　　徐福被他呛了一下，也不在意，说道：“青牛早就有能力证混元大道，不过他在道心之中，有一事不明，便是因果……若是别人，这根本不算什么，那些证得大道的仙人，谁敢说自己道心中没有困惑之处？不过，青牛这人很是执拗，为了弄清轮回大道，不惜自身踏入轮回，就此轮回千年，以解决道心之惑……”
<br/>　　道风听到这里，忍不住问道：“为了一个执念，轮回千年？”
<br/>　　徐福点头，说道：“别人肯定不会这么做，但青牛就是做了，虽然他每一世轮回，都与佛、道有缘，而且灵根极佳，加上没有喝孟婆汤，百世记忆完全保留，因此修行速度惊为天人，每一世都是绝顶强者。不过，他的肉身毕竟限制了他的实力，而且法力也需要重新积累，因此，才没有强到超过道法之外，你才有机会斩去他一只手……但是，你要知道，这只是他一世的法力，他有百世积累的对道法的理解，真正的万法皆通，一旦堪破大道，实力之强，必然惊世骇俗。他若真想堪破大道，随时都可以做到。”
<br/>　　道风说道：“我很好奇，他百世轮回，什么时候是个头？”
<br/>　　“他当年立下宏愿，什么时候，在某一世靠着当世的修为证得混元大道，便是圆满。”
<br/>　　建文帝站在道风身后，一直都没有开口，听到这里，也是实在忍不住了，问道：“这不是舍近求远吗，他既然千年前就得道了，何必这么费事。”
