　　第154章
<br/>　　当是出门没看黄历，程大雷带一堆人出来，也就是浑水摸鱼装装样子，就当带大家野炊好了。
<br/>　　怎么就那么倒霉，偏偏在此地遇到熊老大，还吃了人家的坐骑。
<br/>　　程大雷手下都是一帮老弱，战斗力几乎为零的渣渣，而秦蛮也最多和熊老大打个平手，但熊老大身边还有几个亡命徒。
<br/>　　此刻熊老大血灌双瞳，拎着双斧向程大雷追来。
<br/>　　“呔！”程大雷忽然起身，冷笑道：“来得好，我在此地等你好久了！”
<br/>　　吱嘎！
<br/>　　熊老大像刹车一般生生停住，目光紧张的盯着左右，心道：难道有埋伏！
<br/>　　程大雷拍拍身上的雪粉，目光向周围转了一圈，随之笑道：“我奉幽州王之令，在此地恭候许久。来，上次你我未分胜负，这次与我战个三百回合，论个雌雄！”
<br/>　　熊老大注意到程大雷观察周围的目光，心道：是啦，此地必有他们的埋伏，他们在此大摇大摆生火造饭，就是为了引我过来。
<br/>　　“狗贼，我不像你一样，习惯做杨龙停的走狗，今日我便劈了你这狗贼！”
<br/>　　程大雷握紧剑柄，心道：难道没有唬住这人，若真打起来，必是一场血战。
<br/>　　突然，只见熊老大大吼一声，转过身带着手下没命逃窜。
<br/>　　“呔，那里走，与我大战三百回合！”
<br/>　　“哈，姓程的，我不会中你埋伏的，哈哈！”熊老大便跑便喊。
<br/>　　程大雷送了口气，冲手下人摆摆手：“赶紧撤，赶紧撤。”
<br/>　　程大雷都有几分佩服自己的急智，不然打起来，即便自己和秦蛮可以逃生，像小黑老张这样的人却是活不过今天。
<br/>　　……
<br/>　　见身后没有追兵追来，跑得气喘吁吁的熊老大才停下脚步。幸亏自己反应过，不然和程大雷缠斗起来，必然陷入他们的圈套之中，然后被一网打尽。
<br/>　　“大哥，咱们往那里走！”手下问。
<br/>　　“向东走！”熊老大道：“他们以为咱们是向南走的，怎料到我们中途折向，却向东走了，其必然找不到我们。”
<br/>　　“老大英明！”
<br/>　　“哈，弟兄们等躲过这阵，咱们占个山头依旧大块吃肉大口喝酒。我知道怎么做山贼，给我三年时间，我还是这里的老大。”
<br/>　　熊老大带着手下向东走，一路小心隐藏自己的脚印，忽听得前方有脚步声。熊老大停住脚步，就见程大雷带着人从对面树林中走过来。
<br/>　　隔着五十步，双方目光相对，都是一样的错愕。
<br/>　　“就这么倒霉么！”程大雷。
<br/>　　“就这么倒霉么！”熊老大。
<br/>　　“呔！”程大雷面带微笑：“想不到吧，我就知道你会从这里走！”
<br/>　　“姓程的，我和你没完！”
<br/>　　熊老大带着手下掉头就跑。
<br/>　　程大雷拍拍胸口，冲身边的人吼道：“刚才谁带的路，到底谁带的路！”
<br/>　　秦蛮有些尴尬：“要不大当家你说怎么走？”
<br/>　　程大雷看看周围：“咱们先出山，找到官道就回山寨，这里真的太危险了。”
<br/>　　“会不会再碰到熊老大？”秦蛮。
<br/>　　“应该不会那么倒霉的。”
<br/>　　程大雷心里有算计，熊老大为躲避追兵，肯定是往山林里钻，而自己是向山外走，双方背道而驰，是绝对碰不上的。
<br/>　　“山林里不知埋伏了多少伏兵，咱们在山里已经不安全了。今天就出山，找户人家夺些吃的。”
<br/>　　程大雷带人转过一个山坡，忽然听见说话声，就看到熊老大从另一个方向出来了。
<br/>　　双方目光相对，面面相觑。
<br/>　　“……”程大雷。
<br/>　　“……”熊老大。
<br/>　　“这个你相信缘分么？”程大雷试探着问。
<br/>　　熊老大双目喷火，大吼一声：“跑呀！”
<br/>　　“老大，老大……”手下气喘吁吁道：“实在跑不动了。”
<br/>　　连续碰到三次，熊老大带着手下就狂奔了三次。不仅仅是手下，就连熊老大此刻也是筋疲力尽。
<br/>　　“老大，和他们拼了，不过一死而已。”
<br/>　　程大雷就等着熊老大跑呢，自己也好趁机会开溜。可就见熊老大这次竟然转过身来，恶狠狠的盯着自己。
<br/>　　“你怎么不跑了？”
<br/>　　“你怎么不追？”
<br/>　　“……”程大雷要怎么说，难道说哥们，我其实一直避着你走的。
<br/>　　“我已经猜到了，此地其实并没有伏兵，否则我定然不会一个都没遇见。你这狗贼虚张声势，定是想要耗尽我等体力，混蛋，我竟中了你这狗贼的奸计。”
<br/>　　“……”程大雷。
<br/>　　程大雷冷笑一声，锵地拔出佩剑，口中道：“既然已知道中计，还不速来送死，更待何时！”
<br/>　　“狗贼，我和你拼命！”熊老大拎着斧头冲过来。
<br/>　　一道铁枪横挡接住了他，秦蛮吼道：“大当家，战就战了，和他废话这么多做什么！”
<br/>　　“退后！退后！”程大雷呵斥身后老弱：“谁也不许上前！”
<br/>　　秦蛮接住熊老大，而熊老大的十几名喽啰，已经向程大雷扑过来，看来他们也明白擒贼先擒王的道理。
<br/>　　程大雷不能让手下老弱上前送死，只能以寡敌多，撑到秦蛮解决熊老大后，再来相助自己。
<br/>　　这是唯一的胜机，且希望渺茫。毕竟自己连斧头都没带，身上只有一柄剑，若有斧头都话，还可以虚张声势一下。
<br/>　　不，也不是没有办法，自己还有一招剑法，那招准传说级的剑法——阿飞快剑。
<br/>　　名曰：直来直往。
<br/>　　想到此处已经有人迫进程大雷，大刀向程大雷砍来。程大雷是后退的姿势，根本不适合出剑，可鬼使神差间，手腕已抬起，在一个绝对不适合出剑的状态下，刺出一剑。
<br/>　　那一瞬间，脑海中似乎出现一个舞剑的少年，心神合一，冥冥中似乎有人抬着程大雷的手刺出这一剑。
<br/>　　好快的剑！
<br/>　　一剑穿喉，血流如注。
<br/>　　程大雷没有半点升级为高手后的得意，身体只有一个感觉，疼！
<br/>　　手臂的骨头像裂开般，扎心扎肺。程大雷明悟，以自己目前的实力，还无法完美驾驭这一剑。
<br/>　　如果能完美驾驭，则其更快。
<br/>　　现在的自己，最多刺出七剑。
<br/>　　那么自己就是七剑之内的高手，简称七剑高手。
<br/>　　似乎七剑也就够了。
<br/>　　程大雷直起身子，剑指群贼，风将他鬓前长发吹起。
<br/>　　“你们可听过一种剑法，唤做……直来直往！”
<br/>
