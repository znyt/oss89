　　第七重神殿之中，苏莫心中微微叹了口气。
<br/>　　当初千渊殿殿主夏千渊说的不错，他的武道，已经比较失衡了。
<br/>　　太过依赖元神和血脉，暂时来看，以前并无什么大的问题。
<br/>　　但是，修为想要达到真神，需要某一种奥义修到极致，参悟到法则的力量。
<br/>　　所以，他想要达到真神之间，还是需要参悟奥义。
<br/>　　再者，来到这时空神殿之中，看到这时空奥义，苏莫算是明白了。
<br/>　　真正的强者，不是你血脉有多强大，也不是你元神有多强大，而是你掌控的法则有多强大。
<br/>　　试问，所谓的时空至尊，掌控时空法则，简直就是无敌的存在。
<br/>　　一个强者，若是没有自己的道，血脉再强，元神再强，面对这恐怖的时空法则，恐怕都难以应对。
<br/>　　苏莫再次的叹了口气，看来以后，他要主修奥义和法则了。
<br/>　　当然，这并不是说修为不提升，修炼法则大道，也是在修为的基础之上。
<br/>　　少倾，苏莫不再多想，意沉体内，沟通吞噬元神，感知吞噬之力，参悟吞噬奥义。
<br/>　　他已经很久没有参悟吞噬奥义了，如今修为飚升，神魂壮大，吞噬元神威力暴涨，参悟起来速度极快。
<br/>　　时间不断的流逝，很快便又过去了两个月。
<br/>　　大殿之中，骆妤掌控了殿中的时间规则，其次是沁儿，最后是褚彦。
<br/>　　“骆妤姐，褚彦哥，我们要不要去第八重神殿？”沁儿大眼睛闪亮，充满了跃跃欲试之色。
<br/>　　“沁儿，第七重神殿已经是我们的极限了，去第八重危险太大了！”骆妤摇了摇头，剩下的时间已经不多了，她们若是去第八重神殿，无法掌控其内的时间规则，将永远出不去了。
<br/>　　“是啊，在这里修炼到结束吧！时空之城三年只能进入一次。”褚彦道。
<br/>　　“我们出来数年了，去了很多地方，也该回人王宫了！”骆妤郑重的说道，她们若是再不回去，宫中怕是要派人来寻了。
<br/>　　她和褚彦倒是没有什么，关键是沁儿身份特殊，不能在外时间太久。
<br/>　　万一有个闪失，她和褚彦麻烦就大了。
<br/>　　“没意思！”沁儿闻言，不由的瞥了瞥嘴，显得很是郁闷。
<br/>　　好不容易出来一趟，并没有见识到什么有意思的事情。
<br/>　　“骆妤姐、褚彦哥，听说永恒神殿有很多厉害的天才，不如我们去讨教一番如何？”沁儿又问道。
<br/>　　“沁儿，我们必须回去了！”骆妤面色郑重的说道。
<br/>　　“那好吧！”沁儿见此，无奈的叹了口气，虽然她身份不凡，但是，却也要听骆妤和褚彦的。
<br/>　　出来之前，她父亲可是说了，一切都要听从骆妤和褚彦的安排。
<br/>　　若是不然，回去之后，难免要受到责罚。
<br/>　　“沁儿，你修为已达一品中级真神，天赋逆天，还是尽快修炼到高级才是正途！”褚彦微笑着说道。
<br/>　　“知道了！”
<br/>　　沁儿无奈的叹了口气，随即他眼角余光，瞥见了不远处的苏莫。
<br/>　　“呀，这人还没掌控这里的时间法则啊？”沁儿问道。
<br/>　　“沁儿，此人不太可能掌控第七重时光神殿的法则，必然会死在此地！”褚彦瞥了苏莫一眼，不在意的说道。
<br/>　　“那我们要不要救他？”沁儿秀美微微一皱。
<br/>　　“救他？”褚彦闻言，微微摇了摇头，道：“我们救不了他，在时空之城中，一切只能靠自己！”
<br/>　　“沁儿，你不要管他了，他只是一个无名的小人物而已！”骆妤叮嘱道。
<br/>　　“好吧！”
<br/>　　沁儿点了点头，随即，她便闭上了眼眸，开始修炼。
<br/>　　……
<br/>　　修炼之中，苏莫微微睁开了眼眸，瞥了一眼褚彦三人。
<br/>　　三人的谈话，他都听到了，那沁儿倒是挺热心。
<br/>　　至于另外两人，则有些冷漠了。
<br/>　　当然，这并不是说两人坏，这只是一般人正常的情况。
<br/>　　让苏莫意外的是，这三人貌似是什么人王宫的人？
<br/>　　人王宫，是六大王宫之一吗？
<br/>　　之前，三人说过天王宫和地王宫，那么，人王宫显然也是六宫之一了。
<br/>　　苏莫思索了少许，没有过多思索，便继续参悟奥义。
<br/>　　时间飞逝，他的吞噬奥义，进展极快，两年时间，便达到了七阶大圆满。
<br/>　　三年多的时间，达到了八阶的初期。
<br/>　　“时间差不多要到了，我们走吧！”
<br/>　　“快走，不能错过了时间！”
<br/>　　“提前出去吧！”
<br/>　　这时，第七重神殿之中，很多人纷纷起身，离开了神殿。
<br/>　　这些人均是留在此地，依靠时间流速，来修炼的。
<br/>　　九日时间一到，就要立刻离开时空之城，不然会死。
<br/>　　“沁儿，骆妤，我们也走吧！”褚彦长身而起，向沁儿和骆妤道。
<br/>　　两人也结束了修炼，随即，三人离开了神殿。
<br/>　　临走之时，沁儿看了苏莫一眼，微微的叹了口气。
<br/>　　这人，怕是要出不去了！
<br/>　　很快，大殿之中的人，便走的差不多了，只剩下寥寥几人。
<br/>　　剩下的这些人，或神情慌张，或面色阴沉，显然是无法掌控时间规则。
<br/>　　苏莫睁开了眼眸，长身而起，也大步走了出去。
<br/>　　在此地待了九日，他的修为从三变到四变，吞噬奥义更是达到了八阶初期，算是收获极大。
<br/>　　那么，接下来，就是去轮回神朝了。
<br/>　　至于这时空神殿，他短时间不会在进来了。
<br/>　　因为，据说每一次开启，规则都不相同，要重头再来。
<br/>　　而且，好像是有限制，每一个人，每三年才能进入一次。
<br/>　　不过，苏莫心中决定，过些年要再来一次。
<br/>　　这时空神殿之中，有大机缘、大造化。
<br/>　　很快，苏莫走出了第一重神殿，来到了时门的出口处。
<br/>　　“咦！你居然出来了？”沁儿看到苏莫，显得很是惊讶。
<br/>　　而褚彦和骆妤两人，也是微微惊讶，这个段惊天，貌似还不简单。
<br/>　　看来，定然是在最后的时刻，掌控了第七重神殿的时间规则。
<br/>　　“多谢姑娘挂念了！”苏莫向对方微微抱拳。
<br/>　　“谁挂念你了？我只是有些意外而已！”沁儿闻言，顿时小嘴一撇。
<br/>　　“姑娘，你们是要回轮回神朝吗？”苏莫微笑着向沁儿问道。
<br/>　　之前，他听到了三人的对话，知晓三人要回人王宫。
<br/>　　“是啊，难道你也要去！”沁儿点头。
<br/>　　“正是，不如我们结伴同行如何？”苏莫微笑着说道。
<br/>　　他准备以这三人为切入点，进入人王宫之中。
<br/>　　当然，具体情况，还要到了轮回神朝之后，再做决定。
<br/>　　“可以啊！我并不介意！”沁儿轻笑一声道。
<br/>　　而此时此刻，褚彦和骆妤两人，听闻苏莫和沁儿之言，对视一眼，不禁纷纷脸色微沉。
<br/>　　就在此时，时门再次打开了，所有人纷纷涌了出去。
<br/>　　苏莫和沁儿等人，也立刻随着人潮，飞了出去。
