　　这‘苏’字雕刻在石凳上，字迹歪曲，痕迹崭新，明显是刚刚用指尖刻画而出。
<br/>　　“被认出来了？”苏莫望着这字迹，脸色不禁一沉。
<br/>　　那恒飞肯定是认出了他的身份，不然不可能在石凳上留下一个苏字。这让苏莫心一沉，那恒飞他好像从未见过吧，怎么会被认出来？
<br/>　　当然，他也知道，见过他的人数之不尽，比如在真圣界、在大荣城，数以千万计的武者见过他，他也不可能全部都记住。
<br/>　　苏莫心中沉吟，对方发现了他的身份，居然来提醒他，而且没有要挟他，看来倒没有什么敌意。
<br/>　　不过，连一个武帝境八重的武者，都能识破他的身份，这让苏莫颇感无奈。
<br/>　　只可惜，他并没有什么高深的隐瞒之术，也就是改变了面貌，收敛了气息而已。
<br/>　　这种简单的隐藏之法，身份实在是太容易被识破了！
<br/>　　“不管了，要尽快炼制分身，然后离开古灵星河！”
<br/>　　少倾之后，苏莫叹了口气，连一名武帝境八重的武者都能识破他的身份，以后还指不定有多少人识破呢！
<br/>　　到时候，他活着的消息人尽皆知，巫族大祭司再次杀来，则必死无疑了。
<br/>　　他必须要尽快离开了，万界山的任务，也只能让分身去想办法完成了。
<br/>　　至于聂乾和韩盈，很快也会被古天意救出来，他已经没有什么好担心的了。
<br/>　　这一刻，苏莫心中做了坚定的决定，必须尽快离开了，一刻都不能耽误。
<br/>　　他不能被束缚在皇祖圣朝，不然只会自寻死路。
<br/>　　少倾之后，苏莫大步的走出了石殿，向赖羽所在的石殿走去。
<br/>　　二十九小队所在的一组石殿，有一座较大，便是队长所住的地方。
<br/>　　咚！咚！咚！
<br/>　　来到石殿门前，苏莫轻轻的敲了敲殿门。
<br/>　　“上官昊，有何事？”石殿的殿门打开，赖羽淡淡的问道。
<br/>　　“赖队长，我要离开营地一段时间，特来向你报备一声！”苏莫沉声说道，他离开古灵星河之前，必须要先炼制分身，所有现在要出去找寻合适的肉身。
<br/>　　“离开营地？”
<br/>　　赖羽闻言皱眉，随即冷冷的说道：“不允！”
<br/>　　“为何不允？”苏莫眼眸微眯了下来，这里即便是军营，也不可能不让人离开吧！
<br/>　　“你今天才刚到赤龙营，就要暂离军营，未免太不将我这个队长放在眼里了吧！”赖羽冷冷的说道。
<br/>　　“难道有事都不能离开？”苏莫脸色沉了下来。
<br/>　　“不错，待满一个月之后，才有离开营地的资格！”赖羽说道，他心中冷笑，这个上官昊该不会是回去向八皇子禀报消息吧！
<br/>　　而且，他已经有了安排，过段时间就安排机会整死苏莫，那就更不能让苏莫离开了。
<br/>　　“你确定不允？”苏莫淡淡的问道。
<br/>　　“不允！”赖羽言语决绝，丝毫不给苏莫面子。
<br/>　　“请问赖队长，小队长有资格随时离开营地吗？”苏莫问道。
<br/>　　赖羽闻言一怔，随口道：“小队长当然能随时离开，你什么意思？”
<br/>　　“没有什么意思，从今以后，我就是二十九小队的队长！”苏莫淡漠的说道。
<br/>　　“什么？”
<br/>　　赖羽闻言，顿时脸色一沉，眸中寒光爆闪，道：“你的意思是要挑战我了？”
<br/>　　赖羽心中杀机缓缓升腾，这上官昊居然如此的狂妄，自认为能战胜他，夺得小队长的职务？
<br/>　　“不错，我，挑战你！”苏莫声音淡淡，一字一句的说道。
<br/>　　“呵呵！”
<br/>　　赖羽笑了起来，嘴角勾起了森冷的笑意，道：“真是狂妄，八皇子居然安排你这样的人来赤龙营，当真可笑啊！”
<br/>　　赖羽心中不屑，他本以为苏莫进入赤龙营之后，会谨小慎微，稳步发展，没想到刚来第一天就要挑战他！
<br/>　　“废话少说，到哪里战斗，打败了你，我还等着离开营地呢！”苏莫淡淡的说道，神情颇为轻松，仿佛打败赖羽没有任何难度一般。
<br/>　　“你……！”赖羽闻言，顿时脸色一黑，他感受到了巨大的侮辱。
<br/>　　这个上官昊，神情淡然，好像丝毫没有将他放在眼里，这种藐视的姿态，顿时让他心中大怒。
<br/>　　“一个小小的武帝境八重武者，居然也敢如此的狂妄，你以为你是真圣榜上的天才吗？”
<br/>　　赖羽怒喝一声，道：“走，我们先去通知中队长，而后去虚云殿！”
<br/>　　言罢，赖羽大步离开，径直向祝英所在的石殿走了过去。
<br/>　　苏莫紧跟着对方而去，既然对方不允他离开营地，那他就只能用强了。
<br/>　　“我的天，这个上官昊刚来就要挑战老大！”
<br/>　　“好胆色，以武帝境八重修为就敢挑战队长！”
<br/>　　“走，我们先去虚云殿等着！”
<br/>　　赖羽的喝声，已经惊动了二十九小队的其他人，其他八人纷纷向虚云殿而去。
<br/>　　甚至，连附近其它小队的人，都被惊动了。
<br/>　　“厉害啊！”恒飞一边向虚云殿走去，一边暗暗惊讶，其实他也不确定这个上官昊是不是苏莫，只是发觉特别的像而已。
<br/>　　但是，通过等会的战斗，他就能断定上官昊是不是苏莫。
<br/>　　赖羽和苏莫两人，很快便来到了祝英的石殿中，见到了中队长祝英。
<br/>　　“中队长，上官昊向我挑战，现在我和他就去虚云殿！”赖羽对祝英说道。
<br/>　　“哦！”祝英闻言双眉一挑，随即目光落在了苏莫的身上。
<br/>　　刚来赤龙营，就要挑战小队长，欲取而代之，这个上官昊看来是有备而来啊。
<br/>　　不过，他倒是来了兴趣，倒是要看看武帝境八重的上官昊，如何能打败武帝境九重的赖羽。
<br/>　　“上官昊，你可知道我们赤龙营的规矩？”祝英微笑着问道。
<br/>　　“什么规矩？”苏莫沉声问道。
<br/>　　“若你能打败赖羽，自然便能取而代之，成为二十九小队的队长，但你若是败了，以后有什么不公平的待遇，可不能有怨言！”祝英告诫道，他觉得有必要提醒一下上官昊。
<br/>　　免得对方败了之后，被赖羽打压，再到八皇子那里告状。
<br/>　　“不会！”苏莫摇了摇头。
<br/>　　“那好，既然如此，我们一起去虚云殿吧！”祝英微笑着说道。
<br/>　　言罢，他一马当先，带着赖羽和苏莫，向虚云殿而去。
