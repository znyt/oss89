　　钟声的出现，响彻整个归墟城中。
<br/>　　楚风眠听到这钟声响起的一刻，也是缓缓的睁开了双眼，在他旁边的寒月漓也是同样如此。
<br/>　　两道身影，顿时便是从这洞府之中直接走了出去。
<br/>　　而在外面，整个归墟城中的武者，几乎都听到这钟声的一刻，便是一个个都从洞府中走了出来。
<br/>　　一道道目光，注视着天空之上。
<br/>　　“轰隆！”
<br/>　　就在无数武者从洞府中走出来的一刻，一股庞大的灵力，突然从天而降。
<br/>　　这灵力，如同一条长河一般，从天空之上一直延续在归墟城中。
<br/>　　而在这长河之中，无数身影，纷纷落下，这每一道身影身上的气息，都是恐怖到极致，就算是其中最弱的几道气息，都是至少有着超越生死的地步。
<br/>　　这些人，便是九域的大人物，全部降临了。
<br/>　　来选择进入归墟天的天才，许多人的实力都是要足以跟生死境巅峰媲美，甚至是超越生死境巅峰。
<br/>　　这些前来进入归墟天的天才，一个个都是心高气傲之辈，所以想要压制住他们，必须要是超越生死的强者才行。
<br/>　　所以如今这降临的强者中，最弱的也要是超越生死，踏上生死台阶碎命之境的人物。
<br/>　　“诸位都是来自于九域的天才，都是为了这归墟天的开启而来，这归墟天开启已经是无数次了，其中的种种情况你们也应该知道，我这里就不一一讲解了。”
<br/>　　在这群强者中的一尊白发老者，突然开口道。
<br/>　　这白发老者身上的气息，比起周围的所有强者，都要强大几分，显然是在生死台阶上，都是造诣极高的一位强者。
<br/>　　这白发老者的实力，深不可测，楚风眠感觉到，这白发老者想要出手，几乎都足以瞬间轰杀楚风眠。
<br/>　　现在楚风眠虽然是已经是足以跟碎命之境，甚至是长生之境的强者抗衡一二了，但是这仅仅只是生死台阶最前的两重。
<br/>　　真正达到生死台阶极高的强者，楚风眠还是远远无法对付的。
<br/>　　楚风眠站在原地，也是隐藏了自己身上的种种奇遇，以防止被其他强者发现，以防万一。
<br/>　　这名老者，一边开口，一边目光便是扫视着在场众人。
<br/>　　这次夺得归墟天资格的一百个名额的武者，如今便是已经全部到齐。
<br/>　　进入归墟天，虽然是危险重重，但是能够拿到名额的武者，却是没有任何一人会选择放弃，毕竟归墟天中可是拥有这无数宝藏，机遇。
<br/>　　这样好的机会，谁都不愿意放弃，就算是危险重重，但是如今能够实力达到这一步的。
<br/>　　除了自己的资质之外，几乎每个人都是从一次次的生死中摸爬滚打出来，没有任何一人，是怕死之人。
<br/>　　在扫视一遍人群之中，这白发老者便是点了点头，随后继续开口道。
<br/>　　“归墟天的开启，一共只有一月时间，一个月后，我们会再度联手，撕裂空间，一共持续一个时辰，你们必须要在这一个时辰之中离开，不然便是就只能永远的困死在归墟天中。”
<br/>　　白发老者冷冷的开口道。
<br/>　　听到这番话，楚风眠便是点了点头。
<br/>　　归墟天这样的地方，在其中每时每刻都是危险重重，一个月的时间，是最好的选择，若是真的在其中待上三年的话，可能这无数武者，没有任何一人可以活着出来的。
<br/>　　那才是真正的得不偿失。
<br/>　　“废话就不多说了，我们这就要开启归墟天的通道，所有武者，都可以进入其中！”
<br/>　　白发老者冷声开口，顿时凝聚灵力。
<br/>　　在他的身边，那些强者，也都是纷纷凝聚力量。
<br/>　　这些强者，都是来自于九域之中各大势力的强者，足足是有着数十位之多，一同凝聚力量，全部注入到这了白发老者的身躯之中。
<br/>　　只见这白发老者大手一挥，猛然冲着天空之上撕裂过去，这天空，在他的一爪之下都是被撕裂，出现了无数裂痕。
<br/>　　在这裂痕的背后，便是显现出来的一处截然不同的区域，如同另外一方世界。
<br/>　　这天空的背后，正是归墟天。
<br/>　　“嗖！嗖！嗖！”
<br/>　　这空间被撕裂的瞬间，无数身影便是动了。
<br/>　　就在楚风眠周围的洞府，几道身影便是凌空而动，踏空而去，直接一跃向着撕裂的空间冲了过去。
<br/>　　这几人的身上，刻有星图，凌空踏出，每一脚落下便是化为一方星陨，这些武者便是七大宗门之一，天枢殿的弟子。
<br/>　　初次之外，几乎剩下七大宗门的弟子，也早已经纷纷上前。
<br/>　　这七大宗门的弟子，便是第一时间动手，几乎是要抢占先机。
<br/>　　归墟天中，宝藏机缘无数，但是抢占先机，就算是快上一秒，也许都是足以多得到一道奇遇，所以无数武者，都是想要争夺，进入其中。
<br/>　　“碰！”
<br/>　　但是就在这时刻，在空中却是突然爆发出了一股灵力。
<br/>　　只见一尊武者，也想要跟七大宗门的弟子争夺，想要向前一步，抢占先机，却是被一尊九魔城弟子，一掌直接给打飞到了地上，受了不少的伤。
<br/>　　“该死！”
<br/>　　“这七大宗门的人，真是霸道！”
<br/>　　一尊尊武者，看着这一幕都是打骂一声。
<br/>　　这七大宗门的弟子，似乎已经是有着约定，不互相攻击，抢占先机。
<br/>　　但是越是其他宗门的弟子，想要进入其中，他们便是都会阻拦。
<br/>　　甚至这七大宗门弟子，都是留下了几人，来阻拦其他武者，好让其他七大宗门弟子，在其中抢占先机，大肆掠夺。
<br/>　　楚风眠也看得见，这些七大宗门的弟子，都在攻击这其他武者，不让其他武者进入其中。
<br/>　　“这七大宗门的弟子，比起万年前看来还要霸道了。”
<br/>　　楚风眠看着天空之上，突然冷笑一声。
<br/>　　这样阻拦下去，其中的宝藏，只怕都会被七大宗门的弟子，先搜刮一遍。
<br/>　　倒时候在进入，只能是捡别人挑剩下的。
<br/>　　楚风眠这次前来归墟天，可不是来捡别人剩下的东西的。
<br/>　　“我们走！”
<br/>　　楚风眠对着旁边的寒月漓，轻声说了一声。
<br/>　　他的身躯，便是直接冲着天空之上，飞了上去。
