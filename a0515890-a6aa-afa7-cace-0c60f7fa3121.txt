        傅夜沉的背后，是一片蔚蓝色的大海。但他挺拔的英姿，却并未被大海虚化。
<br/>
<br/>        童筱颖看着傅夜沉渐渐地看出了神，却没有忘记此刻心情的焦虑，给她带来的不安。
<br/>
<br/>        现场的乐队，奏起了婚礼进行曲。
<br/>
<br/>        连中凯将苏漫雪的手，交到了傅夜沉的面前，但傅夜沉并未接手。
<br/>
<br/>        三人之间略显尴尬，司仪立马上前解围。
<br/>
<br/>        交响乐戛然而止，司仪宣读一番婚姻誓言词后，便开始质问站在他右侧的苏漫雪。
<br/>
<br/>        “童筱颖女士，你是否愿意这个男子成为你的丈夫与他缔结婚约？无论贫穷还是富贵，或任何其他理由，都爱他，照顾他，尊重他，接纳他，永远对他忠贞不渝直至生命尽头？”
<br/>
<br/>        “我愿意！”苏漫雪学童筱颖的声音如出一辙。
<br/>
<br/>        司仪又看向站在自己左边的傅夜沉，质问：“傅夜沉先生，你是否愿意这个女人成为你的妻子与她缔结婚约？无论疾病还是健康，或任何其他理由，都爱她，照顾她，尊重她，接纳她，永远对她忠贞不渝直至生命尽头？”
<br/>
<br/>        “我不愿意。”傅夜沉突然冷冷地回答道。
<br/>
<br/>        台下，连中凯立马气得脸色发紫，还未等他开骂，傅夜沉目光冷冽，接着对面前的女人质问道：“苏漫雪，你以为你还能像以前那样，冒充筱颖，成为我傅夜沉的女人吗？”
<br/>
<br/>        “大哥，你在说什么啊？我听不懂。”苏漫雪娇嗔，双手却不安地捏着头纱的一角，不停地在指尖缠绕。
<br/>
<br/>        傅夜沉冷哼，直接无视掉苏漫雪，朝台下走去。
<br/>
<br/>        就在大家疑惑不解的时候，只见傅夜沉来到了最后一排坐着的一个戴着纱巾和墨镜，遮挡住了所有面容的女人跟前。
<br/>
<br/>        童筱颖怔愣地仰头看向傅夜沉，他俊脸从容淡定，完全看不出他此刻是什么心情。
<br/>
<br/>        下一秒，傅夜沉伸手将童筱颖从长椅上拉了起来，一把揽住她的腰肢，紧紧地扣入自己怀中，嘴角上扬，噙着一抹痞痞的笑意：“老婆大人，你老公我今天帅不帅？”
<br/>
<br/>        “？！”童筱颖一脸懵。
<br/>
<br/>        傅夜沉会心而笑，替她摘掉了脸上的墨镜和头巾，见她嘴唇上破了皮，留有血痂，顿时脸上笑容渐敛。
<br/>
<br/>        他抬起手来，打了个响指。
<br/>
<br/>        顷刻间，从四面八方疾步跑过来八个戴着墨镜身着西装革履的黑衣人。
<br/>
<br/>        他们二话不说，便直接上台将苏漫雪腾空架起。
<br/>
<br/>        苏漫雪吓得艰难求饶：“傅夜沉！你放了我！你放了我！看在我曾经给你生过孩子的份上……唔……”
<br/>
<br/>        黑衣人一记拳头，直接粗暴简单地将苏漫雪打晕了过去。
<br/>
<br/>        台下一片唏嘘。
<br/>
<br/>        连中凯听了苏漫雪那话后，直接走到童筱颖跟前，将童筱颖从傅夜沉的怀中拉了出来。
<br/>
<br/>        “哼！你这个行为不检点的男人，休想我把女儿嫁给你！”连中凯气急败坏地说道。
<br/>
<br/>        童筱颖懊恼地甩开连中凯的手，重新挽住傅夜沉的臂弯，没好气地对连中凯说道：“我的事情，跟你无关！”
<br/>
<br/>        “他都跟别的女人有孩子了，你还嫁给他，你不是傻吗？”连中凯此刻就像热锅上的蚂蚁。
<br/>
<br/>        连华生看不下去了，不得不上前将连中凯拉到一旁。
<br/>
<br/>        台上的司仪拿着话筒，弱弱地问道：“那个……傅夜沉先生，您的婚礼还继续吗？”
<br/>
<br/>        “当然继续。”傅夜沉会心一笑，牵着童筱颖的手往台上有人。
<br/>
<br/>        司仪立马又将刚刚的婚礼宣誓词，重新质问了一遍。
<br/>
<br/>        直到傅夜沉和童筱颖彼此为彼此戴上婚戒后，台下终于响起了雷鸣般的掌声。
<br/>
<br/>        连华生把苏漫雪的事情，从头到尾地给连中凯讲了一遍后，连中凯义愤填膺地说道：“哼！这个女人！竟然敢欺负我女儿！看我怎么收拾她！”
<br/>
<br/>        “不需要你动手，阿沉这回是不会再放过苏漫雪了。”连华生忍俊不禁道。
<br/>
<br/>        连中凯立马拍了拍连华生的肩膀，警告道：“小子，从今以后，你得给我好好保护你妹妹！若是你妹妹被傅夜沉那厮欺负了，我第一个唯你是问！”
<br/>
<br/>        “他们夫妻俩的事，跟我有什么关系啊？”连华生一脸苦瓜像。
<br/>
<br/>        连中凯振振有词道：“谁叫你跟傅夜沉是一伙的？”
<br/>
<br/>        “我去！长这么大，我还是第一次发现，你这老头，居然是女儿奴！”连华生不禁用鄙夷的眼神看着连中凯。
<br/>
<br/>        就在大家的掌声中，司仪要求新郎给新娘一个吻时，连中凯立马一个激灵站出来，反对道：“傅夜沉，我不许你当众吻我女儿！”
