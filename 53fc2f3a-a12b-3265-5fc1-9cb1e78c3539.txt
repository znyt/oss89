　　一招之下。
<br/>　　这虚家老祖，居然是被直接逼退了。
<br/>　　就连虚家老祖自己，都不敢相信这一幕。
<br/>　　虚家老祖是什么人物？斗玄国度中最为强大的存在之一，生死境二重的强者。
<br/>　　是虚家这个庞然大物的顶点，赫赫有名的大人物，已经度过生死大劫，近乎永生不死的存在。
<br/>　　“畜生！你居然隐藏了自己的实力！”
<br/>　　被打飞的虚家老祖，一下子冲了过来，怒发冲冠，头发根根林立，无边怒火，爆发而出。
<br/>　　能够一招打飞了他，虚家老祖不相信楚风眠真的是什么御风境巅峰的境界。
<br/>　　他可是一尊生死境强者，一个御风境巅峰的武者怎么可能把他打飞。
<br/>　　只有一种可能性，便是楚风眠隐藏了自己的实力。
<br/>　　“你有隐藏实力又如何？老夫看的出来，你的境界还没有踏入生死境！最多只能是圣者中的佼佼者！”
<br/>　　“苍天血泪！”
<br/>　　虚家老祖怒喝一声道，突然之间，天地便是下起了漫天血雨。
<br/>　　这每一滴血雨，都蕴含有着无尽强大的魔力。
<br/>　　若是滴在武者的身上，都会直接灼烧身躯，甚至是至少烧死。
<br/>　　这是他们虚家的魔功，血雨一出，天地变色，足以毁灭要做城池。
<br/>　　但是楚风眠，却是脸色平静到了极点。
<br/>　　面对着漫天血雨，他的表情都没有丝毫一点担心。
<br/>　　楚风眠脚下一动，身形直接来到了血雨之中，向着那虚家老祖冲了过去。
<br/>　　这一滴滴蕴含无尽魔力的血雨，在快要滴在楚风眠的身上，居然都是被直接化解开来。
<br/>　　没有任何一滴血雨，可以真正滴在楚风眠的身上。
<br/>　　楚风眠就这样，大摇大摆的走入到血雨之中，在虚家老祖不可置信的眼神中，一下子冲到了他的面前。
<br/>　　“不可能！”
<br/>　　看着楚风眠在这血雨之中，丝毫不受影响，视如无物的走入其中，他的眼睛都是露出了惶恐之色，差点晕死了过去。
<br/>　　这血雨，就算是同为生死境的强者，对付起来，都必须要小心谨慎才行。
<br/>　　楚风眠居然可以做到视如无物，这简直是令他不可置信到极点。
<br/>　　他自信观察，才发现，这些血雨，居然都是被楚风眠背后的远古战神虚影全部化解了。
<br/>　　这一滴滴的血雨，全部成为了楚风眠的力量，都被楚风眠背后的远古杀神虚影全部化解吞噬了。
<br/>　　他们虚家的魔功，被楚风眠背后的远古杀神虚影，死死的给克制住了。
<br/>　　“该死！”
<br/>　　虚家老祖大骂道。
<br/>　　楚风眠的杀戮剑术可以大成，还都是因为他们虚家，卖给了楚风眠那么一大堆的圣者精血，妖神精血。
<br/>　　如今吃亏的，反而是他们。
<br/>　　不过。
<br/>　　虚家老祖感觉着楚风眠身上的魔血之力，突然一声冷笑。
<br/>　　就算是杀戮剑术大成了又如何？他的魔血一旦催动，这楚风眠便是就直接被他所控制。
<br/>　　虚家老祖思考之际，楚风眠的身形已经是来到了他的面前。
<br/>　　“给我过来！”
<br/>　　楚风眠大手一抓，直接催动灵力，化为了一道灵力大手。
<br/>　　就在虚家老祖的面前，猛然对着他抓了过去。
<br/>　　楚风眠一抓，身体中催动起那一千二百九十六滴远古战龙精血，顿时化为了一道巨大无比的战龙之爪。
<br/>　　这一爪，简直骇人到极点。
<br/>　　“小子，你的实力的确不错！但是很可惜，你还是太嫩了点。”
<br/>　　虚家老祖看着楚风眠，突然大笑起来，脸色尽是阴冷。
<br/>　　“你以为我们虚家的那些圣者精血，妖神精血是那么好得到的吗？”
<br/>　　“魔血深种！统统给我激发出来！”
<br/>　　随着虚家老祖的一声怒喝，在楚风眠的身躯之中，那些魔血，突然全部激发了出来。
<br/>　　感觉到楚风眠身上的魔血被激发，虚家老祖的眼神中尽是冷笑，更多的则是贪婪。
<br/>　　楚风眠小小年纪，便是有着如此惊人的财富，还有着如此实力。
<br/>　　身上肯定是有着极大的奇遇，他今日杀了楚风眠，不单单是夺回那天元神丹，更是可以得到楚风眠身上的奇遇。
<br/>　　“哈哈哈哈，小子，现在你下跪求饶都晚了！你会被魔血控制身体，完全的臣服与我，成为我的奴隶！”
<br/>　　虚家老祖，大笑连连道，他的表情中，便是看起来已经是胜券在握的样子。
<br/>　　他伸出手，指着楚风眠来，突然一股灵力催动起来，大声喝道。
<br/>　　“跪下！”
<br/>　　这两个字喊出的一刻，楚风眠的身躯，仿佛都是受到了一股巨大无比力量的影响，想要跪下一样。
<br/>　　楚风眠身躯中的魔血之力，都被这虚家老祖彻底的激发出来，要夺取了楚风眠的身躯。
<br/>　　可是下一刻，在他想象中的一幕，却是并没有发生。
<br/>　　楚风眠，依然是完好无损的站在原地。
<br/>　　甚至是在楚风眠脸上的表情中，都是能够看出来其中的几分嘲讽？甚至是有点玩弄的意思。
<br/>　　“怎么回事？你的身体中明明有着魔血的力量，怎么可能不受到我的控制？”
<br/>　　虚家老祖一愣，指着楚风眠，眼神中尽是不可置信，内心在疯狂咆哮。
<br/>　　他想不明白，为何楚风眠可以化解了他的魔血。
<br/>　　这魔血，只要是有着一定数量，在武者的体内，就算是在强大的武者，也不可能完全化解的了。
<br/>　　因为魔血会融入到人的血脉之中去，除非是自废自己的血脉，不然永远无法清除。
<br/>　　在他看来，这简直已经是稳操胜券的事，可眼前的变化，是他完全无法想象的。
<br/>　　“老家伙，你真以为你的那小小魔血，可以控制了我？”
<br/>　　楚风眠看着虚家老祖，眼神中尽是不屑的说道。
<br/>　　“我跟九魔城那些老魔头打交道的时候，你还不知道在哪里了，不过就是这些小小魔血罢了，我想要炼化，瞬息之间便是可以做到。”
<br/>　　楚风眠一边说着，一边就在这虚家老祖的面前。
<br/>　　他身上的灵力，突然凝聚，一股战龙之力，压迫这那魔血，顿时这魔血的力量，被彻底消散。
<br/>　　化为了精粹的灵力，全部融入到了楚风眠的身躯之中去，成为了楚风眠的给养。
