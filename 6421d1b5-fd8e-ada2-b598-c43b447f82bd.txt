<br/>　　天地之中响彻起恐怖的轰鸣声。

<br/>　　“我的天！荒圣君四人联手之下，居然都被打破了？”

<br/>　　这一掌轰击而下，古白匆忙的凝聚灵力，同样也是一掌打了过去，却是其中的力量都有直接打散。

<br/>　　一声惨叫声传来，只见古白的身躯，在这一掌的力量之下，纷纷破碎，整个人都是变化的血肉模糊的样子，五脏六腑都被直接打碎。
<br/>　　若非是因为古白的实力已经是达到了生死境极限的地步，这样的伤势，让大部分的生死境武者来，只怕都必死无疑。

<br/>　　可现在他的身躯五脏六腑早已经是被搅碎，如今古白已经是没有多少力量了。
<br/>　　他的目光露出几分惊恐。
<br/>　　这个情况下，几乎是楚风眠愿意，便是轻易取了这古白的性命。
<br/>　　“古月峡谷的古白，居然是一招都无法抵挡住，怎么可能？”
