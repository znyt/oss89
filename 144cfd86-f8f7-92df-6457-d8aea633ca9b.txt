　　说实话，现在大唐的国库有钱，而且还不是一般的有钱，去掉李承乾曾经答应那些勋贵们的分成以后，每年国库的收入都在五千万贯左右。
<br/>　　所以李承乾并不在乎给那些勋贵们一点好的待遇，用一点小钱来换取这些人的忠心。
<br/>　　而那些勋贵其实差的也并不是钱，主要还是一个面子的问题，毕竟给大唐卖了一辈子命，就算是没有功劳也有苦劳，没有苦劳也还有疲劳。而李承乾这一举措所代表的意义则是一种认可，认可他们为大唐所做的贡献，这正是勋贵们所需要的。
<br/>　　敲定了这些事情，时间已经到了中午，安排那些老家伙吃了一顿之后，李承乾便将他们全都打发了回去，让他们这几个老头子好好考虑一下接下来工资需要如何涨的问题。
<br/>　　最终，老头子们终于走了，李承乾苦笑着走出书房，抬头看着头顶那块并不大的天空发出一声长长的叹息。
<br/>　　自从当了这个破皇帝，整个人都不自由了，以前当太子的时候，他还可以没事溜出宫去到处走走，可是现在……现在走到哪里都是人一群马一伙，弄不好还要被一群老货弹劾，好不尴尬。
<br/>　　也不知道为什么会有那么多人对这个位置那么感兴趣，明明就是一个没有自由兼之十分苦逼的职业，如果有可能他是真的不想接手这个位置。
<br/>　　李承乾十分矫情的想着，顺便活动了一下自己的脖子，然后转头对身后的小白问道：“程妃回来了没有？”
<br/>　　“回陛下，还没有！”白月宁回答道。
<br/>　　还没有……，这败家娘们儿，一出去就不想着回来，李承乾撇撇嘴，顿了顿说道：“收拾一下，一会儿陪朕去卢国公府上走一趟。”
<br/>　　“是！”小白答应了一声，转身去了。
<br/>　　不过显然所谓收拾一下并不是让小白收拾，而是李承乾自己收拾，跟以往溜出宫一样，他需要换衣服，需要安排护卫，好多事情都要准备，这些事情到了他的口中统统变成了收拾一下。
<br/>　　……
<br/>　　程咬金在半个月前已经回到了长安，每日里除了聚会就是饮酒，小日子过的好不舒服。
<br/>　　好不容易等他喝够了，这才想起自己还有个外孙女，于是便让人进宫商量能不能把琅琊接到自己府上住几天。
<br/>　　李承乾对此自然是无可无不可，但是考虑到这老货不怎么靠谱的性格，他还是决定让程琳带着娃一起过去，至少有她守在一边，老程应该不会折腾的那么过份。
<br/>　　可是让他十分意外的是，闺女这一去就是五天，每天除了有人回来报个平安之外，完全没了其他消息，似乎如果他不去催问一下，那老东西就不放人一般。
<br/>　　而程咬金呢，这老货其实还真没有什么别特的想法，甚至他早就已经想把家里那一大一小两个丫头快点送回宫去。
<br/>　　但是奈何大的好说，小的却不是那么听话，老程大道理说了一堆，可是在程府玩嗨了的琅琊就是不听，死活也不肯回去。
<br/>　　毕竟在宫里这也不许，那也不行，而在外公家却可以随便，想什么时候睡就什么时候睡，想玩刀就玩刀，想骑马就骑马，这可把这个七、八岁的小丫头给开心坏了。
<br/>　　程琳那娘们儿也是，回了家之后每天除了麻将就是麻将，通宵达旦无休无止，每天除了吃就是玩，连管孩子的心思都没了，至于当初答应李承乾的差事……随它去吧，爱咋咋地，先乐呵几天再说！
<br/>　　李承乾就是在这样的情况下来到程府的，在下人的引领下来到前厅坐下，立刻就看到老程风风火火的自后面赶了出来。
<br/>　　“老臣见过陛下，陛下……”
<br/>　　“丈人免礼。”看着准备施礼的老程，李承乾连忙起身将他拦了下来，同时说道：“丈人何必每次都是如此，你我乃是一家，若总是这般却是让小婿没办法再登门了。”
<br/>　　“嘿嘿……”程老魔头发出一声怪笑，随后坐到李承乾身边：“陛下是来接人的吧？”
<br/>　　“接人？呃……不，我就是来看看。”李承乾多精明的一个人，瞅着所剩无几的胡子，连忙否认自己此行的目的。
<br/>　　“唉，陛下不用为难，其实说来琳丫头也的确是不像话，出了宫就不知道回去，实在是有些过份了。”程咬金嘴角抽了抽，像是没有听懂李承乾的话一般，自故自的说道。
<br/>　　“这没什么，为人子女承欢膝下乃是本份，小四这样做也是一片孝心，朕何来为难之说。”李承乾脸上露出一抹坏笑，就像没有听到程咬金的话一般。
<br/>　　而就在此时，李承乾听到耳中传来一阵凌乱的马蹄声传来，同时伴随着他闺女嘻嘻哈哈的笑声，接着便看到一个不大的影子，骑着一匹高头大马从外面窜进了会客厅。
<br/>　　战马被见机不对的小白冲上去拉住了缰绳，而那个不大的影子则迅速从马上跳下来，甜甜的对小白一笑，然后便冲向李承乾：“爹爹！”
<br/>　　因为教育的关系，李承乾一直没有让几个孩子对自己用什么‘父皇’之类的称呼，都是自己的儿女，扯那个犊子根本没有必要。
<br/>　　再说皇室之人本就亲情淡薄，再弄个什么‘父皇’‘母妃’之类的官称，则更显得没有人情味了些。
<br/>　　所以不管是李象，还是李小象，又或者琅琊，对李承乾的称呼统一是‘爹’，除此之外再没无其他。
<br/>　　书归正传，却说李承乾伸手拉住冲过来的闺女，上下看了她一眼，然后正色问道：“丫头，外公就在那边，为何不去打招呼？”
<br/>　　“外公刚刚还在和人家玩呢，才分开不到一刻钟，所以不用打招呼啊，倒是爹爹有好几天没见了！”琅琊乖巧的说道，与刚刚骑着马冲进来的样子判若两人。
<br/>　　李承乾看了一眼尴尬的老程，在琅琊的头上拍了一下：“你这小丫头最是调皮，这几天在公外家里没有惹什么麻烦吧？”
<br/>　　“没有没有，不信您问问外公！”琅琊把小脑袋摇的飞快，坚定的否认了自己的一切恶行。
